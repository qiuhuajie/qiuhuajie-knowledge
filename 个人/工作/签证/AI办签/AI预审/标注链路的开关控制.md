# 标注链路的开关控制
* https://switch.alibaba-inc.com/#/switchList?appName=travelvc&envGroup=onlineInnerGroup&unit=
* 标注任务注册抽样率：AUTO_LABELING_TASK_REGISTER_SAMPLE_RATE
* 注册材料审核自动数据标注任务执行降级：REGISTER_MATERIAL_LABELING_TASK_EXECUTE_CLOSE
* 注册材料审核自动数据标注任务执行时间mock（mock成一分钟之后）：REGISTER_LABELING_TASK_EXECUTE_TIME_MOCK
	* 根据当前配置值：
		* EXECUTE_START_TIME = 0（起始偏移 0 分钟，即从 00:00 开始）
		* EXECUTE_TIME_SPAN = 301（区间跨度 301 分钟）
	* 所以实际效果是：任务会被安排在明天 00:00 ~ 05:01（约凌晨 0 点到 5 点）之间的随机时刻执行，目的是将标注任务分散到凌晨低峰期，避免集中请求。
