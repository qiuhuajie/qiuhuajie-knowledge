# 1. 介绍
![[IMG-20260315213656237.png|Untitled 268.png]]
1. **介绍：**
    
    1. 迭代器模式（Iterator Pattern）是 Java 和 .Net 编程环境中非常常用的设计模式。这种模式用于顺序访问集合对象的元素，不需要知道集合对象的底层表示
    
    1. 迭代器模式属于行为型模式
    
1. ==**意图：**==**==提供一种方法顺序访问一个聚合对象中各个元素，而又无须暴露该对象的内部表示==**
1. ==**主要解决：**==
    
    1. **==迭代器模式解决了不同集合统一遍历问题==**（解释见下）
    
    1. 不同的方式来遍历整个整合对象
    
1. **何时使用：**遍历一个聚合对象
1. **如何解决：**把在元素之间游走的责任交给迭代器，而不是聚合对象
1. **关键代码：**定义接口：hasNext, next
1. **应用实例：**JAVA 中的 iterator
1. **优点：**
    
    1. 它支持以不同的方式遍历一个聚合对象
    
    1. 迭代器简化了聚合类
    
    1. 在同一个聚合上可以有多个遍历
    
    1. 在迭代器模式中，增加新的聚合类和迭代器类都很方便，无须修改原有代码
    
1. **缺点：**由于迭代器模式将存储数据和遍历数据的职责分离，增加新的聚合类需要对应增加新的迭代器类，类的个数成对增加，这在一定程度上增加了系统的复杂性
1. **使用场景**
    
    1.  访问一个聚合对象的内容而无须暴露它的内部表示
    
    1. 需要为聚合对象提供多种遍历方式
    
    1. 为遍历不同的聚合结构提供一个统一的接口
    
1. **注意事项：**迭代器模式就是分离了集合对象的遍历行为，抽象出一个迭代器类来负责，这样既可以做到不暴露集合的内部结构，又可让外部代码透明地访问集合内部的数据。
# 2. 登场角色
## 2.1 Iterator 模式中有以下登场角色
1. **lterator(迭代器)**
    
    - 该角色负责定义按顺序逐个遍历元素的接口(API)
    
    - 也即 Java 提供的 `**lterator**` 接口
    
1. **Concretelterator(具体的迭代器)**
    
    该角色负责实现 Iterator角色所定义的接口(API)
    
1. **Aggregate(集合)**
    
    1. 该角色负责定义创建Iterator 角色的接口(API)
    
    1. 这个接口(API)是一个方法，会创建出“按顺序访问保存在我内部元素的人”
    
1. **ConcreteAggregate(具体的集合)**
    
    1. 该角色负责实现Aggregate角色所定义的接口(API)
    
    1. 它会创建出具体的Iterator 角色，即Concretelterator 角色

# 3. 代码示例
1. `Department`
    
    ```Java
    @Data
    @AllArgsConstructor
    @NoArgsConstructor
    public class Department {
        private String name;
        private String des;
    }
    ```
    
1. `College` ➡️ **Aggregate(集合)**
    
    ```Java
    public interface College {
        String getName();
    
        void addDepartment(String name, String des);
    
        Iterator createIterator();
    }
    ```
    
1. `College01` ➡️ **ConcreteAggregate(具体的集合)，**College01 中 department 以**数组**形式存储==**（迭代器模式解决了不同集合统一遍历问题）**==
    
    ```Java
    public class College01 implements College{
    
        Department[] departments;
        int length = 0;
    
        public College01() {
            this.departments = new Department[5];
            addDepartment("java", "java");
            addDepartment("c++", "c++");
        }
    
        @Override
        public String getName() {
            return "计算机学院";
        }
    
        @Override
        public void addDepartment(String name, String des) {
            Department department = new Department(name, des);
            departments[length] = department;
            length++;
        }
    
        @Override
        public Iterator createIterator() {
            return new CollegeIterator01(departments);
        }
    }
    ```
    
1. `College02` ➡️ **ConcreteAggregate(具体的集合)，**College02 中 department 以**链表**形式存储
    
    ```Java
    public class College02 implements College{
    
        List<Department> departments;
        int size = -1;
    
        public College02() {
            departments = new ArrayList<>();
            addDepartment("刑法", "刑法");
            addDepartment("民法", "民法");
        }
    
        @Override
        public String getName() {
            return "法学院";
        }
    
        @Override
        public void addDepartment(String name, String des) {
            departments.add(new Department(name, des));
        }
    
        @Override
        public Iterator createIterator() {
            return new CollegeIterator02(departments);
        }
    }
    ```
    
1. `CollegeIterator01` ➡️ **Concretelterator(具体的迭代器)**
    
    ```Java
    public class CollegeIterator01 implements Iterator{
    
        Department[] departments;
        int position = 0;
    
        public CollegeIterator01(Department[] departments) {
            this.departments = departments;
        }
    
        @Override
        public boolean hasNext() {
            if (position >= departments.length || departments[position] == null) {
                return false;
            }else {
                return true;
            }
        }
    
        @Override
        public Object next() {
            Department cur = departments[position++];
            return cur;
        }
    
        @Override
        public void remove() {
            Iterator.super.remove();
        }
    }
    ```
    
1. `CollegeIterator02` ➡️ **Concretelterator(具体的迭代器)**
    
    ```Java
    public class CollegeIterator02 implements Iterator{
        List<Department> departments;
        int index = -1;
    
        public CollegeIterator02(List<Department> departments) {
            this.departments = departments;
        }
    
        @Override
        public boolean hasNext() {
            if (index >= departments.size() - 1) {
                return false;
            }else {
                return true;
            }
        }
    
        @Override
        public Object next() {
            return departments.get(++index);
        }
    
        @Override
        public void remove() {
            Iterator.super.remove();
        }
    }
    ```
    
1. `IteratorDemo`
    
    ```Java
    public class IteratorDemo {
        public static void main(String[] args) {
            // 初始化两个学院
            ArrayList<College> colleges = new ArrayList<>();
            colleges.add(new College01());
            colleges.add(new College02());
    
            // 遍历所有学院
            for (College college : colleges) {
                System.out.println(college.getName());
                // 获得当前学院的迭代器
                Iterator iterator = college.createIterator();
                // 遍历当前学院的所有 dep
                while (iterator.hasNext()){
                    Department next = (Department) iterator.next();
                    System.out.println(next.getName());
                }
            }
        }
    }
    ```
