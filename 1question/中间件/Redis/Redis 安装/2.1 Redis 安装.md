# 1. **系统设置**
1. 防火墙
    
    ```Shell
      systemctl stop firewalld
      systemctl disable firewalld
      systemctl status firewalld
    ```
    
1. SELinux
    
    ```Shell
      # 永久关闭
      sed -i 's/enforcing/disabled/' /etc/selinux/config
    
      # 临时关闭
      setenforce 0
    ```
    
1. yum 源更新
    
    ```Shell
      curl -o /etc/yum.repos.d/CentOS-Base.repo https://mirrors.aliyun.com/repo/Centos-7.repo
    ```
    
    ```Shell
      yum -y update
    
      reboot
    ```
    
    ```Shell
      yum clean
    
      yum makecache
    ```
    
# 2. **安装 Redis**
1. 安装Redis依赖：Redis是基于C语言编写的，因此首先需要安装Redis所需要的gcc依赖
    
    ```PowerShell
     yum install -y gcc tcl
    ```
    
1. 上传安装包并解压
    
    ```Shell
     [root@redis01 ~]# cd /usr/local/src/
     [root@redis01 src]# ls
     redis-6.2.6.tar.gz
    ```
    
    ```Shell
     [root@redis01 src]# tar -zxvf redis-6.2.6.tar.gz
     [root@redis01 src]# ls
     redis-6.2.6  redis-6.2.6.tar.gz
    ```
    
1. 进入redis目录，运行编译命令
    
    ```Shell
     [root@redis01 src]# cd redis-6.2.6
     [root@redis01 redis-6.2.6]# make && make install
    ```
    
1. 默认的安装路径是在 `/usr/local/bin`目录下，安装成后可以在这里看到命令
    
    ```Shell
     [root@redis01 src]# cd /usr/local/bin/
     [root@redis01 bin]# ll
     total 18904
     -rwxr-xr-x 1 root root 4829512 Apr 26 09:53 redis-benchmark
     lrwxrwxrwx 1 root root      12 Apr 26 09:53 redis-check-aof -> redis-server
     lrwxrwxrwx 1 root root      12 Apr 26 09:53 redis-check-rdb -> redis-server
     -rwxr-xr-x 1 root root 5003792 Apr 26 09:53 redis-cli
     lrwxrwxrwx 1 root root      12 Apr 26 09:53 redis-sentinel -> redis-server
     -rwxr-xr-x 1 root root 9518920 Apr 26 09:53 redis-server
    ```
    
    > 该目录下的命令会默认配置到系统环境变量中
    > 
    > 其中：
    > 
    > - `redis-cli`：是redis提供的命令行客户端
    > 
    > - `redis-server`：是redis的服务端启动脚本
    > 
    > - `redis-sentinel`：是redis的哨兵启动脚本
    
# 3. 可修改**配置文件**
1. 如果要让Redis以`后台`方式启动，则必须修改Redis配置文件
1. **配置文件位置：`/usr/local/src/redis-6.2.6/redis.conf`**
1. 配置文件备份：
    
    ```Shell
     cp redis.conf redis.conf.bak
    ```
    
1. 修改的配置项：
    
    1. 基础的修改项
        
        ```Shell
         # 监听的地址，默认是127.0.0.1，会导致只能在本地访问。修改为0.0.0.0则可以在任意IP访问，生产环境不要设置为0.0.0.0
         bind 0.0.0.0
         # 守护进程，修改为yes后即可后台运行
         daemonize yes
         # 密码，设置后访问Redis必须输入密码
         requirepass 123321
        ```
        
    
    1. 可选的修改项
        
        ```Shell
         # 监听的端口
         port 6379
         # 工作目录，默认是当前目录，也就是运行redis-server时的命令，日志、持久化等文件会保存在这个目录
         dir .
         # 数据库数量，设置为1，代表只使用1个库，默认有16个库，编号0~15
         databases 1
         # 设置redis能够使用的最大内存
         maxmemory 512mb
         # 日志文件，默认为空，不记录日志，可以指定日志文件名
         logfile "redis.log"
        ```
        
    
# 4. **启动 Redis**
1. 方式一：默认启动
    
    1. 安装完成后，在任意目录输入redis-server命令即可启动Redis：
        
        ```Shell
         redis-server
        ```
        
    
    1. 这种启动属于`前台启动`，会阻塞整个会话窗口，窗口关闭或者按下`CTRL + C`则Redis停止
    
    1. 不推荐使用
    
1. 方式二：指定配置启动
    
    1. 使用`redis-server` 命令，并指定配置文件 来启动 Redis 服务
        
        ```Bash
         redis-server redis.conf
        ```
        
    
    1. 利用`redis-cli`来执行 `shutdown` 命令，即可停止 Redis 服务
        
        ```Shell
         # 因为之前配置了密码，因此需要通过 -u 来指定密码
         redis-cli -u 123321 shutdown
        ```
        
    
1. 方式三：systemd 管理 Redis
    
    1. 新建一个系统服务文件
        
        ```Shell
         vi /etc/systemd/system/redis.service
        ```
        
        ```Shell
         [Unit]
         Description=redis-server
         After=network.target
         
         [Service]
         Type=forking
         ExecStart=/usr/local/bin/redis-server /usr/local/src/redis-6.2.6/redis.conf
         PrivateTmp=true
         
         [Install]
         WantedBy=multi-user.target
        ```
        
    
    1. 然后重载系统服务：
        
        ```PowerShell
         systemctl daemon-reload
        ```
        
        ```Shell
         systemctl start redis
         systemctl enable redis
         systemctl status redis
        ```
