# 一、搭建点评项目
项目架构（前后端分离）
![[IMG-20260620224127665.png|800]]
1. **构建按 mysql 数据库，执行sql文件**
    1. 数据库名：hmdp
    2. 其中的表有：
        - tb_user：用户表
        - tb_user_info：用户详情表
        - tb_shop：商户信息表
        - tb_shop_type：商户类型表
        - tb_blog：用户日记表（达人探店日记）
        - tb_follow：用户关注表
        - tb_voucher：优惠券表
        - tb_voucher_order：优惠券的订单表
2. **搭建SpringBoot后端项目**
    1. 导入项目
    2. 修改application配置文件（mysql 连接与 redis 连接）

    ```YAML
    server:
      port: 8081
    spring:
      application:
        name: hmdp
      datasource:
        driver-class-name: com.mysql.jdbc.Driver
        url: jdbc:mysql://127.0.0.1:3306/hmdp?useSSL=false&serverTimezone=UTC
        username: root
        password: ZXT774276296qq..
      redis:
        host: 192.168.10.151
        port: 6379
        password: 111111
        lettuce:
          pool:
            max-active: 10
            max-idle: 10
            min-idle: 1
            time-between-eviction-runs: 10s
      jackson:
        default-property-inclusion: non_null # JSON处理时忽略非空字段
    mybatis-plus:
      type-aliases-package: com.hmdp.entity # 别名扫描包
    logging:
      level:
        com.hmdp: debug
    ```
    3. 启动项目
    4. 访问测试：[http://localhost:8081/shop-type/list](http://localhost:8081/shop-type/list)
3. **启动前端 Nginx 项目，前端静态页面资源都放在 Nginx 代理服务器上**
    1. 进入项目目录下
    2. cmd

    ```Bash
    start nginx.exe
    ```
    3. 访问测试：[http://localhost:8080/](http://localhost:8080/)
