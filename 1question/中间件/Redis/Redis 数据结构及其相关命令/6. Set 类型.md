# 一、介绍
1. Redis 的 Set 结构与 Java 中的 ==HashSet== 类似[[Set 接口]]
2. 因为也是一个 hash 表，因此具备与 HashSet 类似的特征：
    - ==**无序**==
    - ==**元素不可重复**==
    - 查找快
    - 支持交集、并集、差集等功能

# 二、Set 的常见命令
- `SADD` key member ... ：向set中添加一个或多个元素

    ```Shell
     127.0.0.1:6379> SADD s1 a b c
     (integer) 3
    ```
- `SREM` key member ... : 移除set中的指定元素

    ```Shell
     127.0.0.1:6379> SREM s1 a
     (integer) 1
    ```
- `SCARD` key： 返回set中元素的个数

    ```Shell
     127.0.0.1:6379> SCARD s1
     (integer) 2
    ```
- `SISMEMBER` key member：判断一个元素是否存在于set中

    ```Shell
     127.0.0.1:6379> SISMEMBER s1 a
     (integer) 0
     127.0.0.1:6379> SISMEMBER s1 b
     (integer) 1
    ```
- `SMEMBERS`：获取set中的所有元素

    ```Shell
     127.0.0.1:6379> SMEMBERS s1
     1) "c"
     2) "a"
     3) "b"
    ```
    ![[IMG-20260405035438341.png|611]]
- `SINTER` key1 key2 ... ：
    - 求 key1 与 key2 的交集
    - ==**可以用来实现 共同关注**====[[6.2 共同关注]]====[[6.2 共同关注]]====[[6.2 共同关注]]==

    ```Shell
     127.0.0.1:6379> SADD s2 a b c
     (integer) 3
     127.0.0.1:6379> SADD s3 b c d
     (integer) 3
     127.0.0.1:6379> SINTER s2 s3
     1) "c"
     2) "b"
    ```
- `SDIFF` key1 key2 ... ：求key1与key2的差集

    ```Shell
     127.0.0.1:6379> SDIFF s2 s3
     1) "a"
     127.0.0.1:6379> SDIFF s3 s2
     2) "d"
    ```
- `SUNION` key1 key2 ..：求key1和key2的并集

    ```Shell
     127.0.0.1:6379> SUNION s2 s3
     1) "a"
     2) "c"
     3) "d"
     4) "b"
    ```

# 三、案例练习
- 将下列数据用Redis的Set集合来存储：
    - 张三的好友有：李四、王五、赵六
    - 李四的好友有：王五、麻子、二狗
- 利用Set的命令实现下列功能：
    - 计算张三的好友有几人
    - 计算张三和李四有哪些共同好友
    - 查询哪些人是张三的好友却不是李四的好友
    - 查询张三和李四的好友总共有哪些人
    - 判断李四是否是张三的好友
    - 判断张三是否是李四的好友
    - 将李四从张三的好友列表中移除

    ```Bash
     127.0.0.1:6379> SADD zs lisi wangwu zhaoliu
     (integer) 3
     127.0.0.1:6379> SADD ls wangwu mazi ergou
     (integer) 3
     127.0.0.1:6379> SCARD zs
     (integer) 3
     127.0.0.1:6379> SINTER zs ls
     1) "wangwu"
     127.0.0.1:6379> SDIFF zs ls
     2) "lisi"
     3) "zhaoliu"
     127.0.0.1:6379> SUNION zs ls
     4) "lisi"
     5) "wangwu"
     6) "zhaoliu"
     7) "mazi"
     8) "ergou"
     127.0.0.1:6379> SISMEMBER zs lisi
     (integer) 1
     127.0.0.1:6379> SISMEMBER ls zhangsan
     (integer) 0
     127.0.0.1:6379> SREM zs lisi
     (integer) 1
    ```
