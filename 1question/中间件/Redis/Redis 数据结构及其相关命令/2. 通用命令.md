> 💡 **通用命令可以操作任何数据类型的 key - value**
# 一、KEYS
- 查看符合模板的所有 key
- **不建议在生产环境设备上使用**
- 用法

    ```Bash
     127.0.0.1:6379> help KEYS
       KEYS pattern
       summary: Find all keys matching the given pattern
       since: 1.0.0
       group: generic
    ```
- 示例

    ```Shell
     127.0.0.1:6379> KEYS *
     1) "name"
     127.0.0.1:6379> KEYS n*     # 支持模糊匹配
     2) "name"
    ```
# 二、DEL
- 删除一个指定的key
- 用法

    ```Shell
     127.0.0.1:6379> help DEl
       DEL key [key ...]
       summary: Delete a key
       since: 1.0.0
       group: generic
    ```
- 示例

    ```Shell
     127.0.0.1:6379> MSET k1 v1 k2 v2 k3 v3
     OK
     127.0.0.1:6379> KEYS *
     1) "name"
     2) "k3"
     3) "k2"
     4) "k1"
     127.0.0.1:6379> DEL k1 k2
     (integer) 2
     127.0.0.1:6379> KEYS *
     5) "name"
     6) "k3"
    ```
# 三、EXISTS
- 判断key是否存在
- 用法

    ```Shell
     127.0.0.1:6379> help EXISTS
       EXISTS key [key ...]
       summary: Determine if a key exists
       since: 1.0.0
       group: generic
    ```
- 示例

    ```Shell
     127.0.0.1:6379> EXISTS k3
     (integer) 1
    ```
# 四、EXPIRE
- 给一个key设置有效期，有效期到期时该key会被自动删除
- 用法

    ```Shell
     127.0.0.1:6379> help EXPIRE
       EXPIRE key seconds
       summary: Set a key's time to live in seconds
       since: 1.0.0
       group: generic
    ```
- 示例

    ```Shell
     127.0.0.1:6379> KEYS *
     1) "name"
     2) "k3"
     127.0.0.1:6379> EXPIRE k3 5
     (integer) 1
     127.0.0.1:6379> KEYS *
     3) "name"
     4) "k3"
     127.0.0.1:6379> TTL k3
     (integer) -2
     127.0.0.1:6379> KEYS *
     5) "name"
    ```
# 五、TTL
- 查看一个KEY的剩余有效期
- 用法

    ```Shell
     127.0.0.1:6379> help TTL
       TTL key
       summary: Get the time to live for a key
       since: 1.0.0
       group: generic
    ```
- 示例

    ```Shell
     127.0.0.1:6379> TTL k3
     (integer) -2                # -2表示失效
     127.0.0.1:6379> KEYS *
     1) "name"
     127.0.0.1:6379> SET k4 v4
     OK
     127.0.0.1:6379> TTL k4
     (integer) -1                # -1表示永久有效
    ```
