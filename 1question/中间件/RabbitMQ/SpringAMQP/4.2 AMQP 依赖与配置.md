1. 在父工程mq-demo中引入依赖

    ```XML
    <!--AMQP依赖，包含RabbitMQ-->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-amqp</artifactId>
    </dependency>
    ```
2. 在publisher服务的`application.yml`中添加配置：

    ```YAML
     logging:
       pattern:
         dateformat: MM-dd HH:mm:ss:SSS
     spring:
       rabbitmq:
         host: 192.168.10.120 # rabbitMQ的ip地址
         port: 5672 # 端口
         username: itcast
         password: 123321
         virtual-host: /
    ```
3. 在consumer服务的`application.yml`中添加配置：

    ```YAML
     logging:
       pattern:
         dateformat: MM-dd HH:mm:ss:SSS
     spring:
       rabbitmq:
         host: 192.168.10.120 # rabbitMQ的ip地址
         port: 5672 # 端口
         username: itcast
         password: 123321
         virtual-host: /
         listener:
           simple:
             prefetch: 1 # ⭐每次只能获取一条消息，处理完才能获取下一个消息
    ```
    - `listener.simple.prefetch`：消息预取，详见 工作队列模型
