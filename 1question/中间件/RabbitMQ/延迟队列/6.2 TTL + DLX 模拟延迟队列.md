- 示例：利用 TTL 和 死信交换机 模拟延迟队列的效果
    

    ![[IMG-20260404031929496.png]]

    
    1. 声明`ttl.direct`交换机，和`ttl.queue`队列，它们之间使用`ttl`路由键绑定
    
    1. 声明`dl.direct`交换机作为死信交换机，和`dl.queue`队列，它们之间使用`dl`路由键绑定
    
    1. 给发送的消息设置了`TTL`为`5s`，队列`ttl.queue`设置了`TTL`为`10s`
    
    1. **当消息被发送至`ttl.queue`队列中后，并没有消费者取走，但由于设置了 `TTL` ，经过 `5s` 后，==该消息变为 死信==**
    
    1. **==死信消息会被投递到死信交换机==，进而被路由到`dl.queue`队列，而真正的==消费者监听的就是==`dl.queue`==队列==**
    
    1. 这样就实现了：**当消息被发送者发送后**`**5s**`**，消费者才会接受到消息，达到延时消息的效果**
    
# **a. 声明死信交换机、队列**

声明死信交换机、队列，**并让一个消费者监听死信队列**

```Java
 @Slf4j
 @Component
 public class SpringRabbitListener {
 
     @RabbitListener(bindings = @QueueBinding(
             value = @Queue(name = "dl.queue", durable = "true"),
             exchange = @Exchange(name = "dl.direct"),
             key = "dl"
     ))
     public void listenDlQueue(String msg){
         log.info("消费者收到了 dl.queue 中的延迟消息！");
     }
 }
```
# **b. 声明模拟的延迟队列**

声明死信交换机、队列，**并为其设置绑定死信交换机**

```Java
 @Configuration
 public class TTLMessageConfig {
 
     @Bean
     public DirectExchange TTLExchange(){
         return new DirectExchange("ttl.direct");
     }
 
     @Bean
     public Queue TTLQueue(){
         return QueueBuilder
                 .durable("ttl.queue")
                 .ttl(10000)                         //⭐设置队列的TTL
                 .deadLetterExchange("dl.direct")    //⭐给队列设置死信队列
                 .deadLetterRoutingKey("dl")         //⭐使用的路由键
                 .build();
     }
 
     @Bean
     public Binding TTLBinding(){
         return BindingBuilder.bind(TTLQueue()).to(TTLExchange()).with("ttl");
     }
 }
```
# **c. 消息发送**
```Java
 @Test
 public void testTTLMessage(){
     //定义一个消息对象
     Message message = MessageBuilder
             .withBody("test ttl Message".getBytes(StandardCharsets.UTF_8))
             .setDeliveryMode(MessageDeliveryMode.PERSISTENT)
             .setExpiration("5000")                                              //⭐设置消息的TTL
             .build();
     //发送消息
     rabbitTemplate.convertAndSend("ttl.direct","ttl",message);
 }
```
