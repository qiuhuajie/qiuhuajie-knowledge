1. 要实现根据字母做补全，就必须对文档按照拼音分词。在GitHub上恰好有elasticsearch的拼音分词插件
2. 地址：[https://github.com/medcl/elasticsearch-analysis-pinyin](https://github.com/medcl/elasticsearch-analysis-pinyin)
3. 安装方式与IK分词器一样，分三步：
    1. 解压
    2. 上传到虚拟机中，elasticsearch的plugin目录

    ```Bash
     [root@zkSpringCloud ~]# cd /var/lib/docker/volumes/es-plugins/_data
     [root@zkSpringCloud _data]# ls
     ik  py
     [root@zkSpringCloud _data]# cd py
     [root@zkSpringCloud py]# ls
     elasticsearch-analysis-pinyin-7.12.1.jar  nlp-lang-1.7.jar  plugin-descriptor.properties
     [root@zkSpringCloud py]# pwd
     /var/lib/docker/volumes/es-plugins/_data/py
    ```
    3. 重启elasticsearch
