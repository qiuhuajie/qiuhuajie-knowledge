# 一、酒店数据准备
1. 之前的酒店数据索引库没有自动补全功能，现在需要删除，并重新创建
2. 实现思路如下：
    1. 修改hotel索引库结构，设置自定义拼音分词器
    2. 修改索引库的name、all字段，使用自定义分词器
    3. **索引库添加一个新字段suggestion，类型为completion类型，使用自定义的分词器**
    4. 给HotelDoc类添加suggestion字段，内容包含brand、business
    5. 重新导入数据到hotel库
3. 步骤：
    1. 删除原先索引库

    ```Plain
     DELETE /hotel
    ```
    2. 创建新的索引库

    3. 给HotelDoc类添加suggestion字段，内容包含brand、business

    ```Java
     @Data
     @NoArgsConstructor
     @AllArgsConstructor
     public class HotelDoc {
         private Long id;
         private String name;
         private String address;
         private Integer price;
         private Integer score;
         private String brand;
         private String city;
         private String starName;
         private String business;
         private String location;
         private String pic;
         private Object distance;
         private String isAD;
         private List<String> suggestion;    //现有字段信息（需要实现补全的）组成的数组
         public HotelDoc(Hotel hotel) {
             this.id = hotel.getId();
             this.name = hotel.getName();
             this.address = hotel.getAddress();
             this.price = hotel.getPrice();
             this.score = hotel.getScore();
             this.brand = hotel.getBrand();
             this.city = hotel.getCity();
             this.starName = hotel.getStarName();
             this.business = hotel.getBusiness();
             this.location = hotel.getLatitude() + ", " + hotel.getLongitude();
             this.pic = hotel.getPic();
             if (this.business.contains("/")) {
                 String[] strings = business.split("/");
                 this.suggestion = new ArrayList<String>();
                 suggestion.add(this.brand);
                 Collections.addAll(this.suggestion,strings);
             }else {
                 this.suggestion = Arrays.asList(this.brand,this.business);  //suggestion由需要自动补全功能的字段拼接而成
             }
         }
     }
    ```

    ```Java
     @Test

     }
    ```

# 二、自动补全 RestClient 测试

编写测试类，看是否可以使用RestClient实现自动补全搜索

```Java
 public class HotelSearchTest {
     private RestHighLevelClient client;
     @BeforeEach
     void setUp(){
         this.client = new RestHighLevelClient(RestClient.builder(HttpHost.create("http://192.168.10.123:9200")));
     }
     @AfterEach
     void tearDown() throws IOException {
         this.client.close();
     }
     @Test
     void testSuggestion() throws IOException {
         //1. 准备 request
         SearchRequest request = new SearchRequest("hotel");
         //2. 准备 DSL，设置请求参数
         request.source().suggest(new SuggestBuilder().addSuggestion(
                 "hotel_suggestion",
                 SuggestBuilders
                         .completionSuggestion("suggestion")
                         .prefix("s")
                         .skipDuplicates(true).size(10)
         ));
         //3. 发送请求
         SearchResponse response = client.search(request, RequestOptions.DEFAULT);
         //4. 解析响应
         Suggest suggest = response.getSuggest();
         //4.1 根据名字获取补全结果
         CompletionSuggestion suggestion = suggest.getSuggestion("hotel_suggestion");
         //4.2 获取options，并遍历
         List<CompletionSuggestion.Entry.Option> options = suggestion.getOptions();
         for (CompletionSuggestion.Entry.Option option : options) {
             String text = option.getText().string();
             System.out.println(text);
             /*三里屯
             上地产业园
             上海火车站地区
             四川北路商业区
             松岗商业中心区
             水贝珠宝城
             沙头角
             深圳北站地区
             蛇口
             首都机场*/
         }
     }
 }
```

# 三、接口实现
1. 查看前端页面，可以发现当我们在输入框键入时，前端会发起ajax请求：

    ![[IMG-20260405035507278.png|620]]

2. 在服务端编写接口，接收该请求，返回补全结果的集合，类型为List<String>
    1. controller

    ```Java
     @GetMapping("/suggestion")
     public List<String> getSuggestion(@RequestParam("key") String prefix){
         return hotelService.getSuggestion(prefix);
     }
    ```
    2. service

    ```Java
     @Override
     public List<String> getSuggestion(String prefix) {
         try {
             //1. 准备 request
             SearchRequest request = new SearchRequest("hotel");
             //2. 准备 DSL，设置请求参数
             request.source().suggest(new SuggestBuilder().addSuggestion(
                     "hotel_suggestion",
                     SuggestBuilders
                             .completionSuggestion("suggestion")
                             .prefix(prefix)
                             .skipDuplicates(true).size(10)
             ));
             //3. 发送请求
             SearchResponse response = client.search(request, RequestOptions.DEFAULT);
             //4. 解析响应
             ArrayList<String> result = new ArrayList<>();
             Suggest suggest = response.getSuggest();
             //4.1 根据名字获取补全结果
             CompletionSuggestion suggestion = suggest.getSuggestion("hotel_suggestion");
             //4.2 获取options，并遍历
             List<CompletionSuggestion.Entry.Option> options = suggestion.getOptions();
             for (CompletionSuggestion.Entry.Option option : options) {
                 String text = option.getText().string();
                 result.add(text);
             }
             return result;
         } catch (IOException e) {
             throw new RuntimeException(e);
         }
     }
    ```
