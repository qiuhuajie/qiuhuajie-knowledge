    ![[IMG-20260620224143007.png|800]]
2. 定义**参数实体类**：接收前端请求参数

    ```Java
     @Data
     public class RequestParams {
         private String key;
         private Integer page;
         private Integer size;
         private String sortBy;
     }
    ```
3. 定义**结果实体类**：

    ```Java
     @Data
     @AllArgsConstructor
     @NoArgsConstructor
     public class PageResult {
         private Long total;            //结果总数
         private List<HotelDoc> hotels;  //结果集
     }
    ```
