# 一、操作说明
⭐文档操作的基本步骤：
- 初始化`RestHighLevelClient`
- 创建`XxxRequest`。XXX是Index、Get、Update、Delete
- 准备参数（Index和Update时需要）
- 发送请求。调用`RestHighLevelClient.xxx()`方法，xxx是index、get、update、delete
- 解析结果（Get时需要）
# 二、创建文档
> 示例：添加酒店数据到索引库
1. 先查询酒店数据，然后给这条数据创建倒排索引，即可完成添加

    ![[IMG-20260620224142969.png|800]]
2. 代码：

    ```Java
     @SpringBootTest
     public class HotelDocumentTest {
         @Autowired
         private HotelService hotelService;
         private RestHighLevelClient client;
         @BeforeEach
         void setUp(){
             this.client = new RestHighLevelClient(RestClient.builder(HttpHost.create("http://192.168.10.123:9200")));
         }
         @AfterEach
         void tearDown() throws IOException {
             this.client.close();
         }
         @Test
         void creatHotelDocument() throws IOException {
             //先从数据库中查出数据
             Hotel hotel = hotelService.getById(61083L);
             //由于数据库里的字段与所需ES文档字段有差异，所以需要转换
             HotelDoc hotelDoc = new HotelDoc(hotel);
             //1. 创建 Request 对象
             IndexRequest request = new IndexRequest("hotel").id(hotelDoc.getId().toString());
             //2. 准备请求参数：JSON 文档内容
             request.source(JSON.toJSONString(hotelDoc), XContentType.JSON);
             //3. 发送请求
             client.index(request, RequestOptions.DEFAULT);
         }
     }
    ```
3. 测试验证：DevTools

    ```Bash
     GET /hotel/_doc/61083
    ```

# 三、批量导入数据到 ES
1. 需求：批量查询酒店数据，然后批量导入索引库中
2. 思路：
    1. 利用`mybatis-plus`查询酒店数据
    2. 将查询到的酒店数据（Hotel）转换为文档类型数据（HotelDoc）
    3. 利用`JavaRestClient`中的`Bulk`批处理，实现批量新增文档，示例代码如下

    ![[IMG-20260620224159790.png|778]]
3. 代码：

    ```Java
     @Test
     void bulkHotelDocument() throws IOException {
         //1. 创建 Request 对象
         BulkRequest request = new BulkRequest();
         //2. 准备参数，添加多个 request
         //2.1 从数据库中查出所有酒店数据
         List<Hotel> hotels = hotelService.list();
         //2.2 转换数据
         for (Hotel hotel : hotels) {
             HotelDoc hotelDoc = new HotelDoc(hotel);
             request.add(new IndexRequest("hotel")
                     .id(hotelDoc.getId().toString())
                     .source(JSON.toJSONString(hotelDoc),XContentType.JSON));
         }
         //3. 发送请求
         client.bulk(request, RequestOptions.DEFAULT);
     }
    ```
4. 测试验证：DevTools

    ```Plain
     # 查询所有
     GET /hotel/_search
    ```
# 四、查看文档
> 示例：添加酒店数据到索引库
1. 根据id查询到的文档数据是json，需要反序列化为java对象：

    ![[IMG-20260620224206146.png|800]]
2. 代码：

    ```Java
     @SpringBootTest
     public class HotelDocumentTest {
         private RestHighLevelClient client;
         @BeforeEach
         void setUp(){
             this.client = new RestHighLevelClient(RestClient.builder(HttpHost.create("http://192.168.10.123:9200")));
         }
         @AfterEach
         void tearDown() throws IOException {
             this.client.close();
         }
         @Test
         void getHotelDocument() throws IOException {
             //1. 创建 Request 对象
             GetRequest request = new GetRequest("hotel").id("61083");
             //2. 发送请求
             GetResponse response = client.get(request, RequestOptions.DEFAULT);
             //3. 解析响应结果
             String json = response.getSourceAsString();
             //4. 查询结果反序列化
             HotelDoc hotelDoc = JSON.parseObject(json, HotelDoc.class);
             System.out.println(hotelDoc);
         }
     }
    ```
# 五、修改文档
> 示例：添加酒店数据到索引库
修改文档数据有两种方式：
- 方式一：**全量更新**。再次写入id一样的文档，就会删除旧文档，添加新文档
- 方式二：**局部更新**。只更新部分字段，我们演示方式二

    ![[IMG-20260620224211721.png|800]]
    ```Java
     @Test
     void updateHotelDocument() throws IOException {
         //1. 创建 Request 对象
         UpdateRequest request = new UpdateRequest("hotel","61083");
         //2. 准备参数
         request.doc(
                 "price","11111",
                 "starName","四钻"
         );
         //3. 发送请求
         client.update(request, RequestOptions.DEFAULT);
     }
    ```
