# 一、概述
1. 全称 **Structured Query Language**，结构化查询语言
2. 是操作关系型数据库的编程语言，定义了一套操作关系型数据库统一标准
3. **SQL 语句不区分大小写**，但是数据库表名、列名和值是否区分，依赖于具体的 DBMS 以及配置
# 二、SQL语句的分类
1. DQL：
    1. 数据查询语言，Data Query Language ，数据查询语言，**用来查询数据库中表的记录**
    2. 凡是带有select关键字的都是查询语句：`select...`
2. DML：
    1. 数据操作语言，Data Manipulation Language，数据操作语言，用来**对数据库表中的数据**进行增删改
    2. 凡是对表当中的数据进行增删改的都是DML：`insert` 、`delete`、`update`主要是操作表中的数据data
3. DDL：
    1. 数据定义语言，Data Definition Language，用来定义数据库对象，如数据库，表， 字段
    2. 凡是带有`create`、`drop`、`alter`的都是DDL，**DDL主要操作的是表的结构，不是表中的数据**
4. TCL：事务控制语言，包括：
    1. 事务提交：`commit`
    2. 事务回滚：`rollback`
5. DCL：
    1. 数据控制语言，Data Control Language，**用来创建数据库用户、控制数据库的访问权限**
    2. 例如：
        1. 授权`grant`
        2. 撤销权限`revoke`
