1. **返回每个顾客不同订单的总金额**
    
    ![[IMG-20260405035438315.png|Untitled 231.png]]
    
    ```SQL
    SELECT
        o.cust_id cust_id,
        tb.total_ordered total_ordered
    FROM (
        SELECT order_num, SUM(item_price * quantity) total_ordered
        FROM OrderItems
        GROUP BY order_num
        ) as tb,
        Orders o
    WHERE
        tb.order_num = o.order_num
    ORDER BY
        total_ordered DESC;
    ```
    
1. **从 Products 表中检索所有的产品名称以及对应的销售总数**
    
    ![[IMG-20260405035451120.png|Untitled 1 164.png]]
    
```SQL
select p.prod_name, tb.quant_sold
from (
    select prod_id, sum(quantity) as quant_sold
    from OrderItems
    group by prod_id
    ) as tb,
    Products p
where tb.prod_id = p.prod_id;
```

1. 案例一：找出每个员工的部门名称以及工资等级，要求显示员工名、部门名、薪资、薪资等级
    
    ```Plain
     select
         e.ename,e.sal,d.dname,s.grade
     from
         emp e
     join
         dept d
     on
         e.deptno = d.deptno
     join
         salgrade s
     on
         e.sal between s.losal and s.hisal;
    ```
    
1. 案例二：找出每个员工的部门名称以及工资等级，还有上级领导，要求显示员工名、领导名、部门名、薪资、薪资等级
    
    ```Plain
     select
         e.ename,e.sal,d.dname,s.grade,l.ename
     from
         emp e
     join
         dept d
     on
         e.deptno = d.deptno
     join
         salgrade s
     on
         e.sal between s.losal and s.hisal
     left join
         emp l
     on
         e.mgr = l.empno;
    ```
