### `**format**` **设置千分位**
格式化数字：format(数字, '格式')
```Plain
 select ename,format(sal, '$999,999') as sal from emp;
         +--------+-------+
         | ename  | sal   |
         +--------+-------+
         | ALLEN  | 1,600 |
         | WARD   | 1,250 |
         +--------+-------+
```
### `**round**` **四舍五入**

> select后面可以跟某个表的字段名（可以等同看做变量名），也可以跟字面量/字面值（数据）
> 
> ```Plain
>  mysql> select 'abc' as bieming from emp;    \#按数据搜索需要使用单引号，否则会视为字段名
>      +---------+
>      | bieming |
>      +---------+
>      | abc     |
>      | abc     |
>      | abc     |
>      | abc     |
>      +---------+
>  select 1000 as num from emp;    # 1000 也是被当做一个字面量/字面值。
>      +------+
>      | num  |
>      +------+
>      | 1000 |
>      | 1000 |
>      | 1000 |
>      +------+
> ```
```Plain
 select round(1236.567, 0) as result from emp;   \#保留整数位。
 select round(1236.567, 1) as result from emp;   \#保留1个小数
 select round(1236.567, 2) as result from emp;   \#保留2个小数
 select round(1236.567, -1) as result from emp;  \#保留到十位
     +--------+
     | result |
     +--------+
     |   1240 |
     |   1240 |
     |   1240 |
     +--------+
```
### `**rand**` **生成随机数**
```Plain
 select round(rand()*100,0) from emp;        \#100以内的随机数
```
