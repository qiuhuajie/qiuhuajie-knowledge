### `**lower**` **转换小写**
```Plain
 mysql> select lower(ename) as ename from emp;
```
### `**upper**` **转换大写**
```SQL
 mysql> select upper(name) as name from t_student;
```
例题：**==将首字母大写==**
```SQL
 \#将第一字母大写
 select upper(substr(name,1,1)) from t_student;
 \#搜出除第一个字母的其他字母
 select substr(name,2,length(name) - 1) from t_student;
 \#concat合并
 select concat(upper(substr(name,1,1)),substr(name,2,length(name) - 1)) as result from t_student;
     +----------+
     | result   |
     +----------+
     | Jack_son |
     +----------+
```
### `**substr**` **取子串**
```SQL
# substr( 被截取的字符串, 起始下标,截取的长度) 
select substr(ename, 1, 1) as ename from emp;
```
1. 注意：起始下标从1开始，没有0
1. 找出员工名字第一个字母是A的员工信息
    
    1. 第一种方式：模糊查询
        
        ```SQL
         select ename from emp where ename like 'A%';
        ```
        
    
    1. 第二种方式：substr函数
        
        ```SQL
         select
             ename
         from
             emp
         where
             substr(ename,1,1) = 'A';
        ```
        
    
### `**CONCAT(s1,s2,...)**` **拼接**
```Plain
 select concat(empno,ename) from emp;
```
### `**CONCAT_WS(x, s1,s2,...,sn)**`
1. 同 `**CONCAT(s1,s2,...)**` 函数，但是每个字符串之间要加上 `**x**`，`**x**` 可以是分隔符
1. 合并多个字符串，并添加分隔符
    
    ```Plain
     SELECT CONCAT_WS("-", "SQL", "Tutorial", "is", "fun!")AS ConcatenatedString;
    ```
