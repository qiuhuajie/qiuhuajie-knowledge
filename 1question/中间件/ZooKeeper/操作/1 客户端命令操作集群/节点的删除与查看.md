# 一、节点的删除与查看
1. 删除节点

    ```Bash
     [zk: localhost:2181(CONNECTED) 2] delete /sanguo/jin
     [zk: localhost:2181(CONNECTED) 4] ls /sanguo
     [shuguo, weiguo]
    ```
2. 递归删除节点

    ```Bash
     [zk: localhost:2181(CONNECTED) 6] deleteall /sanguo/shuguo
     [zk: localhost:2181(CONNECTED) 7] ls /sanguo
     [weiguo]
    ```
3. 查看节点状态

    ```Bash
     [zk: localhost:2181(CONNECTED) 8] stat /sanguo
     cZxid = 0x200000002
     ctime = Thu Aug 19 20:41:23 CST 2021
     mZxid = 0x20000000f
     mtime = Thu Aug 19 22:10:01 CST 2021
     pZxid = 0x200000014
     cversion = 9
     dataVersion = 2
     aclVersion = 0
     ephemeralOwner = 0x0
     dataLength = 4
     numChildren = 1
    ```
