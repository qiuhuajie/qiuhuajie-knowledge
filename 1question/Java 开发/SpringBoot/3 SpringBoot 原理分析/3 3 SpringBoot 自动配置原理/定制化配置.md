# 1. 方式一：自己写配置类，自己往里添加想要的组件⭐
1. 编写自定义的配置类 xxxConfiguration + @Bean，用于**替换**、**增加**容器中默认组件
1. 举例：2.8.4 中 自定义实现 HandlerExceptionResolver 处理异常 的方式
1. 示例：
    
    ```Java
     @Configuration(proxyBeanMethods = true)
     public class MyConfig {
     
         //上面例子中是自动配置的字符过滤器
         //这个是自定义组件配置的字符过滤器（这个会优先生效）
         @Bean
         public CharacterEncodingFilter filter(){
             return null;
         }
     }
    ```
    
# 2. **方式二：**给容器中再扩展一些组件⭐
1. 编写一个配置类**实现 WebMvcConfigurer 接口** 即可定制化web功能
1. 举例：[[4.6 拦截器]]
    
    在配置类中，将上面的拦截器 注册到 WebMvcConfigurer 容器中
    
    ```Java
     @Configuration
     public class AdminWebConfig implements WebMvcConfigurer {
         @Override
         public void addInterceptors(InterceptorRegistry registry) {
             registry.addInterceptor(new LoginIntercepter())
                     .addPathPatterns("/**")
                 .excludePathPatterns("/","/login","/css/**","/fonts/**","/images/**","/js/**");
         }
     }
    ```
