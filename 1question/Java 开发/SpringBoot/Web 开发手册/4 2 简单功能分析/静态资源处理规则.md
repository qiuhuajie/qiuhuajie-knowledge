# 一、静态资源处理规则
1. SpringBoot启动默认加载 xxxAutoConfiguration 类（自动配置类）

    ![[IMG-20260405035413996.png|624]]

2. SpringMVC功能的自动配置类 WebMvcAutoConfiguration（生效）

    ![[IMG-20260404031830276.png|800]]

3. WebMvcAutoConfiguration 配置类

    ```Java
     @Configuration(
         proxyBeanMethods = false
     )
     @ConditionalOnWebApplication(
         type = Type.SERVLET
     )
     //条件装配（可以实现 WebMvcConfigurationSupport.class 来代替自动的mvc配置）
     @ConditionalOnClass({Servlet.class, DispatcherServlet.class, WebMvcConfigurer.class})
     @ConditionalOnMissingBean({WebMvcConfigurationSupport.class})
     @AutoConfigureOrder(-2147483638)
     @AutoConfigureAfter({DispatcherServletAutoConfiguration.class, TaskExecutionAutoConfiguration.class, ValidationAutoConfiguration.class})
     public class WebMvcAutoConfiguration {
        ...
     }
    ```
4. 在 WebMvcAutoConfiguration容器中 又配置了 WebMvcAutoConfigurationAdapter（也是一个配置类）

    ```Java
     @Configuration(
         proxyBeanMethods = false
     )
     @Import({WebMvcAutoConfiguration.EnableWebMvcConfiguration.class})
     //与配置文件进行绑定
     //与WebMvcProperties.class 绑定的前缀：spring.mvc
     //与ResourceProperties.class 绑定的前缀：spring.resources
     @EnableConfigurationProperties({WebMvcProperties.class, ResourceProperties.class})
     @Order(0)
     public static class WebMvcAutoConfigurationAdapter implements WebMvcConfigurer {
             ...
     }
    ```
    > 扩展知识：
    > 
    > 如果一个配置类只有一个有参构造器，那么所有有参构造器参数的值都会从容器中确定

    ```Java
     // WebMvcAutoConfigurationAdapter 只有一个有参构造器
     public WebMvcAutoConfigurationAdapter(ResourceProperties resourceProperties, WebMvcProperties mvcProperties, ListableBeanFactory beanFactory, ObjectProvider<HttpMessageConverters> messageConvertersProvider, ObjectProvider<WebMvcAutoConfiguration.ResourceHandlerRegistrationCustomizer> resourceHandlerRegistrationCustomizerProvider, ObjectProvider<DispatcherServletPath> dispatcherServletPath, ObjectProvider<ServletRegistrationBean<?>> servletRegistrations) {
         this.resourceProperties = resourceProperties;
         this.mvcProperties = mvcProperties;
         this.beanFactory = beanFactory;
         this.messageConvertersProvider = messageConvertersProvider;
         this.resourceHandlerRegistrationCustomizer = (WebMvcAutoConfiguration.ResourceHandlerRegistrationCustomizer)resourceHandlerRegistrationCustomizerProvider.getIfAvailable();
         this.dispatcherServletPath = dispatcherServletPath;
         this.servletRegistrations = servletRegistrations;
     }
    ```

    **构造器参数：**

    1. **`ResourceProperties resourceProperties`**：获取和spring.resources绑定的所有的值的对象
    2. **`WebMvcProperties mvcProperties`**：获取和spring.mvc绑定的所有的值的对象
    3. **`ListableBeanFactory beanFactory`**：获取Spring的beanFactory IOC容器
    4. **`HttpMessageConverters`** messageConvertersProvider
    5. **`WebMvcAutoConfiguration.ResourceHandlerRegistrationCustomizer`** resourceHandlerRegistrationCustomizer：获取资源处理器的自定义器
    6. **`DispatcherServletPath`** dispatcherServletPath：
    7. **`ServletRegistrationBean`** servletRegistrations：给应用注册Servlet、Filter...
5. 所有**资源处理的默认规则**都在：WebMvcAutoConfiguration 配置类里的 addResourceHandlers

    ```Java
     public void addResourceHandlers(ResourceHandlerRegistry registry) {
         //判断 resourceProperties.isAddMappings() 是false 则会认为默认资源失效
         //resourceProperties是与 spring.resources前缀 绑定的
         //所以可以使用 spring.resources.isAddMappings = false 来禁用默认
         if (!this.resourceProperties.isAddMappings()) {
             logger.debug("Default resource handling disabled");
         } else {
             //获取缓存策略（静态资源存多少秒）
             Duration cachePeriod = this.resourceProperties.getCache().getPeriod();
             CacheControl cacheControl = this.resourceProperties.getCache().getCachecontrol().toHttpCacheControl();
             if (!registry.hasMappingForPattern("/webjars/**")) {
                 this.customizeResourceHandlerRegistration(registry.addResourceHandler(new String[]{"/webjars/**"}).addResourceLocations(new String[]{"classpath:/META-INF/resources/webjars/"}).setCachePeriod(this.getSeconds(cachePeriod)).setCacheControl(cacheControl));
             }
             //mvcProperties.getStaticPathPattern
             //从配置文件 找到spring.mv前缀下的静态资源路径
             //如果没有配置，默认是"/**"
             String staticPathPattern = this.mvcProperties.getStaticPathPattern();
             //使用 getStaticLocations() 获取到 staticPathPattern
             if (!registry.hasMappingForPattern(staticPathPattern)) {
                 this.customizeResourceHandlerRegistration(registry.addResourceHandler(new String[]{staticPathPattern}).addResourceLocations(WebMvcAutoConfiguration.getResourceLocations(this.resourceProperties.getStaticLocations())).setCachePeriod(this.getSeconds(cachePeriod)).setCacheControl(cacheControl));
             }
         }
     }
    ```
