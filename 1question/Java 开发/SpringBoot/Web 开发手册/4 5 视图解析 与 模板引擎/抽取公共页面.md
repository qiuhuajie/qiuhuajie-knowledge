# 一、抽取公共页面
1. 每个页面都有公共侧边导航栏：其中每个按钮对跳转一个页面，但是，每页面里都重复写了一遍侧边导航栏的代码
2. **如果要改某一个页面的 跳转目标 则需要去每个页面去改，很麻烦**，需要将每个页面的公共部分提出来
3. 新建一个新的common.html

    **将公共部分的代码抽取出来，放在一个<div>中，并给它一个标识**：

```HTML
 <!---方式一：-->
 <div id="common-left-side">
 <!---方式二：-->
 <div th:fragment="common-left-side">
```
```HTML
 <!--common.html-->
 <!DOCTYPE html>
 <!--引入 th 名称空间-->
 <html lang="en" xmlns:th="http://www.thymeleaf.org">
 <html lang="en">
 <!--公共head-->
 <head th:fragment="common-head">
         <link th:href="@{/css/style.css}" rel="stylesheet">
         <link th:href="@{/css/style-responsive.css}" rel="stylesheet">
         <!--[if lt IE 9]>
         <script th:src="@{/js/html5shiv.js}"></script>
         <script th:src="@{/js/respond.min.js}"></script>
         <![endif]-->
 </head>
 <body>
     <!-- 公共左侧边栏-->
     <div id="common-left-side" class="left-side sticky-left-side">
                 ...
     </div>
     <!-- 公共顶部 header-->
     <div id="common-header" class="header-section">
                 ...
     </div>
     <!-- 公共 js -->
     <div id="common-script">
         <script th:src="@{/js/jquery-1.10.2.min.js}"></script>
         <script th:src="@{/js/jquery-ui-1.9.2.custom.min.js}"></script>
     </div>
 </body>
 </html>
```
4. 原前端页面 main.html

    **在原页面 引入 提出的公共部分的方式**，引入的方式有三种：

    1. th:insert：是最简单的：它将简单地插入指定的片段作为其宿主标记的主体
    2. th:replace：实际上用指定的片段替换它的主机标签
    3. th:include：类似于th:insert，但不是插入片段，而是仅插入该片段的内容。（不推荐）

    ```HTML
     格式：
     <div th:replace="新的公共页面名 :: 公共页面中的需要引入的各个部分标识"></div>
     示例：
     <div th:replace="common :: \#common-left-side"></div>
     标识方法一：id  ----   引入时：\#common-left-side
     标识方法二：th:fragment  ----   引入时：common-left-side
    ```
    ```HTML
     <!DOCTYPE html>
     <html lang="en" xmlns:th="http://www.thymeleaf.org">
     <html lang="en">
     <head>
             <!--...其他非公共部分保留-->
         <!--将公共head部分引入-->
         <div th:replace="common :: common-head"></div>
     </head>
     <body class="sticky-header">
     <section>
         <!-- 引入公共左侧边栏-->
         <div th:replace="common :: \#common-left-side"></div>
         <div class="main-content" >
             <!-- 引入公共header-->
             <div th:replace="common :: \#common-header"></div>
             <!--...其他非公共部分保留-->
         </div>
     </section>
     <!--引入公共script-->
     <div th:replace="common :: \#common-script"></div>
     <!--非公共script 需要保留-->
     ...
     </body>
     </html>
    ```
5. 处理页面跳转的controller

    ```Java
     @Controller
     public class tableController {
         @GetMapping("/basic_table")
         public String basic_table(){
             return "table/basic_table";
         }
         @GetMapping("/dynamic_table")
         public String dynamic_table(){
             return "table/dynamic_table";
         }
         @GetMapping("/responsive_table")
         public String responsive_table(){
             return "table/responsive_table";
         }
         @GetMapping("/editable_table")
         public String editable_table(){
             return "table/editable_table";
         }
     }
    ```
