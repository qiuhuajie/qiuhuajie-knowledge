# 一、导入 jdbc 场景 starter
```XML
 <dependency>
     <groupId>org.springframework.boot</groupId>
     <artifactId>spring-boot-starter-data-jdbc</artifactId>
 </dependency>
```
1. spring-boot-starter-data-jdbc 自动导入了一些包
    - **默认导入的是** ==**HikariDataSource 连接池**== （性能最好）

    ![[Attachment/1question/大数据/Java 开发/SpringBoot/5 数据访问/IMG-20260405035413873.png|409]]
# 二、数据库驱动
1. 引入MySql 依赖

    ```XML
     <dependency>
         <groupId>mysql</groupId>
         <artifactId>mysql-connector-java</artifactId>
     </dependency>
    ```
2. 默认的版本（底层版本仲裁时 8.0.22）

    ```XML
     <version>8.0.22</version>
    ```
3. 使用想要的 mysql 版本
    1. 方式1：

    ```XML
     <dependency>
         <groupId>mysql</groupId>
         <artifactId>mysql-connector-java</artifactId>
         <version>8.0.21</version>
     </dependency>
    ```
    2. 方式2：

    ```XML
     <properties>
         <java.version>1.8</java.version>
         <mysql-version>8.0.21</mysql-version>
     </properties>
    ```
# 三、测试
1. 写数据库连接的配置

    ```YAML
     spring:
       datasource:
         url: jdbc:mysql://localhost:3306/userdb?useSSL=false
         username: root
         password: ZXT774276296qq..
         driver-class-name: com.mysql.jdbc.Driver
       jdbc:
         template:
           # 配置 jdbctemplate ，在查询时的超时时间
           query-timeout: 3
     #    当前默认的就是hikari数据连接池
     #    type: com.zaxxer.hikari.HikariDataSource
    ```
    > 加 ?useSSL=false 这个参数，是因为在启动时会报警告（但也能启动）
    >
    > ```Plain
    >  Mon Feb 14 00:33:57 CST 2022 WARN: Establishing SSL connection without server's identity verification is not recommended. According to MySQL 5.5.45+, 5.6.26+ and 5.7.6+ requirements SSL connection must be established by default if explicit option isn't set. For compliance with existing applications not using SSL the verifyServerCertificate property is set to 'false'. You need either to explicitly disable SSL by setting useSSL=false, or set useSSL=true and provide truststore for server certificate verification.
    > ```
2. 启动项目看是否配置好

    ![[IMG-20260404031835596.png|800]]
3. 编写测试代码

    ```Java
     @Slf4j
     @SpringBootTest
     class SpringBootAdminApplicationTests {
         //容器中此时已经有了 jdbcTemplate 组件了，直接注入即可
         @Autowired
         JdbcTemplate jdbcTemplate;
         @Test
         void contextLoads() {
             String sql = "select count(*) from t_book";
             Long aLong = jdbcTemplate.queryForObject(sql, Long.class);
             log.info("表中一共有{}条数据",aLong);
             //表中一共有8条数据
         }
     }
    ```
