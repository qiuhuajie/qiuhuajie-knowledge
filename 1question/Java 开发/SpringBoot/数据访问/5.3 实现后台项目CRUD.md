> 使用 MyBatis-PLus
1. 项目结构：
    > 整合mybatis-plus 后 可以不在使用 sql 映射文件了（简单sql功能情况下）

2. 实体类

    ```Java
     @NoArgsConstructor
     @AllArgsConstructor
     @Data
     //如果表面和类名不相同，在这里可以指定该类是与哪个表对应
     @TableName("user")
     public class User {
         //由于表中没有这个 字段，但是登陆时还需要使用
         //使用TableField注解=false，表示这个字段不在表中
         @TableField(exist = false)
         private String userName;
         @TableField(exist = false)
         private String password;
         private Long id;
         private String name;
         private Integer age;
         private String email;
     }
    ```
3. mapper

    ```Java
     //这里 默认是去 数据库中找名为 user 的表操作
     public interface UserMapper extends BaseMapper<User> {
         //只需要继承 mybatis 提供的 BaseMapper，这个mapper就有了基本的增删改查功能
         //对于基础的操作，无需在写抽象方法
         //但复杂的，还是是要写 sql 映射
     }
    ```
4. service 接口

    ```Java
     //service 接口继承 mybatisplus 提供的 IService ，泛型中添加的类型就是 该service 操作的 对象
     public interface UserService extends IService<User> {
     }
    ```
5. service 实现类

    ```Java
     //service 实现类继承 mybatisplus 提供的 ServiceImpl ，泛型中添加的类型就是 该service 操作的 对象 和 操作对象的 mapper
     @Service
     public class UserServiceImpl extends ServiceImpl<UserMapper, User> implements UserService {
     }
    ```
6. mybatis 配置类**（配置分页拦截器，mybatis-plus 提供的分页插件）**

    ```Java
     @Configuration
     public class myBatisConfig {
         @Bean
         public MybatisPlusInterceptor mybatisPlusInterceptor() {
             MybatisPlusInterceptor interceptor = new MybatisPlusInterceptor();
             //⭐new 一个分页拦截器
             PaginationInnerInterceptor paginationInnerInterceptor = new PaginationInnerInterceptor(DbType.H2);
             //设置 分页拦截器的属性值
             paginationInnerInterceptor.setOverflow(true);
             paginationInnerInterceptor.setMaxLimit(500L);
             interceptor.addInnerInterceptor(paginationInnerInterceptor);
             return interceptor;
         }
     }
    ```
7. controller

    ```Java
     @Controller
     public class tableController {
         @Autowired
         UserService userService;
         //查询数据
         @GetMapping("/dynamic_table")
         public String dynamic_table(@RequestParam(value = "pn",defaultValue = "1") Integer pn,
                                     Model model) {
             //先new 一个 page 分页对象（pn：当前第几页，2：每页显示多少条记录）
             Page<User> userPage = new Page<>(pn, 2);
             //分页查询结果
             Page<User> page = userService.page(userPage, null);
             //page里可以取出 records 并操作（也就是查出来的一个个user对象）
             //List<User> records = page.getRecords();
             //将查出来的数据放在 model 中，方面在页面中取出数据
             model.addAttribute("page",page);
             return "table/dynamic_table";
         }
         //删除数据
         @GetMapping("/user/delete/{id}")
         //将要删除的 id ，和删除后 继续留在当前页（效果）页码 以参数传入
         public String deleteUser(@PathVariable(value = "id") Long id,
                                  @RequestParam(value = "pn",defaultValue = "1") Integer pn,
                                  RedirectAttributes ra){
             //删除
             userService.removeById(id);
             //设置重定向转去的页码
             ra.addAttribute("pn",pn);
             //重定向
             return "redirect:/dynamic_table";
         }
     }
    ```

9. 页码前端

    ```HTML
     <div>
         <div>
             <!--使用 tymeleaf行内写法-->
             <div>
                 当前第 [[${page.current}]] 页，总计 [[${page.pages}]] 页，共 [[${page.total}]] 条记录
             </div>
         </div>
         <div>
             <div>
                 <ul>
                     <li><a href="#">← 上一页</a></li>
                     <!--实现当前页高亮激活-->
                     <!--\#numbers.sequence(1,page.pages)：实现按总页数展示页码按钮个数-->
                     <li th:class="${num == page.current?'active':''}"
                         th:each="num:${\#numbers.sequence(1,page.pages)}">
                         <a th:href="@{/dynamic_table(pn=${num})}">[[${num}]]</a>
                     </li>
                     <li><a href="#">下一页 → </a></li>
                 </ul>
             </div>
         </div>
     </div>
    ```
