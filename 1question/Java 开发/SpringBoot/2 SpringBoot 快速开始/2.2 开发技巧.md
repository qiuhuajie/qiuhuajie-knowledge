# 1. **lombok**
1. **IDEA2020.3版本之后，IDEA本身已经集成了Lombok插件，故无需安装**
    
    > 注意看 IEDA 的版本！
    
1. 引入依赖：pom.xml
    
    ```XML
     <dependency>
         <groupId>org.projectlombok</groupId>
         <artifactId>lombok</artifactId>
     </dependency>
    ```
    
1. lombok的使用
    
    1. 原始代码：
        
        ```Java
         package com.atguigu.boot.bean;
         
         import org.springframework.boot.context.properties.ConfigurationProperties;
         import org.springframework.stereotype.Component;
         
         //@Component
         
         @ConfigurationProperties(prefix = "mycar")
         public class Car {
         
             private String brand;
             private Integer price;
         
             public String getBrand() {
                 return brand;
             }
         
             public void setBrand(String brand) {
                 this.brand = brand;
             }
         
             public Integer getPrice() {
                 return price;
             }
         
             public void setPrice(Integer price) {
                 this.price = price;
             }
         
             @Override
             public String toString() {
                 return "Car{" +
                         "brand='" + brand + '\'' +
                         ", price=" + price +
                         '}';
             }
         }
        ```
        
    
    1. 使用lombok的注解简化开发
        
        ```Java
         @Data                               //get set 方法
         @AllArgsConstructor     //全属性构造器
         @NoArgsConstructor      //空参构造器
         @ToString                       //toString()
         @EqualsAndHashCode    //重写hashcode
         @ConfigurationProperties(prefix = "mycar")
         public class Car {
         
             private String brand;
             private Integer price;
        
         }
        ```
        
    
# 2. 简化日志开发
1. SLF4J不同于其他日志类库，与其它日志类库有很大的不同
1. SLF4J(Simple logging Facade for Java)不是一个真正的日志实现，而是一个抽象层（ abstraction layer）
1. 它允许你在后台使用任意一个日志类库
```Java
 @Slf4j
 @RestController
 public class HelloController {
 
     @RequestMapping("/hello")
     public String handle01(){
         //@Slf4j会在编译时，自动创建log
         log.info("====hello请求进来了====");
         return "Hello SpringBoot 2";
     }
 }
 
```
# 3. **devtools**
1. pom引入依赖
    
    ```XML
     <dependency>
         <groupId>org.springframework.boot</groupId>
         <artifactId>spring-boot-devtools</artifactId>
         <optional>true</optional>
     </dependency>
    ```
    
1. 在父工程 pom 文件中加入
    
    > 2、3、4步是在 springCloud 中学的，但好像只做 1 也可以
    
    ```XML
     <build>
       <plugins>
         <plugin>
           <groupId>org.springframework.boot</groupId>
           <artifactId>spring-boot-maven-plugin</artifactId>
           <configuration>
             <fork>true</fork>
             <addResources>true</addResources>
             <excludes>
               <exclude>
                 <groupId>org.projectlombok</groupId>
                 <artifactId>lombok</artifactId>
               </exclude>
             </excludes>
           </configuration>
         </plugin>
       </plugins>
     </build>
    ```
