# 2. **创建 POM**
1. 创建一个 maven 工程
1. pom 文件中添加父工程
    
    > **子工程中可以使用 父工程中引入的所有依赖**
    
    ```XML
     <?xml version="1.0" encoding="UTF-8"?>
     <project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 https://maven.apache.org/xsd/maven-4.0.0.xsd">
         <modelVersion>4.0.0</modelVersion>
     
         <groupId>com.example</groupId>
         <artifactId>myproject</artifactId>
         <version>0.0.1-SNAPSHOT</version>
     
         <parent>
             <groupId>org.springframework.boot</groupId>
             <artifactId>spring-boot-starter-parent</artifactId>
             <version>2.3.4.RELEASE</version>
         </parent>
    
     </project>
    ```
    
1. 修改 pom 文件后，点击右上角按钮即可更新依赖

# 4. **编写代码**
```Java
 @SpringBootApplication //告诉SpringBoot这是一个SpringBoot应用
 public class MainApplication {
     public static void main(String[] args) {
         //主程序入口（固定写法）
         SpringApplication.run(MainApplication.class,args);
     }
 }
```
```Java
 //@Controller
 //@ResponseBody //该处理器类下的每一个方法返回的数据，直接写给浏览器，而不是跳转到某个页面
 
 @RestController //是 @Controller和 @ResponseBody 的替换
 public class HelloController {
 
     @RequestMapping("/hello") //表示用 handle01()来处理 /hello 请求
     public String handle01(){
         return "Hello SpringBoot 2";
     }
 }
```

# 7. **简化部署**
1. 把项目打成`**.jar**`包，直接在目标服务器执行即可。（SpringMVC 需要打成`**.war**`包）
1. pom.xml中添加插件
    
    ```XML
      <build>
          <plugins>
              <plugin>
                  <groupId>org.springframework.boot</groupId>
                  <artifactId>spring-boot-maven-plugin</artifactId>
              </plugin>
          </plugins>
     </build>
    ```
