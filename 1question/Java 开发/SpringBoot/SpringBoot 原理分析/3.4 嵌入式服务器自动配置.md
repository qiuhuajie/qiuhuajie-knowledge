> 💡 SpringBoot 不需要自己配置 服务器，是因为底层自动嵌入了服务器
# 一、嵌入式服务器原理
1. 默认支持的webServer
    1. Tomcat, Jetty, or Undertow
    2. **ServletWebServerApplicationContext 容器启动寻找 ServletWebServerFactory 并引导创建服务器**[[2.3 IOC 容器的填充]]
2. 原理：
    1. SpringBoot应用启动发现当前是Web应用，其中 web场景包中会 导入tomcat
    2. web 应用会创建一个 web 版的 ioc容器 **`ServletWebServerApplicationContext`**
    3. ServletWebServerApplicationContext 启动的时候寻找 **ServletWebServerFactory**（Servlet 的web服务器工厂），这个工厂生产的就是 Servlet 的web服务器
    4. SpringBoot底层默认有很多的WebServer工厂
        1. **`TomcatServletWebServerFactory`**
        2. **`JettyServletWebServerFactory`**
        3. **`UndertowServletWebServerFactory`**
    5. 这些工厂，在底层 由 ServletWebServerFactoryAutoConfiguration 自动配置好
    6. ServletWebServerFactoryAutoConfiguration 导入了ServletWebServerFactoryConfiguration（配置类）
    7. ServletWebServerFactoryConfiguration 配置类 根据动态判断系统中到底导入了那个Web 服务器的包（默认是web-starter导入tomcat包），所以 容器中就有自动有一个 TomcatServletWebServerFactory 的WebServer工厂
    8. TomcatServletWebServerFactory 创建出Tomcat 服务器（TomcatWebServer）并启动
    9. TomcatWebServer 的构造器拥有一个初始化方法 ：initialize() ------>里面调用 this.tomcat.start();
    > 💡 说白了内嵌服务器，
    > **就是自动调用启动服务器的代码**（前提：tomcat核心jar包存在）

# 二、切换嵌入式服务器

切换 内嵌 服务器 为 **undertow （一般不会这么做，就用 tomcat ）**

![[IMG-20260404031829742.png|800]]

1. 引入依赖

    ```XML
     <dependency>
         <groupId>org.springframework.boot</groupId>
         <artifactId>spring-boot-starter-web</artifactId>
         将 web starter 中默认引入的 tomcat 服务器移除
         <exclusions>
             <exclusion>
                 <groupId>org.springframework.boot</groupId>
                 <artifactId>spring-boot-starter-tomcat</artifactId>
             </exclusion>
         </exclusions>
     </dependency>
     引入 undertow 服务器的 依赖
     <dependency>
         <groupId>org.springframework.boot</groupId>
         <artifactId>spring-boot-starter-undertow</artifactId>
     </dependency>
    ```
2. 效果：

# 三、定制化 内嵌服务器
1. 方式1：**（推荐）**
    1. WebServerFactoryCustomizer 把配置文件的值和**ServletWebServerFactory 进行绑定**
    2. 故可以 修改配置文件的 **server.xxx** 来实现定制化 内嵌服务器
2. 方式2：

    直接自定义 **ConfigurableServletWebServerFactory**（自己写一个 服务器工厂，造 定制化的 内嵌服务器）

3. 方式3：

    使用 WebServerFactoryCustomizer 定制化器
