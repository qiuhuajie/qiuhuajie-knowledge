# 1. 介绍
1. 若mapper接口中的方法参数为实体类对象时
1. 此时可以使用`${}`和`#{}`，通过访问实体类对象中的属性名获取属性值，注意`${}`需要手动加单引号
# 2. 示例
mapper接口：
```Java
 public interface UserMapper {
     User selectUser(User user);
 }
```
sql 映射文件：

> [!important] ⭐传入的参数是一个实体类，直接使用实体类的属性就可以获取到值
```XML
 <mapper namespace="com.qhj.mapper.UserMapper">
     <!--User selectUser(String name,int age)-->
     <select id="selectUser" resultType="com.qhj.pojo.User">
         select * from user where name = #{name} and age = #{age}
     </select>
 </mapper>
```
测试：
```Java
 User user = new User();
 user.setName("Tom");
 user.setAge(28);
 User tom = userMapper.selectUser(user);
 System.out.println(tom.toString());
 //User{id=3, name='Tom', age=28, email='test3@baomidou.com'}
```
