# 单个字面量 参数的获取

# 1. 介绍

1. 若mapper接口中的方法参数为单个的字面量类型
2. 此时可以使用 ${} 和 #{} 以任意的名称获取参数的值，注意${}需要手动加单引号

# 2. 示例

mapper接口：

```java
 public interface UserMapper {
     User selectUser(String name);
 }
```

sql 映射文件：

```xml
 <mapper namespace="com.qhj.mapper.UserMapper">
     <!--User selectUser(String name)-->
     <select id="selectUser" resultType="com.qhj.pojo.User">
         select * from user where name = #{name}
         <!--或 select * from user where name = '${name}'-->
     </select>
 </mapper>
```

测试：

```java
 public class MyBatisTest {
     @Test
     public void Test01() throws IOException {
         InputStream is = Resources.getResourceAsStream("mybatis-config.xml");
         SqlSessionFactoryBuilder sqlSessionFactoryBuilder = new SqlSessionFactoryBuilder();
         SqlSessionFactory sqlSessionFactory = sqlSessionFactoryBuilder.build(is);
         SqlSession sqlSession = sqlSessionFactory.openSession(true);
         UserMapper userMapper = sqlSession.getMapper(UserMapper.class);

         User tom = userMapper.selectUser("Tom");
         System.out.println(tom.toString());
         //User{id=3, name='Tom', age=28, email='test3@baomidou.com'}
     }
 }
```
