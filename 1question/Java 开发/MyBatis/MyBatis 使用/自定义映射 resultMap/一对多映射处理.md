# 一对多映射处理

# 1. 问题藐视

1. 根据部门id查新部门以及部门中的员工信息
2. 一对多关联查询，返回的是一个 List
3. 部门 dept ：员工 emp ➡️  1     ：    n

# 2. 解决方案

## 2.1 方案1：**collection 处理映射**

```java
 public class Dept {
     private int did;
     private String deptName;
     private List<Emp> empList;
         ...
 }
```

```java
 public interface DeptMapper {
     Dept getDeptEmpByDid(@Param("did") int did);
 }
```

```xml
 <mapper namespace="com.qhj.mapper.DeptMapper">

     <resultMap id="deptEmpMap" type="com.qhj.pojo.Dept">
         <id property="did" column="did"></id>
         <result property="deptName" column="dept_name"></result>
 
         <collection property="empList" ofType="com.qhj.pojo.Emp">
             <id property="eid" column="eid"></id>
             <result property="empName" column="emp_name"></result>
             <result property="age" column="age"></result>
         </collection>
     </resultMap>
 
     <select id="getDeptEmpByDid" resultMap="deptEmpMap">
         select dept.*,emp.* from t_dept dept left join t_emp emp on dept.dept_name = emp.dept where dept.did=#{did}
     </select>
 </mapper>
```

> `collection`的属性 `ofType`：设置collection标签所处理的集合属性中存储数据的类型
>  

```java
 DeptMapper deptMapper = sqlSession.getMapper(DeptMapper.class);
 Dept deptEmpByDid = deptMapper.getDeptEmpByDid(3);
 System.out.println(deptEmpByDid);
 //结果：
 //Dept{did=3, deptName='c', empList=[Emp{eid=2, empName='lisi', age=34, dept=null}, Emp{eid=3, empName='wangwu', age=23, dept=null}]}
```

## 2.2 方案2：**分步查询** ☑️

> 先查部门，再根据查出的部门的 dept_name 去查员工
>  

### **step1：查询部门信息：**

1. DeptMapper
    
    ```java
     public interface DeptMapper {
         Dept getDeptByStep(@Param("dept_name") String dept_name);
     }
    ```
    
2. DeptMapper.xml
    
    ```xml
     <mapper namespace="com.qhj.mapper.DeptMapper">
    
         <resultMap id="deptEmpStep" type="com.qhj.pojo.Dept">
             <id property="did" column="did"></id>
             <result property="deptName" column="dept_name"></result>
             <collection property="empList" fetchType="eager"
                         select="com.qhj.mapper.EmpMapper.getEmpListByName" column="dept_name">
             </collection>
         </resultMap>
     
         <select id="getDeptByStep" resultMap="deptEmpStep">
             select * from t_dept where dept_name = #{dept_name}
         </select>
    
     </mapper>
    ```
    

### **step2：根据部门的名字 dept_name 查询该部门所有员工的信息**

1. EmpMapper
    
    ```java
     public interface EmpMapper {
         List<Emp> getEmpListByName(@Param("dept_name") String dept_name);
     }
    ```
    
2. EmpMapper.xml
    
    ```xml
     <mapper namespace="com.qhj.mapper.EmpMapper">
    
         <select id="getEmpListByName" resultType="com.qhj.pojo.Emp">
             select * from t_emp where dept = #{dept_name}
         </select>
     
     </mapper>
    ```
    
3. **测试：**
    
    ```java
     DeptMapper deptMapper = sqlSession.getMapper(DeptMapper.class);
     Dept dept = deptMapper.getDeptByStep("c");
     System.out.println(dept);
     //结果：
     //Dept{did=3, deptName='c', empList=[Emp{eid=2, empName='lisi', age=34, dept=null}, Emp{eid=3, empName='wangwu', age=23, dept=null}]}
    ```
