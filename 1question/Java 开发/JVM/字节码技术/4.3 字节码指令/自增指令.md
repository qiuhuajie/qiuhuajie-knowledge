# 一、a++ 和 ++a 区别的原理
1. `iinc` 指令：**直接在局部变量 slot （槽位）上进行运算**
2. `iload` 指令：**在栈帧里，将局部变量表中指定 slot （槽位）的数据，加载到操作数栈中**
3. ⭐==**`a++`== ==和== ==`++a`== ==的区别是：==**
    1. **`a++` ：先执行 `iload` ，再执行 `iinc**`
    2. **`++a` ：先执行 `iinc` ，再执行 `iload**`

# 二、代码示例
## 1. 原始Java代码
```Java
 public class Demo3_2 {
     public static void main(String[] args) {
         int a = 10;
         int b = a++ + ++a + a--;
         System.out.println(a);
         System.out.println(b);
     }
 }
```
