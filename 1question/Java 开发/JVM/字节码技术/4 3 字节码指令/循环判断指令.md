# **原始Java代码**
`while`
```Java
 public class Demo {
     public static void main(String[] args) {
         int a = 0;
         while (a < 10) {
             a++;
         }
     }
 }
```
`do-while`
```Java
 public class Demo {
     public static void main(String[] args) {
         int a = 0;
         do {
             a++;
         } while (a < 10);
     }
 }
```
`for`
```Java
 public class Demo {
     public static void main(String[] args) {
         for (int i = 0; i < 10; i++) {
         }
     }
 }
```

# **练习题**
1. 源代码
    
    ```Java
     public class Demo {
         public static void main(String[] args) {
             int i = 0;
             int x = 0;
             while (i < 10) {
                 x = x++;
                 i++;
             }
             System.out.println(x); // 结果是 0
         }
     }
    ```
