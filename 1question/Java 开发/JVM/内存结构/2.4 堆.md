# 1. 介绍
1. 堆（Heap）
    
    1. 供所有==**类实例对象**====和====**数组对象**==分配内存的区域
    
    1. 通过 `new` 关键字创建对象都会使用堆内存，几乎所有实例对象都在会这里进行内存的分配
    
1. 堆内存是==线程共享==的，堆中对象都需要考虑线程安全的问题
1. 堆内存涉及垃圾回收机制
# 2. **堆内存大小**
1. 可以在运行代码时，通过虚拟机指令来指定：
    
    - `Xmx`：最大堆大小 `Xmx8m`
    
    - `Xms`：初始堆大小
    
    - `Xmn`：年轻代大小
    
1. 堆内存溢出问题：
    
    1. 演示代码
        
        ```Java
         public class heapDemo1 {
                 public static void main(String[] args) {
                     int i = 0;
                     try {
                         List<String> list = new ArrayList<>();
                         String a = "hello";
                         while (true) {
                             list.add(a); // hello, hellohello, hellohellohellohello ...
                             a = a + a;  // hellohellohellohello
                             i++;
                         }
                     } catch (Throwable e) {
                         e.printStackTrace();
                         System.out.println(i);
                     }
                 }
         }
        ```
        
    
    1. 运行结果：堆空间内存溢出 `OutOfMemoryError: Java heap space`

# 3. **堆内存诊断⭐**
1. 诊断演示：
    
    1. 代码
        
        ```Java
         public class heapDemo2 {
             /**
              * 演示堆内存
              */
             public static void main(String[] args) throws InterruptedException {
                 System.out.println("1...");
                 Thread.sleep(30000);
         
                 //new 一个10M的数组
                 byte[] array = new byte[1024 * 1024 * 10]; // 10 Mb
                 System.out.println("2...");
                 Thread.sleep(20000);
         
                 //数组引用设为null，也就意味着内存区域可以被当作垃圾回收掉
                 array = null;
                 System.gc();        //垃圾回收
                 System.out.println("3...");
                 Thread.sleep(1000000L);
             }
         
         }
        ```
