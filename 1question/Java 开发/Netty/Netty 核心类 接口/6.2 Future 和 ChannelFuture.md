# 一、Future 和 ChannelFuture
1. Netty 中所有的 IO 操作都是异步的，不能立刻得知消息是否被正确处理
2. 但是可以过一会等它执行完成或者直接注册一个监听，具体的实现就是通过 `Future` 和 `ChannelFutures`， 他们可以注册一个监听， 当操作执行成功或失败时监听会自动触发注册的监听事件
3. 常见的方法有
    - `Channel channel()`：返回当前正在进行 IO 操作的通道
    - `ChannelFuture sync()`：等待异步操作执行完毕
