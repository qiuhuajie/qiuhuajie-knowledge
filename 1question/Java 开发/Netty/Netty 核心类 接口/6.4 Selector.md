# 一、Selector
1. Netty **==基于 Selector 对象实现 I/O 多路复用==，通过 Selector 一个线程可以监听多个连接的 Channel 事件**

    ![[IMG-20260621000550426.png|917]]

2. 实现原理：
    1. 当向一个 Selector 中注册 Channel 后，Selector 内部的机制就可以自动**不断地查询（Select）**这些注册的 Channel 是否有已就绪的 I/O 事件（例如可读， 可写， 网络连接完成等）
    2. 这样程序就可以很简单地使用一个线程高效地管理多个 Channel
