# 1. 需求分析
1. 使用传统的 IO 方法传递一个大文件
1. 使用 NIO 零拷贝方式传递（transferTo）一个大文件：⭐**如果 Linux 系统支持 `sendfile()`系统调用，那么 `transferTo()`实际上最后就会使用到 `sendfile()`系统调用函数**
1. 看看两种传递方式耗时时间分别是多少
# 2. 传统文件传输
1. `OldIOServer`
    
    ```Java
    public class OldIOServer {
        public static void main(String[] args) throws Exception {
            ServerSocket serverSocket = new ServerSocket(7001);
    
            while (true) {
                Socket socket = serverSocket.accept();
                DataInputStream dataInputStream = new DataInputStream(socket.getInputStream());
    
                try {
                    byte[] byteArray = new byte[4096];
    
                    while (true) {
                        int readCount = dataInputStream.read(byteArray, 0, byteArray.length);
                        if (-1 == readCount) {
                            break;
                        }
                    }
                } catch (Exception ex) {
                    ex.printStackTrace();
                }
            }
        }
    }
    ```
    
1. `OldIOClient`
    
    ```Java
    public class OldIOClient {
        public static void main(String[] args) throws Exception {
    
            String fileName = "E:/游戏动漫人体结构造型手绘技法_14087650.pdf";
            InputStream inputStream = new FileInputStream(fileName);
    
            Socket socket = new Socket("localhost", 7001);
            DataOutputStream dataOutputStream = new DataOutputStream(socket.getOutputStream());
    
            byte[] buffer = new byte[4096];
            long readCount;
            long total = 0;
    
            long startTime = System.currentTimeMillis();
    
            while ((readCount = inputStream.read(buffer)) >= 0) {
                total += readCount;
                dataOutputStream.write(buffer);
            }
    
            System.out.println("发送总字节数： " + total + ", 耗时： " + (System.currentTimeMillis() - startTime));
    
            dataOutputStream.close();
            socket.close();
            inputStream.close();
        }
    }
    ```
    
1. 运行结果：耗时 `259 ms`

# 3. NIO 零拷贝文件传输
1. `NewIOServer`
    
    ```Java
    public class NewIOServer {
        public static void main(String[] args) throws Exception {
    
            ServerSocketChannel serverSocketChannel = ServerSocketChannel.open();
            serverSocketChannel.socket().bind(new InetSocketAddress(7002));
    
            ByteBuffer byteBuffer = ByteBuffer.allocate(4096);
    
            while (true) {
                SocketChannel socketChannel = serverSocketChannel.accept();
    
                int readcount = 0;
                while (-1 != readcount) {
                    try {
                        readcount = socketChannel.read(byteBuffer);
                    }catch (Exception ex) {
                        // ex.printStackTrace();
                        break;
                    }
                    //
                    byteBuffer.rewind(); //倒带 position = 0 mark 作废
                }
            }
        }
    }
    ```
    
1. `NewIOClient`
    
    ```Java
    public class NewIOClient {
        public static void main(String[] args) throws Exception {
    
            SocketChannel socketChannel = SocketChannel.open();
            socketChannel.connect(new InetSocketAddress("localhost", 7002));
    
            //得到一个文件channel
            String fileName = "E:/游戏动漫人体结构造型手绘技法_14087650.pdf";
            FileChannel fileChannel = new FileInputStream(fileName).getChannel();
    
            //准备发送
            long startTime = System.currentTimeMillis();
    
            long total = 0;
            long position = 0L;
            long n = fileChannel.size() / 8192000L;
            for (int i = 0; i < n + 1; i++) {
                total += fileChannel.transferTo(position, fileChannel.size(), socketChannel);
                position += 8192001L;
            }
    
            System.out.println("发送总字节数： " + total + ", 耗时： " + (System.currentTimeMillis() - startTime));
    
            //关闭
            fileChannel.close();
    
        }
    }
    ```
    
    > [!important] **`transferTo` 底层使用到零拷贝**
    > 
    > 1. 在 linux 下一个 `transferTo` 方法就可以完成传输
    > 
    > 1. 在 windows 下一次调用 `transferTo` 只能发送`8m` , 就需要分段传输文件
    
1. 运行结果：耗时 `52 ms`
