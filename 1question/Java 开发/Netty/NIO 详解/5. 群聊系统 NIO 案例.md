# 1. 需求分析
1. 编写一个 NIO 群聊系统， 实现服务器端和客户端之间的数据简单通讯（非阻塞）
1. 实现多人群聊
1. 服务器端： 可以监测用户上线， 离线，并实现消息转发功能
1. 客户端： 通过channel 可以无阻塞发送消息给其它所有用户， 同时可以接受其它用户发送的消息（由服务器转发得到）
1. 目的： 进一步理解NIO非阻塞网络编程机制
# 2. 架构图
![[IMG-20260405035413800.png|Untitled 364.png]]
1. 客户端和服务端各自维护着一个 `Selector`
1. 自己使用的 `Channel` 都注册在自己的 `Selector` 上进行监听状态的改变
1. **客户端和服务端传输数据时，使用的`SocketChannel` 在内存中并不是同一个实例**
1. 这种方案服务器端用**一个线程**通过**多路复用**搞定所有的 IO 操作（包括连接，读、写等），编码简单，清晰明了，但是如果客户端连接数量较多，将无法支撑，属于**单 Reactor 单进程 / 线程模型**[[5. 群聊系统 NIO 案例]]
# 3. 服务端
`ChartServer`
```Java
public class ChartServer {
    private static final int PORT = 6607;
    private Selector selector;
    private ServerSocketChannel ssc;
    // 1.初始化Selector、ServerSocketChannel，并将服务端通道注册在Selector中
    public ChartServer() {
        try {
            selector = Selector.open();
            ssc = ServerSocketChannel.open();
            ssc.socket().bind(new InetSocketAddress(PORT));
            ssc.configureBlocking(false);
            ssc.register(selector, SelectionKey.OP_ACCEPT);
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
    // 2. 服务端监听方法：轮询的方式使用select()对Selector中的所有Channel的状态改变进行监听
    public void listen() {
        try {
            while (true) {
                if (selector.select(2000) == 0) {
                    continue;
                }
                Set<SelectionKey> selectedKeys = selector.selectedKeys();
                Iterator<SelectionKey> iterator = selectedKeys.iterator();
                if (iterator.hasNext()) {
                    SelectionKey key = iterator.next();
                    if (key.isAcceptable()) {
                        // 3.如果是建立客户端连接事件，则用ServerSocketChannel构造一个新的SocketChannel实例，并注册在Selector中
                        ServerSocketChannel sscTemp = (ServerSocketChannel) key.channel();
                        SocketChannel socketChannel = sscTemp.accept();
                        socketChannel.configureBlocking(false);
                        socketChannel.register(selector, SelectionKey.OP_READ);
                        System.out.println(socketChannel.getRemoteAddress() + "上线了");
                    } else if (key.isReadable()) {
                        // 4.如果是读取客户端连接上的数据事件，则调用读取数据的方法
                        readData(key);
                    }
                    // 5.事件处理完后，将当前SelectionKey从selectedKeys中去除
                    iterator.remove();
                }
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
    // 6.服务端读取数据方法：使用ByteBuffer将有数据传过来的那个SocketChannel中的数据读出来，并转发给其它客户端
    public void readData(SelectionKey key) {
        SocketChannel channel = null;
        try {
            channel = (SocketChannel) key.channel();
            ByteBuffer buffer = ByteBuffer.allocate(1024);
            channel.read(buffer);
            String msg = new String(buffer.array());
            System.out.println("服务器收到客户端发来的数据：" + msg.trim());
            transport(msg, channel);
        } catch (IOException e) {
            try {
                System.out.println(channel.getRemoteAddress() + "下线了");
                key.channel();
                channel.close();
            } catch (IOException e2) {
                e2.printStackTrace();
            }
        }
    }
    // 7.服务读到数据后，转发给其它客户端，这里传入的self是消息的发送方的SocketChannel
    private void transport(String msg, SocketChannel self) {
        try {
            // 获取到当前已经注册的所有SelectionKey，进行遍历
            Set<SelectionKey> keys = selector.keys();
            for (SelectionKey key : keys) {
                SelectableChannel channel = key.channel();
                // 先排除服务端的ServerSocketChannel，再排除自己（数据发送方）的SocketChannel
                if (channel instanceof SocketChannel && channel != self) {
                    SocketChannel dst = (SocketChannel) channel;
                    ByteBuffer byteBuffer = ByteBuffer.wrap(msg.getBytes());
                    dst.write(byteBuffer);
                    System.out.println("服务端转发消息给" + dst.getRemoteAddress());
                }
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
    public static void main(String[] args) {
        ChartServer chartServer = new ChartServer();
        chartServer.listen();
    }
}
```
# 4. 客户端
`ChartClient01` 、`ChartClient02`
```Java
public class ChartClient01 {
    private final String HOST = "127.0.0.1";
    private final int PORT = 6607;
    private Selector selector;
    private SocketChannel socketChannel;
    private String username;
    // 1.初始化Selector、SocketChannel
    public ChartClient01() {
        try {
            selector = Selector.open();
            socketChannel = SocketChannel.open(new InetSocketAddress(HOST, PORT));
            socketChannel.configureBlocking(false);
            socketChannel.register(selector, SelectionKey.OP_READ);
            username = socketChannel.getLocalAddress().toString().substring(1);
            System.out.println(username + " is OK");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
    // 2.发送消息：即向客户端SocketChannel中写数据
    public void sendMsg(String msg) {
        msg = username + " say: " + msg;
        try {
            socketChannel.write(ByteBuffer.wrap(msg.getBytes()));
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
    // 3.读消息：客户端轮询监听服务端转发自其他客户端的消息
    public void readMsg() {
        try {
            while (true) {
                if (selector.select() == 0) {
                    return;
                }
                Set<SelectionKey> selectedKeys = selector.selectedKeys();
                Iterator<SelectionKey> iterator = selectedKeys.iterator();
                while (iterator.hasNext()) {
                    SelectionKey key = iterator.next();
                    if (key.isReadable()) {
                        SocketChannel sc = (SocketChannel) key.channel();
                        ByteBuffer byteBuffer = ByteBuffer.allocate(1024);
                        sc.read(byteBuffer);
                        System.out.println(new String(byteBuffer.array()).trim());
                    }
                    iterator.remove();
                }
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
    public static void main(String[] args) {
        ChartClient01 chartClient = new ChartClient01();
        // 4.新启一个线程让其，不断读数据
        new Thread() {
            public void run() {
                chartClient.readMsg();
            }
        }.start();
        // 5.本客户端发送消息
        Scanner scanner = new Scanner(System.in);
        while (scanner.hasNextLine()) {
            String msg = scanner.nextLine();
            chartClient.sendMsg(msg);
        }
    }
}
```
