# thenCombine()
1. 两个 `CompletionStage` 任务都完成后，最终能把两个任务的结果一起交给 `thenCombine()` 来处理
1. 先完成的先等着，等待其它分支任务
1. 代码示例：
    
    ```Java
    public class CompletableFutureAPIDemo2 {
        public static void main(String[] args) throws ExecutionException, InterruptedException, TimeoutException {
    
            // completableFuture1
            CompletableFuture<Integer> completableFuture1 = CompletableFuture.supplyAsync(() -> {
                System.out.println("task1 finish ");
                return 10;
            });
    
            // completableFuture2
            CompletableFuture<Integer> completableFuture2 = CompletableFuture.supplyAsync(() -> {
                System.out.println("task2 finish ");
                return 20;
            });
    
            // thenCombineResult 对两个结果进行合并
            CompletableFuture<Integer> thenCombineResult = completableFuture1.thenCombine(completableFuture2, (x, y) -> {
                System.out.println("combineTask finish ");
                return x + y;
            });
    
            System.out.println("result : " + thenCombineResult.get());
        }
    }
    ```
