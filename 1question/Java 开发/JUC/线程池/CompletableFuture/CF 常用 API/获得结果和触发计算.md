# 1. 获取结果
## 1.1 get()
1. 获取结果
1. `get()` 方法抛出的是经过检查的异常，如 ExecutionException, InterruptedException，**需要用户手动处理（抛出或者 `try catch`）**
## 1.2 get(long Timeout, TimeUnit unit)
1. 主动结束，并传入一个等待时间
1. 代码示例：
    
    ```Java
    public class CompletableFutureAPIDemo2 {
        public static void main(String[] args) throws ExecutionException, InterruptedException, TimeoutException {
            long startTime = System.currentTimeMillis();
            CompletableFuture<String> completableFuture = CompletableFuture.supplyAsync(() -> {
                try { TimeUnit.SECONDS.sleep(5); } catch (InterruptedException e) { e.printStackTrace(); }
                return "task finish";
            });
    
            System.out.println(completableFuture.get(2, TimeUnit.SECONDS));
            long endTime = System.currentTimeMillis();
            System.out.println("耗时" + (endTime - startTime) + "毫秒");
        }
    }
    ```

## 1.3 join()
1. **作用等同于 `get()`**
1. 区别：`join()` 抛出的是 uncheck 异常（即未经检查的异常，如 `_throws_` `ExecutionException, InterruptedException`），**不会强制开发者对抛出的异常做处理**
1. 代码示例：
    
    ```Java
    public class CompletableFutureAPIDemo2 {
        public static void main(String[] args) {
            long startTime = System.currentTimeMillis();
            CompletableFuture<String> completableFuture = CompletableFuture.supplyAsync(() -> {
                try { TimeUnit.SECONDS.sleep(5); } catch (InterruptedException e) { e.printStackTrace(); }
                return "task finish";
            });
    
            System.out.println(completableFuture.join());
            long endTime = System.currentTimeMillis();
            System.out.println("耗时" + (endTime - startTime) + "毫秒");
        }
    }
    ```

## 1.4 getNow(T valueIfAbsent)
1. 没有计算完成的情况下，返回一个替代结果，这个替代结果就是传入的参数 `valueIfAbsent`
1. 立即获取结果不阻塞
    
    - 计算完，返回计算完成后的结果
    
    - 没算完，返回设定的 `valueIfAbsent` 值
    
1. 代码示例：
    
    ```Java
    public class CompletableFutureAPIDemo2 {
        public static void main(String[] args) throws ExecutionException, InterruptedException, TimeoutException {
            long startTime = System.currentTimeMillis();
            CompletableFuture<String> completableFuture = CompletableFuture.supplyAsync(() -> {
                try { TimeUnit.SECONDS.sleep(5); } catch (InterruptedException e) { e.printStackTrace(); }
                return "task finish";
            });
    
            System.out.println(completableFuture.getNow("task not finish : valueIfAbsent"));
            long endTime = System.currentTimeMillis();
            System.out.println("耗时" + (endTime - startTime) + "毫秒");
        }
    }
    ```

# 2. 主动触发计算
## complete(T value)
1. 返回的是一个 `boolean` ，表示**异步任务是否被打断**
    
    - **打断**则返回 `true` ， 此时 `get()` 会返回 `complete()` 传入的 `value` 值
    
    - **没有打断**则返回 `false` ， 此时 `get()` 会返回任务计算完成的值
    

    ![[IMG-20260404031741221.png|Untitled 3 256.png]]

    
1. 代码示例：
    
    1. 情形一：
        
        ```Java
        public class CompletableFutureAPIDemo2 {
            public static void main(String[] args) throws ExecutionException, InterruptedException, TimeoutException {
                long startTime = System.currentTimeMillis();
                CompletableFuture<String> completableFuture = CompletableFuture.supplyAsync(() -> {
                    try { TimeUnit.SECONDS.sleep(5); } catch (InterruptedException e) { e.printStackTrace(); }
                    return "task finish";
                });
        
                System.out.println(completableFuture.complete("break") + "\t" + completableFuture.get());
                long endTime = System.currentTimeMillis();
                System.out.println("耗时" + (endTime - startTime) + "毫秒");
            }
        }
        ```
        

        ![[IMG-20260404031741290.png|Untitled 4 196.png]]

        
    
    1. 情形二：
        
        ```Java
        public class CompletableFutureAPIDemo2 {
            public static void main(String[] args) throws ExecutionException, InterruptedException, TimeoutException {

                System.out.println("耗时" + (endTime - startTime) + "毫秒");
            }
        }
        ```
