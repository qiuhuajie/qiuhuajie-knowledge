# applyToEither()
1. 谁快用谁
1. 代码示例：
    
    ```Java
    public class CompletableFutureAPIDemo2 {
        public static void main(String[] args) throws ExecutionException, InterruptedException, TimeoutException {
    
    				// completableFuture1 需要执行 500ms
            CompletableFuture<String> completableFuture1 = CompletableFuture.supplyAsync(() -> {
               try { TimeUnit.MILLISECONDS.sleep(500); } catch (InterruptedException e) { e.printStackTrace(); }
                System.out.println("task1 finish");
               return "task1";
            });
    
    				// completableFuture2 需要执行 300ms
            CompletableFuture<String> completableFuture2 = CompletableFuture.supplyAsync(() -> {
                try { TimeUnit.MILLISECONDS.sleep(300); } catch (InterruptedException e) { e.printStackTrace(); }
                System.out.println("task2 finish");
                return "task2";
            });
    
    				// completableFuture 会选择执行最快的那个
            CompletableFuture<String> completableFuture = completableFuture1.applyToEither(completableFuture2, v -> {
                return v + " is winner";
            });
    
            System.out.println(completableFuture.join());
    
        }
    }
    ```
