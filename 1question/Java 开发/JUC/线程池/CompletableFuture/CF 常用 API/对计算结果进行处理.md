# 1. 需要返回结果
## thenApply()

![[IMG-20260405035414091.png|image 10.png]]

1. 计算结果存在**依赖关系**，这两个线程**==串行化==**
    

    ![[IMG-20260404031740547.png|Untitled 561.png]]

    
    - 入参为 `Function`
    
1. 由于存在依赖关系，当前步骤有异常的话就叫停
1. 代码示例：
    
    ```Java
    public class CompletableFutureAPIDemo2 {
        public static void main(String[] args) throws ExecutionException, InterruptedException, TimeoutException {
            long startTime = System.currentTimeMillis();
    
            CompletableFuture<Integer> completableFuture = CompletableFuture.supplyAsync(() -> {
                try { TimeUnit.SECONDS.sleep(5); } catch (InterruptedException e) { e.printStackTrace(); }
                System.out.println("task1 finish");
                return 1000;
            }).thenApply(v -> {
                System.out.println("task2 finish");
                return v + 1;
            }).thenApply(v -> {
                System.out.println("task3 finish");
                return v + 1;
            }).whenComplete((v, e) -> {
                if (e == null) {
                    System.out.println("result :" + v);
                }
            }).exceptionally(e -> {
               e.printStackTrace();
               return -1;
            });
    
            System.out.println(completableFuture.join());
            long endTime = System.currentTimeMillis();
            System.out.println("耗时" + (endTime - startTime) + "毫秒");
    
        }
    }
    ```
    

    运行结果：

    

    ![[IMG-20260405035420401.png|Untitled 1 419.png]]

## handle()
1. 与 `thenApply()` 不同的在于：**有异常也可以往下一步走，根据带的异常参数可以进一步处理**
    

    ![[IMG-20260405035435274.png|Untitled 3 257.png]]

    
    - 入参为 `BiFunction`
    
1. 代码示例：
    
    ```Java
    public class CompletableFutureAPIDemo2 {
        public static void main(String[] args) throws ExecutionException, InterruptedException, TimeoutException {
            long startTime = System.currentTimeMillis();
    
            CompletableFuture<Integer> completableFuture = CompletableFuture.supplyAsync(() -> {
                try { TimeUnit.SECONDS.sleep(5); } catch (InterruptedException e) { e.printStackTrace(); }
                System.out.println("task1 finish");
                return 1000;
            }).handle((v, e) -> {
                System.out.println("task2 finish");
                return v + 1;
            }).handle((v, e) -> {
                int a = 2 / 0;
                System.out.println("task3 finish");
                return v + 1;
            }).whenComplete((v, e) -> {
                if (e == null) {
                    System.out.println("result :" + v);
                }
            }).exceptionally(e -> {
               e.printStackTrace();
               return -1;
            });
    
            System.out.println(completableFuture.join());
            long endTime = System.currentTimeMillis();
            System.out.println("耗时" + (endTime - startTime) + "毫秒");
    
        }
    }
    ```

# 2. 无需返回结果
## thenAccept()
1. 接收任务的处理结果，并消费处理，**无返回结果**
1. 代码示例：
    
    ```Java
    public class CompletableFutureAPIDemo2 {
        public static void main(String[] args) throws ExecutionException, InterruptedException, TimeoutException {
            long startTime = System.currentTimeMillis();
    
            CompletableFuture.supplyAsync(() -> {
                try { TimeUnit.SECONDS.sleep(5); } catch (InterruptedException e) { e.printStackTrace(); }
                System.out.println("task1 finish");
                return 1000;
            }).thenApply(v -> {
                System.out.println("task2 finish");
                return v + 1;
            }).thenApply(v -> {
                int a = 2 / 0;
                System.out.println("task3 finish");
                return v + 1;
            }).thenAccept(v -> System.out.println(v));   // 接收任务的处理结果，并消费处理，无返回结果（只打印）
    
            long endTime = System.currentTimeMillis();
            System.out.println("耗时" + (endTime - startTime) + "毫秒");
    
        }
    }
    ```
