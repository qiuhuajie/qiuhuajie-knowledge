# 一、Nacos 安装
1. 下载：

    https://github.com/alibaba/nacos/releases

![[IMG-20260405035413896.png|656]]

2. 解压，当前目录运行

    ```Plain
     startup.cmd -m standalone # -m 设置运行模式 单机standalone/集群cluster
    ```

    ![[IMG-20260404031842000.png|800]]

3. [http://localhost:8848/nacos](http://localhost:8848/nacos)：默认账号密码都是nacos

    ![[IMG-20260405035420296.png|800]]
