# 1. 简介
1. Ribbon：
    
    1. Spring Cloud Ribbon 是基于Netflix Ribbon实现的一套客户端（服务消费端）**负载均衡的工具**
    
    1. 主要功能
        
        1. **提供客户端的软件负载均衡算法和服务调用**
        
        1. 在配置文件中列出 Load Balancer（简称LB）后面所有的机器，Ribbon 会自动基于某种**负载均衡算法**（如简单轮询，随机、最大权限连接等）去连接这些机器
        
        1. 可以使用Ribbon实现**自定义的负载均衡算法**
        
        1. Ribbon客户端组件提供**一系列完善的配置项**如连接超时、重试等
        
    
1. 官网：
    
    > [!info] Getting Started  
    > Ribbon is a Inter Process Communication (remote procedure calls) library with built in software load balancers.  
    > [https://github.com/Netflix/ribbon/wiki/Getting-Started](https://github.com/Netflix/ribbon/wiki/Getting-Started)  
    
    - Ribbon目前也进入维护模式
