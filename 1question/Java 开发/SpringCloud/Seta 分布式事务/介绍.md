## 2.2 **1 个 ID + 3 个组件模型**
1. **`Transaction ID` (XID)**
    
    - 全局唯一的 事务ID （课程号）
    
1. `**Transaction Coordinator**` **(TC)**
    
    - 事务协调者 （上课老师）
    
    - 维护全局事务的运行状态，负责协调并驱动全局事务的提交或回滚
    
1. `**Transaction Manager**` **(TM)**
    
    - 事务管理器 （助教）
    
    - 控制全局事务的范围，负责开启一个全局事务，并最终发起全局提交或全局回滚的决议
    
1. `**Resource Manager**` **(RM)**
    
    - 资源管理器（学生）
    
    - 管理分支事务处理的资源（DB）、控制分支事务
    
    - 与 TC 交谈以 注册分支事务 和 报告分支事务的状态
    
    - 并接收事务协调器（TC）的指令，驱动分支（本地）事务的提交和回滚
