# 2. **开启 Feign**
- 在主启动类上添加注解 **`@EnableFeignClients`**
    
    ```Java
     @SpringBootApplication
     @EnableFeignClients         //⭐ 开启 Feign
     public class OrderFeignMain80 {
         public static void main(String[] args) {
             SpringApplication.run(OrderFeignMain80.class,args);
         }
     }
    ```
    
# 3. **使用 Feign**
1. **业务类 Feign 接口 @FeignClient**
    
    ```Java
     @Component //加载组件，方便再controller 注入使用
     
     @FeignClient(value = "CLOUD-PAYMENT-SERVICE")       //⭐ 使用 Feign
     public interface PaymentFeignService {
         @GetMapping(value = "/payment/get/{id}")
         CommonResult<Payment> getPaymentById(@PathVariable("id") Long id);
     }
    ```
    
1. **Controller 调用 Feign 接口的方法**
    
    ```Java
     @RestController
     public class OrderFeignController {
         @Resource
         private PaymentFeignService paymentFeignService;
             //调用
         @GetMapping(value = "/consumer/payment/get/{id}")
         public CommonResult<Payment> getPaymentById(@PathVariable("id") Long id) {
             return paymentFeignService.getPaymentById(id);
         }
     }
    ```

# 4. 调用流程解析
- **在服务消费者方写 Feign 接口，里面的方法与 远程服务提供者绑定**
- **当服务消费端的 contrller 调用 消费端的 Feign 接口方法时，就会去调用远程的 服务提供者的对应绑定的方法**
