## 1.2 配置环境变量
1. `path` 中添加 anaconda的安装目录的 `Scripts` 文件夹
    

    ![[IMG-20260404032011274.png]]

    
1. 查看anaconda版本
    
    ```Bash
    conda --version
    ```
    
1. 升级所有工具包
    
    ```Bash
    conda upgrade --all
    ```
    
## 1.3 管理虚拟环境
1. 创建虚拟环境
    
    ```Bash
    conda create --name my_python python=3.7
    ```
    
1. 查看当前所有虚拟环境
    
    ```Bash
    conda env list
    ```
    
1. 切换到指定虚拟环境
    
    ```Bash
    conda activate my_python
    ```
    
1. 退出当前虚拟环境
    
    ```Bash
    conda deactivate
    ```
    
1. 删除虚拟环境
    
    ```Bash
    conda remove --name my_python --all
    ```
    
1. 安装第三方包（可能出现当前用户写权限不够→ \to→使用管理员运行anaconda prompt命令行）
    
    ```Bash
    conda install requests
    ```
    
1. 查看环境内的包信息
    
    ```Bash
    conda list
    ```
    
1. 导出虚拟环境
    
    ```Bash
    conda env export > environment.yaml
    ```
    
1. 导入虚拟环境
    
    ```Bash
    conda env create -f environment.yaml
    ```
    
# 2. Pip
1. 一般在使用conda install安装时会出现包无法找到或者安装失败的情况，此时可以使用pip install来尝试安装（以opencv为例）：
    
    ```Bash
    pip install opencv
    ```
    
1. ==若安装速度过慢可单独指定安装镜像加快安装：==
    
    ```Bash
    pip install opencv -i http://pypi.douban.com/simple --trusted-host=pypi.douban.com
    ```
    
1. 或直接下载 whl 包或者源码包安装，下载地址
    
    > [!info] PyPI · The Python Package Index  
    > The Python Package Index (PyPI) is a repository of software for the Python programming language.  
    > [https://pypi.org/](https://pypi.org/)  
    
1. 此处列举国内常用pip安装镜像：

    1. 阿里云：[https://mirrors.aliyun.com/pypi/simple/](https://mirrors.aliyun.com/pypi/simple/)
