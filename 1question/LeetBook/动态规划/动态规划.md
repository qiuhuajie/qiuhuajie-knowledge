# 1. 介绍
1. 动态规划问题的**一般形式就是==求最值==**
    
    1. 动态规划其实是运筹学的一种最优化方法
    
    1. 只不过在计算机问题上应用比较多，比如说让你求最长递增子序列呀，最小编辑距离呀等等
    
1. 求解动态规划的**核心问题是**==**穷举**==
    
    1. 因为要求最值，肯定要把所有可行的答案穷举出来，然后在其中找最值
    
    1. 首先，动态规划的穷举有点特别，因为这类问题**存在[重叠子问题]**，如果**暴力穷举的话效率会极其低下**，所以**需要[备忘录，或者 [DP table] 来优化穷举过程，避免不必要的计算**
    
    1. 而且，**动态规划问题一定会具备[最优子结构]，才能通过子问题的最值得到原问题的最值**
    
1. ==动态规划三要素：重叠子问题、最优子结构、状态转移方程==
## 1.1 最优子结构
1. 最优子结构规定的是子问题与原问题的关系：即==**一个问题的最优解是由它的各个子问题的最优解决定的**==（**用动态规划法求解的**==**前提**==）
1. 动态规划要解决的都是一些问题的最优解，即从很多解决问题的方案中找到最优的一个
    
    1. 当我们在求一个问题最优解的时候，如果可以把这个问题**分解成多个子问题，然后用自底向上的方式，递归地找到每个子问题的最优解**
    
    1. 最后**通过一定的数学方法（状态转移方程）对各个子问题的最优解进行组合得出最终的结果**
    
1. 例如
    
    1. 原问题的解为 $f(n)$，其中 $f(n)$也叫状态
    
    1. 状态转移方程$f(n) = f(n - 1) + f(n - 2)$描述了一种原问题与子问题的组合关系
    
    1. 在原问题上还可能有一些选择，不同选择可能对应不同的子问题或者不同的组合方式。例如
        

        $$f(n) = \left\{ \begin{array}{lr} f(n - 1) + f(n - 2) \qquad n = 2k \\ f(n - 1) \qquad\qquad\qquad n = 2k + 1 \\ \end{array} \right.$$

        
    
    1. $n = 2k$ 和 $n = 2k + 1$ 对应了原问题 n 上不同的选择，分别对应了不同的子问题和组合方式
    
    1. 找到了最优子结构，也就能推导出一个状态转移方程 $f(n)$，通过这个状态转移方程，能很快的写出问题的递归实现方法
    
## 1.2 状态转移方程
1. **通过状态转移方程，**==**对各个子问题的最优解进行组合**==**得出最终的结果**
1. 只有列出正确的[状态转移方程，才能正确地穷举
1. 在实际的算法问题中，写出状态转移方程是最困难的
1. ⭐**思考状态转移方程：**
    
    1. 明确[状态]：这个问题有什么状态
    
    1. 明确[选择]：对于每个状态，可以做出什么选择使得状态发生改变
    
    1. 定义 dp 数组/函数的含义：如何定义 dp 数组来表示状态和选择
    
    1. 明确 basecase：最简单的情况
    
## 1.3 重叠子问题
1. 重叠子问题规定的是子问题与子问题的关系
    
    1. 在递归地寻找每个子问题的最优解的时候，有可能会重复地遇到一些更小的子问题，而且这些**==子问题会重叠地出现在子问题里，出现这样的情况，会有很多重复的计算==**
    
    1. **动态规划可以保证每个重叠的子问题只会被求解一次，可以减少很多重复的计算**
    
1. 例如，斐波那契问题的状态转移方程 $f(n) = f(n - 1) + f(n - 2)$
    
    1. 在求 $f(5)$ 时，需要先求子问题 $f(4)$ 和 $f(3)$ ，得到结果后再组合成原问题 $f(5)$ 的解
    
    1. 递归地求 $f(4)$ 时，又要先求子问题 $f(3)$ 和 $f(2)$ ，这里的 $f(3)$ 与求 $f(5)$ 时的子问题 $f(3)$ 重复了

    ```Plain
     假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

    ```

    ```Java
     public int climbStairs(int n) {
        
        if (n == 1) return 1;
        // dp数组
        int[] dp = new int[n];
        // 初值
        dp[0] = 1;
        dp[1] = 2;
        // 遍历
        for (int i = 2; i < n; i++) {
            dp[i] = dp[i - 1] + dp[i - 2];
        }
    
        return dp[n - 1];
    }
    ```

    ```Plain
     你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。
    ```

    - dp[i]：以下标 i（包括i）以内的房屋结尾的子数组，最多可以偷窃的金额
    
    ```Java
     public int rob(int[] nums) {
        if (nums.length == 1) return nums[0];
        
        // dp 数组
        int[] dp = new int[nums.length];
    
        // 初量
        dp[0] = nums[0];
        dp[1] = Math.max(nums[0], nums[1]);
    
        // 遍历
        for (int i = 2; i < nums.length; i++) {
            dp[i] = Math.max(dp[i - 1], dp[i - 2] + nums[i]);
        }
    
        return dp[nums.length - 1];
    }
    ```
    
## **📑337. 打家劫舍 III**🔥

> [!info] 力扣  
> 
> [https://leetcode.cn/problems/house-robber-iii/description/?favorite=2cktkvj](https://leetcode.cn/problems/house-robber-iii/description/?favorite=2cktkvj)  
1. 问题描述：
    
    ```Plain
    小偷又发现了一个新的可行窃的地区。这个地区只有一个入口，我们称之为 root 。
    
    除了 root 之外，每栋房子有且只有一个“父“房子与之相连。一番侦察之后，聪明的小偷意识到“这个地方的所有房屋的排列类似于一棵二叉树”。 如果 两个直接相连的房子在同一天晚上被打劫 ，房屋将自动报警。
    
    给定二叉树的 root 。返回 在不触动警报的情况下 ，小偷能够盗取的最高金额 。
    ```
    
1. 示例：
    
    ```Plain
    输入: root = [3,2,3,null,3,null,1]
    输出: 7 
    解释: 小偷一晚能够盗取的最高金额 3 + 3 + 1 = 7
    ```
    
1. 代码：
    
    - 使用一个长度为 2 的数组，记录当前节点偷与不偷所得到的的最大金钱。
    
    - dp[0] 记录不偷该节点所得到的的最大金钱，dp[1] 记录偷该节点所得到的的最大金钱
    
    - 在递归的过程中，系统栈会保存每一层递归的参数。所以使用长度为 2 的数组就可以标记树中每个节点的状态
    
    - 后序遍历
        
        - 出口：如果遇到空节点的话，很明显，无论偷还是不偷都是0，所以就返回，这也相当于 dp 数组的初始化
        
        - 通过递归左节点，得到左节点偷与不偷的金钱
        
        - 通过递归右节点，得到右节点偷与不偷的金钱
        
    
    - 状态转移
        
        - 如果是偷当前节点，那么左右孩子就不能偷 `left[1] + right[1] + root.val;`
        
        - 如果不偷当前节点，那么左右孩子就可以偷，至于到底偷不偷一定是选一个最大的 `Math.max(left[0], left[1]) + Math.max(right[0], right[1]);`
        
        - 最后当前节点的状态就是{val2, val1}; 即：{不偷当前节点得到的最大金钱，偷当前节点得到的最大金钱
        
    
    ```Java
     public int rob(TreeNode root) {
        int[] dp = posOrder(root);
        return Math.max(dp[0], dp[1]);
    }
    
    public int[] posOrder(TreeNode root) {
        int[] dp = {0, 0};
    
        if (root == null) return dp;
    
        // 偷左节点获得的最大金额和不偷左节点获得的最大金额
        int[] left = posOrder(root.left);
        // 偷右节点获得的最大金额和不偷右节点获得的最大金额
        int[] right = posOrder(root.right);
    
        // 偷当前节点，那么就不能偷左右节点
        dp[0] = left[1] + right[1] + root.val;
        // 不偷当前节点，则可能偷左右节点，也可能不偷左右节点
        dp[1] = Math.max(left[0], left[1]) + Math.max(right[0], right[1]);
    
        return dp;
    }
    ```
    
## **📑53. 最大连续子数组和**🔥⚔️✔️
1. 问题描述：
    
    ```Plain
    给你一个整数数组 nums ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。
    
    子数组 是数组中的一个连续部分。
    ```
    
1. 示例：
    
    ```Plain
    示例 1：
    输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
    输出：6
    解释：连续子数组 [4,-1,2,1] 的和最大，为 6 。
    
    示例 2：
    输入：nums = [1]
    输出：1
    ```
    
1. 代码：
    
    - 假设已经算出了 `dp[i-1]` ，如何推导出 `dp[i]`
    
    - **`dp[i]` 只有两种选择：**
        
        - 与前一个相邻子数组连接，形成一个和更大的子数组
        
        - 要么前一个相邻子数组的和是负数或0，这时则不与前一个相邻数子组连接，自己作为一个子数组，并抛弃前一个相邻子数组
        
    
    - 从两个选择中，取出和最大的子数组，由此得到状态转移方程：**`dp[i] = Math.max(dp[i - 1] + nums[i], nums[i]);`**
    
    ```Java
     public int maxSubArray(int[] nums) {
        if (nums.length == 0) return 0;
    
        // 状态的定义：dp[i] 表示以 nums[i] 结尾的连续子数组的最大和
        int[] dp = new int[nums.length];
    
        // 初始值 base case
        dp[0] = nums[0];
    
        // 穷举所有状态
        for (int i = 1; i < nums.length; i++) {
            // 状态转移方程
            dp[i] = Math.max(dp[i - 1] + nums[i], nums[i]);
        }
    
        // 返回 dp 数组中最大的值
        int res = Integer.MIN_VALUE;
        for (int i = 0; i < dp.length; i++) {
            if (dp[i] > res) {
                res = dp[i];
            }
        }
        return res;
    }
    ```
    
## **📑152. 乘积最大子数组**🔥

> [!info] 力扣  
> 
> [https://leetcode.cn/problems/maximum-product-subarray/description/?favorite=2cktkvj](https://leetcode.cn/problems/maximum-product-subarray/description/?favorite=2cktkvj)  
1. 问题描述：
    
    ```Plain
     给你一个整数数组 nums ，请你找出数组中乘积最大的非空连续子数组（该子数组中至少包含一个数字），并返回该子数组所对应的乘积。
    
    测试用例的答案是一个 32-位 整数。
    
    子数组 是数组的连续子序列。
    ```
    
1. 示例：
    
    ```Plain
    示例 1:
    输入: nums = [2,3,-2,4]
    输出: 6
    解释: 子数组 [2,3] 有最大乘积 6。
    
    示例 2:
    输入: nums = [-2,0,-1]
    输出: 0
    解释: 结果不能为 2, 因为 [-2,-1] 不是子数组。 
    ```
    
1. 代码：
    
    - **`dp[i]` 表示以下标 `i` 结尾的连续子序列的乘积的最大值**，最后把整个 dp 数组看一遍求最大值即可
    
    - 因此状态转移方程可能是：`dp[i] = max(dp[i - 1] * nums[i], nums[i])`
    
    - ⭐**但是 `nums[i]` 要分正负两种情况**
        
        - 当 `nums[i] >= 0` 时：`num[i]` 乘一个越大的数越大
        
        - 当 `nums[i] < 0` 时：`num[i]` 乘一个越小的数越大
        
    
    - 因此还需要记录一个数：**以下标 `i` 结尾的连续子序列的乘积的最小值**
    
    - 所以一维 dp 数组不够用了，要用一个二维 dp 数组
    
    ```Java
     public int maxProduct(int[] nums) {
        if (nums.length == 1) return nums[0];
        int[][] dp = new int[nums.length][2];
    
        dp[0][0] = nums[0];
        dp[0][1] = nums[0];
    
        for (int i = 1; i < nums.length; i++) {
            if (nums[i] >= 0) {
                // 当  nums[i] >= 0 时：最大乘积 为 当前数 * 之前的最大乘积
                dp[i][0]= Math.max(dp[i - 1][0] * nums[i], nums[i]);
                dp[i][1]= Math.min(dp[i - 1][1] * nums[i], nums[i]);
            } else {
                // 当  nums[i] < 0 时：最大乘积 为 当前数 * 之前的最小乘积
                dp[i][0]= Math.max(dp[i - 1][1] * nums[i], nums[i]);
                dp[i][1]= Math.min(dp[i - 1][0] * nums[i], nums[i]);
            }
        }
    
        // 取出最大值
        int res = 0;
        for (int i = 0; i < dp.length; i++) {
            res = Math.max(res, dp[i][0]);
        }
        return res;
    }
    ```

    ```Java
     public class BagProblem {
        public static void main(String[] args) {
            int[] weight = {1,3,4};
            int[] value = {15,20,30};
            int bagSize = 4;
            testWeightBagProblem(weight,value,bagSize);
        }
    
        /**
         * 动态规划获得结果
         * @param weight  物品的重量
         * @param value   物品的价值
         * @param bagSize 背包的容量
         */
        public static void testWeightBagProblem(int[] weight, int[] value, int bagSize){
    
            // 创建dp数组
            int goods = weight.length;  // 获取物品的数量
            int[][] dp = new int[goods][bagSize + 1];
    
            // 初始化dp数组
            // 创建数组后，其中默认的值就是0
            for (int j = weight[0]; j <= bagSize; j++) {
                dp[0][j] = value[0];
            }
    
            // 填充dp数组
            for (int i = 1; i < weight.length; i++) {
                for (int j = 1; j <= bagSize; j++) {
                    if (j < weight[i]) {
                        /**
                         * 当前背包的容量都没有当前物品i大的时候，是不放物品i的
                         * 那么前i-1个物品能放下的最大价值就是当前情况的最大价值
                         */
                        dp[i][j] = dp[i-1][j];
                    } else {
                        /**
                         * 当前背包的容量可以放下物品i
                         * 那么此时分两种情况：
                         *    1、不放物品i
                         *    2、放物品i
                         * 比较这两种情况下，哪种背包中物品的最大价值最大
                         */
                        dp[i][j] = Math.max(dp[i-1][j] , dp[i-1][j-weight[i]] + value[i]);
                    }
                }
            }
    
            // 打印dp数组
            for (int i = 0; i < goods; i++) {
                for (int j = 0; j <= bagSize; j++) {
                    System.out.print(dp[i][j] + "\t");
                }
                System.out.println("\n");
            }
        }
    }
    ```

1. 代码：

        - 背包的体积为 sum / 2
        
        - 背包要放入的商品（集合里的元素）重量为 元素的数值，价值也为元素的数值
        
        - 背包如果正好装满，说明找到了总和为 sum / 2 的子集。

    ```Java
     public class CanPartition {
        public boolean canPartition(int[] nums) {
            // 计算元素和的一半
            int sum = 0;
            for (int num : nums) {
                sum += num;
            }
            if (sum % 2 == 1) return false;
            int target = sum / 2;
    
            // 初始化 dp 数组
            int[][] dp = new int[nums.length][target + 1];
            for (int j = nums[0]; j < target; j++) {
                dp[0][j] = nums[0];
            }
            
            // 开始遍历
            for (int i = 1; i < nums.length; i++) {
                for (int j = 1; j <= target; j++) {
                    if (j < nums[i]) {
                        // 当前容量 j 放不下 nums[i]，则一定不会放
                        dp[i][j] = dp[i - 1][j];
                    } else {
                        // 当前容量 j 可以放下 nums[i]，则可能会放，也可能不会放
                        dp[i][j]  = Math.max(dp[i - 1][j], dp[i - 1][j - nums[i]] + nums[i]);
                    }
                }
            }
    
            return dp[dp.length - 1][target] == target;
        }
    }
    ```

    ```Java
        int n = nums.length;
        int[] dp = new int[n];
    
        dp[0] = 1;
    
        for (int i = 1; i < n; i++) {
            // 从当前位置挨个找，留下最大的
            for (int j = 1; j <= i; j++) {
                if (nums[i] > nums[i - j]) {
    ```

    - 注意：`dp[i][j]` 定了是是以 `matix[i][j]` 为右下角的正方形，在思考过程中不要乱，把不是它为底的元素也考虑进来
    
    ```Java
     public static int maximalSquare(char[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;
    
        // 定义dp数组
        // dp[i][j]：
        // 当 ij 为 1 时，以当前 ij 为右下角的正方形中全是 1 的正方形的最大边长
        // 当 ij 为 0 时，一定不会出现以当前 ij 为右下角的正方形中全是 1 的正方形
        int[][] dp = new int[m][n];
        
        // 初值
        dp[0][0] = matrix[0][0] == '1' ? 1 : 0;
        for (int i = 1; i < n; i++) {
            dp[0][i] = matrix[0][i] == '1' ? 1 : 0;
        }
        for (int i = 1; i < m; i++) {
            dp[i][0] = matrix[i][0] == '1' ? 1 : 0;
        }
    
        // 遍历
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][j] == '1') {
                    // 当 ij 为 1 时，dp[i][j] 由左上三个格子的最小值 + 1决定
                    dp[i][j] = Math.min(Math.min(dp[i - 1][j - 1], dp[i][j - 1]), dp[i - 1][j]) + 1;
                } else {
                    // 当 ij 为 0 时，一定不会出现以当前 ij 为右下角的正方形中全是 1 的正方形，所以 dp[i][j] = 0
                    dp[i][j] = 0;
                }
            }
        }
        
        // 取出 dp 中的最大边长
        int max = 0;
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                max = Math.max(dp[i][j], max);
            }
        }
        return max * max;
    }
    ```

## **📑1143. 最长公共子序列**
1. 问题描述：
    
    ```Plain
    给定两个字符串 text1 和 text2，返回这两个字符串的最长 公共子序列 的长度。如果不存在 公共子序列 ，返回 0 。
    
    一个字符串的 子序列 是指这样一个新的字符串：它是由原字符串在不改变字符的相对顺序的情况下删除某些字符（也可以不删除任何字符）后组成的新字符串。
    
    例如，"ace" 是 "abcde" 的子序列，但 "aec" 不是 "abcde" 的子序列。
    两个字符串的 公共子序列 是这两个字符串所共同拥有的子序列。
    ```
    
1. 示例：
    
    ```Plain
    示例 1：
    输入：text1 = "abcde", text2 = "ace" 
    输出：3  
    解释：最长公共子序列是 "ace" ，它的长度为 3 。
    ```
    
1. 代码：
    
    - 定义 dp[i][j] 表示： text1[0,i-1] 和 text2[0,j-1] 的最长公共子序列
        
        > [!important]
        > 
        > - 之所以 dp[i][j] 的定义不是 text1[0,i] 和 text2[0,j] ，是为了方便初始化，如果 dp[i][j] 的定义的是 text1[0,i] 和 text2[0,j]
        > 
        >     - 则 dp[0][j] 为 text1[0,0] 和 text2[0,j]， dp[i][0] 为 text1[0,i] 和 text2[0,0]
        > 
        >     - 所以第一行和第一列单独遍历一次，进行初始化
        > 
        > 
        > - 而如果 dp[i][j] 的定义的是 text1[0,i-1] 和 text2[0,j-1]
        > 
        >     - 则 dp[0][j] 为 text1[0,-1] 和 text2[0,j-1]， dp[i][0] 为 text1[0,i-1] 和 text2[0,-1]
        > 
        >     - 而 text1[0,-1] 和 text2[0,-1] 都是空串，所以 第一行和第一列都是 0，简化了初始化
        >  
        
    
    - 状态转移方程
        
        - 当 text1[i - 1] == text2[j - 1] 时，说明两个子字符串的最后一位相等，所以最长公共子序列又增加了 1，所以 dp[i][j] = dp[i - 1][j - 1] + 1
        
        - 当 text1[i - 1] != text2[j - 1] 时，说明两个子字符串的最后一位不相等，那么此时的状态 dp[i][j] 应该是 dp[i - 1][j] 和 dp[i][j - 1] 的最大值
            
            - 举个例子，比如对于 ace 和 bc 而言，他们的最长公共子序列的长度等于 ① ace 和 b 的最长公共子序列长度0 与 ② ac 和 bc 的最长公共子序列长度1 的最大值，即 1
            
        
    
    ```Java
     public int longestCommonSubsequence(String text1, String text2) {
    
        int m = text1.length();
        int n = text2.length();
    
        int[][] dp = new int[m + 1][n + 1];
    
        dp[0][0] = 0;
    
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }
        return dp[n - 1][m - 1];
    }
    ```
