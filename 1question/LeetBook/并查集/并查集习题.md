# 一、📑547. 省份数量
1. 问题描述：

    ```Plain
     有 n 个城市，其中一些彼此相连，另一些没有相连。如果城市 a 与城市 b 直接相连，且城市 b 与城市 c 直接相连，那么城市 a 与城市 c 间接相连。
     
     省份 是一组直接或间接相连的城市，组内不含其他没有相连的城市。
     
     给你一个 n x n 的矩阵 isConnected ，其中 isConnected[i][j] = 1 表示第 i 个城市和第 j 个城市直接相连，而 isConnected[i][j] = 0 表示二者不直接相连。
     
     返回矩阵中 省份 的数量。
    ```

2. 示例：  ```Plain
     输入：isConnected = [[1,1,0],[1,1,0],[0,0,1]]
     输出：2
    ```java
 
3. 代码：

    ```Java
     class findCircleNum {
     
         static ArrayList<Integer> list ;
     
         public static int findCircleNum(int[][] isConnected) {
             //初始化并查集
             list = new ArrayList<Integer>();
             for (int i = 0; i < isConnected.length; i++) {
                 list.add(i);
             }
             UnionFindSet<Integer> unionFindSet = new UnionFindSet<>(list);
     
             //遍历数组，如果为 1 ，且不在对角线上，则合并横纵坐标代表的省份
             for (int i = 0; i < isConnected.length; i++) {
                 for (int j = 0; j < isConnected[0].length; j++) {
                     if (i != j && isConnected[i][j] == 1){
                         unionFindSet.union(i, j);
                     }
                 }
             }
             //最后返回 rootMap 的大小，即集合的数量
             return unionFindSet.rootMap.size();
         }
     
         public static void main(String[] args) {
             int[][] ints = {{1, 1, 0}, {1, 1, 0}, {0, 0, 1}};
             int circleNum = findCircleNum(ints);
             System.out.println(circleNum);
         }
     
     }
     
     class Element<T>{
         public T value;
     
         public Element(T value) {
             this.value = value;
         }
     }
     
     class UnionFindSet<T>{
     
         public HashMap<T, Element<T>> elementMap;
         public HashMap<Element<T>, Element<T>> parentMap;
         public HashMap<Element<T>, Integer> rootMap;
     
         public UnionFindSet(List<T> list) {
             elementMap = new HashMap<>();
             parentMap = new HashMap<>();
             rootMap = new HashMap<>();
             for (T value : list) {
                 Element<T> element = new Element<T>(value);
                 elementMap.put(value, element);
                 parentMap.put(element, element);
                 rootMap.put(element, 1);
             }
         }
     
     
         private Element getRoot(Element<T> a) {
             Element root = a;
             while (parentMap.get(root) != root){
                 root = parentMap.get(root);
             }
             return root;
         }
     
         public boolean find(T a, T b){
             if (elementMap.containsKey(a) && elementMap.containsKey(b)) {
                 return getRoot(elementMap.get(a)) == getRoot(elementMap.get(a));
             }
             return false;
         }
     
         public void union(T a, T b){
             if (elementMap.containsKey(a) && elementMap.containsKey(b)) {
                 Element rootA = getRoot(elementMap.get(a));
                 Element rootB = getRoot(elementMap.get(b));
                 if (rootA != rootB){
                     Element big = rootMap.get(rootA) > rootMap.get(rootB) ? rootA : rootB;
                     Element small = rootA == big ? rootB : rootA;
                     parentMap.put(small, big);
                     rootMap.put(big, rootMap.get(big) + rootMap.get(small));
                     rootMap.remove(small);
                 }
             }
         }
     }
