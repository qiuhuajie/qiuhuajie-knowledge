## 2. 📑236. 二叉树的最近公共祖先🔥⚔️
> ℹ️ 力扣
> [https://leetcode.cn/problems/er-cha-shu-de-zui-jin-gong-gong-zu-xian-lcof/description/?favorite=xb9nqhhg](https://leetcode.cn/problems/er-cha-shu-de-zui-jin-gong-gong-zu-xian-lcof/description/?favorite=xb9nqhhg)
1. 问题描述：

    ```Plain
     给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。
     百度百科中最近公共祖先的定义为：“对于有根树 T 的两个节点 p、q，最近公共祖先表示为一个节点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”
    ```
2. 示例：

    ```Plain
     输入：root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 1
     输出：3
     解释：节点 5 和节点 1 的最近公共祖先是节点 3
    ```
3. 代码：
    - **最近公共祖先只有两种情况：**
        - 情况一：两个结点分别在最近公共祖先的左右子树上
        - 情况二：一个结点是另一个结点的祖先
    - 先写 **判断两个结点中是否有任意一个在给定树上** 的递归遍历

        ```Java
        //判断给定的两个结点是否有任意一个节点在给定树上
        public boolean traverse(TreeNode root, TreeNode p, TreeNode q){
            if (root == null){
                return false;
            }
            boolean left = traverse(root.left,p,q);
            boolean right = traverse(root.right,p,q);
            return left || right || (root.val == p.val || root.val == q.val);
        		//1.p或q刚好为当前结点
            //2.p或q在当前结点的左子树上
            //3.p或q在当前结点的右子树上
        }
        ```
    - 在上述代码中，再添加一个条件判断即可

        ```Java
        TreeNode ancestor = null;
        public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
            traverse(root,p,q);
            return ancestor;
        }
        public boolean traverse(TreeNode cur, TreeNode p, TreeNode q){
            if (cur == null){
                return false;
            }
            boolean left = traverse(cur.left,p,q);
            boolean right = traverse(cur.right,p,q);
            // 如果满足条件1和2中的一个，则root即为最近公共祖先
            // 条件1：p或q在root的左子树和右子树上，如果同时在左右出现，则说明p和q分别在root左右各一个
            if ((left && right)){
                ancestor = cur;
            }
            // 条件2：当前root为p或q中的一个，且另一个也在root的左右子树上，即q在p的子树上，或p在q的子树上
            if (((cur.val == p.val || cur.val == q.val) && (left || right))) {
                ancestor = cur;
            }
            return left || right || (cur.val == p.val || cur.val == q.val);
        }
        ```

    ```Plain
    给你二叉树的根结点 root ，请你将它展开为一个单链表：
    展开后的单链表应该同样使用 TreeNode ，其中 right 子指针指向链表中下一个结点，而左子指针始终为 null 。
    展开后的单链表应该与二叉树 先序遍历 顺序相同。
    ```
2. 示例：

    ```Plain

    输出：[1,null,2,null,3,null,4,null,5,null,6]
    ```
3. 代码：

    ```Java
    public void flatten(TreeNode root) {
        if (root == null) return;
        // 先得到先序序列
        LinkedList<TreeNode> list = new LinkedList<>();
        preOder(root, list);
        // 用先序序列构造单链
        TreeNode p = list.removeFirst();
        while (!list.isEmpty()) {
            p.right = list.removeFirst();
            p.left = null;
            p = p.right;
        }
    }
    private void preOder(TreeNode cur, LinkedList<TreeNode> list) {
        if (cur == null) {
            return;
        }
        // 这里直接把原节点加入即可
        list.add(cur);
        preOder(cur.left, list);
        preOder(cur.right, list);
    }
    ```

## 6. 📑剑指 Offer 34. 二叉树中和为某一值的路径⚔️
1. 问题描述：

    ```Plain
    给你二叉树的根节点 root 和一个整数目标和 targetSum ，找出所有 从根节点到叶子节点 路径总和等于给定目标和的路径。
    叶子节点 是指没有子节点的节点。
    ```
2. 示例：

    ```Plain
    示例 1：
    输入：root = [5,4,8,11,null,13,4,7,2,null,null,5,1], targetSum = 22
    输出：[[5,4,11,2],[5,8,4,5]]
    ```
3. 代码：**二叉树回溯**

    ```Java
     // 结果集
    List<List<Integer>> res;
    public List<List<Integer>> pathSum(TreeNode root, int target) {
        res = new ArrayList<>();
        // 路径保存
        LinkedList<Integer> trace = new LinkedList<>();
        process(trace, root, target);
        return res;
    }
    private void process(LinkedList<Integer> trace, TreeNode cur, int target) {
        // 由于树结构在遍历选择空间时，没法像模板中的使用 fori 控制
        // 这里使用前序遍历，处理当前节点
        if (cur == null) return;
        // 做选择
        trace.add(cur.val);
        target -= cur.val;
        // 记录结果集（target 消耗完，且当前节点是叶子节点）
        if (target == 0 && cur.left == null && cur.right == null) {
            res.add(new LinkedList(trace));
        }
        process(trace, cur.left, target);
        process(trace, cur.right, target);
        // 回溯，撤回当前选择
        trace.removeLast();
    }
    ```

    ```Java
            if (root == null){
                return false;
            }
    				// 遍历到叶子结点时，判断
    ```

    ```Java
     public class PathSumIII {
        // 双递归
        // 第一个递归，遍历所有节点，统计以当前节点为根的树的符合条件的路径数量
        public static int pathSum(TreeNode root, int targetSum) {
            if (root == null) return 0;
            count(root, (long) targetSum);
            int count = sum;
            sum = 0;
            count += pathSum(root.left, targetSum);
            count += pathSum(root.right, targetSum);
            return count;
        }
        static int sum = 0;
        // 第二个递归，统计以 node 为根的树的路径，用 sum 记录
        private static void count(TreeNode node, Long targetSum) {
            if (node == null) return;
            if (targetSum == node.val) sum++;
            count(node.left, targetSum - node.val);
            count(node.right, targetSum - node.val);
        }
    }
    ```

1. 问题描述：

    ```Plain
    给你一棵二叉树的根节点，返回该树的 直径 。
    二叉树的 直径 是指树中任意两个节点之间最长路径的 长度 。这条路径可能经过也可能不经过根节点 root 。
    两节点之间路径的 长度 由它们之间边数表示。
    ```
2. 代码：**递归求树的深度，同时保存目标结果**

    ```Java
     public class DiameterOfBinaryTree {
        int res;
        public int diameterOfBinaryTree(TreeNode root) {
            getDepth(root);
            return res;
        }
        private int getDepth(TreeNode node) {
            if (node == null) return 0;
            int L = getDepth(node.left);
            int R = getDepth(node.right);
            res = Math.max(res, R + L + 1);
            return Math.max(R, L) + 1;
        }
    }
    ```

## 1. 📑剑指 Offer 26. 树的子结构⚔️
1. 问题描述：

    ```Plain
    输入两棵二叉树A和B，判断B是不是A的子结构。(约定空树不是任意一个树的子结构)
    B是A的子结构， 即 A中有出现和B相同的结构和节点值。
    ```
2. 示例：

    ```Plain
     例如:
    给定的树 A:
         3
        / \
       4   5
      / \
     1   2
    给定的树 B：
       4 
      /
     1
    返回 true，因为 B 与 A 的一个子树拥有相同的结构和节点值。
    ```
3. 代码：
    - 开始的思路：比较两树的中序遍历序列：但是 `2，3，2` 和 `3，null，2，2` 两棵树的中序遍历序列是一样的，但是结构不同😢
    - **若树 B 是树 A 的子结构，则必满足以下三种情况之一，因此用或** **`||`** **连接**
        - 以 节点 A 为根节点的子树 包含树 B ，对应 `compare(A, B)`
        - 树 B 是 树 A 左子树 的子结构，对应 `isSubStructure(A.left, B)`
        - 树 B 是 树 A 右子树 的子结构，对应 `isSubStructure(A.right, B)`

    ```Java
    public boolean isSubStructure(TreeNode A, TreeNode B) {
        if (A == null || B == null) return false;
        return compare(A, B) || isSubStructure(A.left, B) || isSubStructure(A.right, B);
    }
    // A，B 从同一个节点开始匹配
    private boolean compare(TreeNode A, TreeNode B) {
        // 树 B 已匹配完成（越过叶子节点），因此返回 true
        if(B == null) return true;
        // 树 B 没有匹配完，树 A 却已匹配完，返回 false
        if (A == null) return false;
        // 节点值不同，返回 false
        if (A.val != B.val) return false;
        return compare(A.left, B.left) && compare(A.right, B.right);
    }
    ```
## 2. 📑617. 合并二叉树🔥
1. 问题描述：

    ```Plain
    给你两棵二叉树： root1 和 root2 。
    想象一下，当你将其中一棵覆盖到另一棵之上时，两棵树上的一些节点将会重叠（而另一些不会）。你需要将这两棵树合并成一棵新二叉树。合并的规则是：如果两个节点重叠，那么将这两个节点的值相加作为合并后节点的新值；否则，不为 null 的节点将直接作为新二叉树的节点。
    返回合并后的二叉树。
    注意: 合并过程必须从两个树的根节点开始。
    ```
2. 示例：

    ![[IMG-20260620195906880.png|753]]

    ```Plain
    输入：root1 = [1,3,2,5], root2 = [2,1,3,null,4,null,7]
    输出：[3,4,5,5,4,null,7]
    ```
3. 代码：
    - 把一个树往另一棵树上合并

    ```Java
    public TreeNode mergeTrees(TreeNode root1, TreeNode root2) {
        return preOrder(root1, root2);
    }
    private TreeNode preOrder(TreeNode root1, TreeNode root2) {
        // 如果其中一棵树的一边为 null 另一边不是 null，那么不论非 null 的上面是啥，将指针指向非 null上就完成合并了
        if (root1 == null) return root2;
        if (root2 == null) return root1;
        root1.val += root2.val;
        root1.left = preOrder(root1.left, root2.left);
        root1.right = preOrder(root1.right, root2.right);
        return root1;
    }
    ```

    ```Java
        int root = pre[pre1];
        //构造当前根节点
        TreeNode node = new TreeNode(root);
        //从中序序列中找出根的索引
        int rootIndex = inorderMap.get(root);
        // 递归构造左右子树
        node.left = build(pre1 + 1, pre1 + rootIndex - in1, in1, rootIndex - 1);
        node.right = build(pre1 + rootIndex - in1 + 1, pre2, rootIndex + 1, in2);
        return node;
    }
    ```
## 2. 📑297. 二叉树的序列化和反序列化🔥⚔️😡

> [https://leetcode.cn/problems/serialize-and-deserialize-binary-tree/description/?favorite=2cktkvj](https://leetcode.cn/problems/serialize-and-deserialize-binary-tree/description/?favorite=2cktkvj)
1. 问题描述：

    ```Plain
     序列化是将一个数据结构或者对象转换为连续的比特位的操作，进而可以将转换后的数据存储在一个文件或者内存中，同时也可以通过网络传输到另一个计算机环境，采取相反方式重构得到原数据。
     请设计一个算法来实现二叉树的序列化与反序列化。这里不限定你的序列 / 反序列化算法执行逻辑，你只需要保证一个二叉树可以被序列化为一个字符串并且将这个字符串反序列化为原始的树结构。
     提示: 输入输出格式与 LeetCode 目前使用的方式一致，详情请参阅 LeetCode 序列化二叉树的格式。你并非必须采取这种方式，你也可以采用其他的方法解决这个问题
    ```
2. 示例：

    ```Plain
     输入：root = [1,2,3,null,null,4,5]
     输出：[1,2,3,null,null,4,5]
    ```
3. 代码：**前序遍历**
    - 一般来说，只有前序遍历结果是无法还原二叉树的，因为缺少空指针信息，至少要得到前、中、后序遍历的两种才能还原二叉树
    - 但这里的 node 列表包含空指针信息（相当于满二叉树生成的前序序列），所以只用 node 列表就可以还原

    ```Java
    public class Codec {
        // Encodes a tree to a single string.
        StringBuilder sb = null;
        public String serialize(TreeNode root) {
            if (root == null) return null;
            sb = new StringBuilder();
            preOrder(root);
            return sb.toString();
        }
        public void preOrder(TreeNode root) {
            // 递归出口
            if (root == null) {
                sb.append("null,");
                return;
            }
            // 先序遍历
            sb.append(root.val).append(",");
            preOrder(root.left);
            preOrder(root.right);
        }
        // Decodes your encoded data to tree.
        LinkedList<String> list = null;
        public TreeNode deserialize(String data) {
            if (data == null) return null;
            // 使用链表保存字符串
            list = new LinkedList<>();
            String[] split = data.split(",");
            Collections.addAll(list, split);
            return deserialize();
        }
        public TreeNode deserialize() {
            // 递归出口
            if (list.isEmpty()) return null;
            // 先序遍历（确定根节点 root，然后递归生成左右子树）
            String s = list.removeFirst();
            if (s.equals("null")) return null;
            TreeNode treeNode = new TreeNode(Integer.parseInt(s));
            treeNode.left = deserialize();
            treeNode.right = deserialize();
            return treeNode;
        }
    }
    ```
