# 一、📑647. 回文子串
> ℹ️ 力扣（LeetCode）官网 - 全球极客挚爱的技术成长平台
> 备战技术面试？力扣提供海量技术面试资源，帮助你高效提升编程技能，轻松拿下世界 IT 名企 Dream Offer。
> [https://leetcode.cn/problems/palindromic-substrings/](https://leetcode.cn/problems/palindromic-substrings/)
1. 问题描述：

    ```Plain
    给你一个字符串 s ，请你统计并返回这个字符串中 回文子串 的数目。
    回文字符串 是正着读和倒过来读一样的字符串。
    子字符串 是字符串中的由连续字符组成的一个序列。
    具有不同开始位置或结束位置的子串，即使是由相同的字符组成，也会被视作不同的子串。
    ```
2. 示例：

    ```Plain
    输入：s = "abc"
    输出：3
    解释：三个回文子串: "a", "b", "c
    ```
3. 代码：

    ```Java
    int count = 0;
    public int countSubstrings(String s) {
        for (int i = 0; i < s.length(); i++) {
            expand(s, i, i);
            expand(s, i, i + 1);
        }
        return count;
    }
    public void expand(String s, int left, int right) {
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            left--;
            right++;
            count++;
        }
    }
    ```
# 二、📑5. 最长回文子串🔥✔️
> ℹ️ 力扣
> [https://leetcode.cn/problems/longest-palindromic-substring/](https://leetcode.cn/problems/longest-palindromic-substring/)
1. 问题描述：

    ```Plain
     给你一个字符串 s，找到 s 中最长的回文子串。
    ```
2. 示例：

    ```Plain
     示例 1：
     输入：s = "babad"
     输出："bab"
     解释："aba" 同样是符合题意的答案。
     示例 2：
     输入：s = "cbbd"
     输出："bb"
     示例 3：
     输入：s = "a"
     输出："a"
     示例 4：
     输入：s = "ac"
     输出："a"
    ```
3. 思路
    1. 难点：回文串的长度可能是奇数，也可能是偶数
    2. 问题解决核心：**从中间开始向两边扩散来判断回文 + 双指针**
    3. 解决思想：
        1. 遍历字符串的所有位置
        2. 从当前遍历到的位置，使用双指针的技巧，向两侧扩展，找到以当前位置为中心的回文串
        3. 将每次返回的回文串与当前结果比较，保留最长的结果
        4. ==但是由于回文串的长度可能是奇数也可能是偶数，如果是 abba 这种回文串，没有一个中心字符，根本不会被检测到==
        5. ==故要，将扩展函数执行两次：==

            ```Java
            String sr = expand(s, i, i);
            String sl = expand(s, i, i + 1);
            ```
    4. 说明
        1. 这个问题还可以用动态规划解决，时间复杂度一样，但是空间复杂度至少要 $O(N^2)$ 来存储 DP table ，这道题是**少有的动态规划非最优解法的问题**
        2. 还有一个巧妙的解法：Manacher 算法，但比较复杂
4. 代码：**中心扩展**

    ```Java
    public static String longestPalindrome(String s){
        String res = "";
        for (int i = 0; i < s.length(); i++) {
    				// 扩展函数执行两次，这样可以实现同时处理回文串为奇数和偶数的情况
            String sr = expand(s, i, i);
            String sl = expand(s, i, i + 1);
            // 保留最大的回文串
            res = res.length() > sr.length() ? res : sr;
            res = res.length() > sl.length() ? res : sl;
        }
        return res;
    }
    // 扩展函数
    public static String expand(String s, int left, int right){
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)){
            left--;
            right++;
        }
        // 将本次扩展得到的回文串返回
        return s.substring(left + 1, right);
    }
    ```
    - `substring(int beginIndex, int endIndex)` 函数的截取范围：`[beginIndex, endIndex)`
    - Params:
        - beginIndex – the beginning index, ==inclusive.==
        - endIndex – the ending index, ==exclusive.==

# 五、📑剑指 Offer 05. 替换空格⚔️✔️
> ℹ️ 力扣
> [https://leetcode.cn/problems/ti-huan-kong-ge-lcof/](https://leetcode.cn/problems/ti-huan-kong-ge-lcof/)
1. 问题描述：

    ```Plain
    请实现一个函数，把字符串 s 中的每个空格替换成"%20"。
    ```
2. 示例：

    ```Plain
    示例 1：
    输入：s = "We are happy."
    输出："We%20are%20happy."
    ```
3. 代码：遍历替换
    - **`StringBuilder`** **的应用**

    ```Java
    class Solution {
        public String replaceSpace(String s) {
            StringBuilder res = new StringBuilder();
            for(Character c : s.toCharArray())
            {
                if(c == ' ') res.append("%20");
                else res.append(c);
            }
            return res.toString();
        }
    }
    ```
