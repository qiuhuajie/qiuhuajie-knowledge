# 一、📑插入操作
1. 在前缀树中插入一个字符串 `S`
    1. 要从根节点开始
    2. 将根据 `S[0]`（S中的第一个字符），选择一个子节点或添加一个新的子节点
    3. 然后到达第二个节点，并根据 `S[1]` 做出选择
    4. 再到第三个节点，以此类推
    5. 最后，依次遍历 `S` 中的所有字符并到达末尾。 末端节点将是表示字符串 `S` 的节点
2. 示例：

    ![[IMG-20260620210446613.png|286]]

3. 代码：

    ```Java
     public void insert(String word) {
         if (word == null){
             return;
         }
         //将要插入的字符串转为为字符数组
         char[] chars = word.toCharArray();
         //遍历指针
         TrieNode node = root;
         //根节点+1，插入到最后根节点的path值就是树中的字符串总数
         root.path++;
         int index = 0;
         //遍历每一个字符
         for (int i = 0; i < chars.length; i++) {
             //获得当前字符 在子节点数组中的索引位置
             index = chars[i] - 'a';
             //如果当前字符不存在，则new一个新节点，放在子节点数组中
             if (node.next[index] == null){
                 TrieNode newTrieNode = new TrieNode();
                 node.next[index] = newTrieNode;
             }
             node.path++;    //沿途+1
             node = node.next[index];    //指针指向下一个节点
         }
         node.end++;     //字符数组遍历完后最后走到的节点end+1
     }
    ```
# 二、📑搜索操作
1. 所有节点的后代都与该节点相对应字符串的有着共同前缀。因此，很容易搜索以特定前缀开头的任何单词
2. 同样地，可以根据给定的前缀沿着树形结构搜索下去
    1. 一旦找不到我们想要的子节点，搜索就以失败终止
    2. 否则，搜索成功

4. 代码：
    > 搜索和插入的代码框架类似，都需要初始化指针、遍历字符数组、根据子节点数组中 字符对应索引上 的存储情况做处理

    ```Java
     public boolean search(String word) {
         if (word == null){
             return false;
         }
         char[] chars = word.toCharArray();
         TrieNode node = root;
         int index = 0;
         for (int i = 0; i < chars.length; i++) {
             index = chars[i] - 'a';
             if (node.next[index] == null){
                 return false;   //当字符的对应索引，在子节点数组上为空，则搜索失败
             }
             node = node.next[index];
         }
         //如果最后end为空，说明不存在word，只是存在以word为前缀的字符串
         return node.end != 0;
     }
    ```
# 三、📑删除操作
```Java
public void delete(String word) {
    //删除之前一定要保证word是存在的
    if (search(word)) {
        char[] chs = word.toCharArray();
        TrieNode node = root;
        int index = 0;
        for (int i = 0; i < chs.length; i++) {
            index = chs[i] - 'a';
            if (--node.next[index].path == 0) {    //走到某点pass=0了，意味着之后不再会有了，直接在此处断开
                node.next[index] = null;   //引用标空，垃圾自动回收
                return;
            }
            node = node.next[index];
        }
        node.end--;
    }
}
```

```Java
public boolean startsWith(String prefix) {
    if (prefix == null){
        return false;
    }
    char[] chars = prefix.toCharArray();
    TrieNode node = root;
    int index = 0;
    for (int i = 0; i < chars.length; i++) {

        }

    }
```

```Java
public int prefixNumber(String pre) {
```

# 五、📑208. 实现 Trie 前缀树
1. 问题描述：

    ```Plain
    Trie（发音类似 "try"）或者说 前缀树 是一种树形数据结构，用于高效地存储和检索字符串数据集中的键。这一数据结构有相当多的应用情景，例如自动补完和拼写检查。
    请你实现 Trie 类：
    Trie() 初始化前缀树对象。
    void insert(String word) 向前缀树中插入字符串 word 。
    boolean search(String word) 如果字符串 word 在前缀树中，返回 true（即，在检索之前已经插入）；否则，返回 false 。
    boolean startsWith(String prefix) 如果之前已经插入的字符串 word 的前缀之一为 prefix ，返回 true ；否则，返回 false 。
    ```
2. 示例：

    ```Plain
    输入
    ["Trie", "insert", "search", "search", "startsWith", "insert", "search"]
    [[], ["apple"], ["apple"], ["app"], ["app"], ["app"], ["app"]]
    输出
    [null, null, true, false, true, null, true]
    解释
    Trie trie = new Trie();
    trie.insert("apple");
    trie.search("apple");   // 返回 True
    trie.search("app");     // 返回 False
    trie.startsWith("app"); // 返回 True
    trie.insert("app");
    trie.search("app");     // 返回 True
    ```
3. 代码：

    ```Java
    class TrieNode{
        int path = 0;
        int end = 0;
        TrieNode[] next = new TrieNode[26]; //如果字符种类很多，可以换成hashMap：HashMap<char, TrieNode> map
    }
    class Trie {
        TrieNode root;
        public Trie() {
            //构建前缀树实际上就是多次调用插入函数。但记住在插入字符串之前要 初始化根节点
            this.root = new TrieNode();
        }
        public void insert(String word) {
            if (word == null){
                return;
            }
            //将要插入的字符串转为为字符数组
            char[] chars = word.toCharArray();
            //遍历指针
            TrieNode node = root;
            //根节点+1，插入到最后根节点的path值就是树中的字符串总数
            root.path++;
            int index = 0;
            //遍历每一个字符
            for (int i = 0; i < chars.length; i++) {
                //获得当前字符 在子节点数组中的索引位置
                index = chars[i] - 'a';
                //如果当前字符不存在，则new一个新节点，放在子节点数组中
                if (node.next[index] == null){
                    TrieNode newTrieNode = new TrieNode();
                    node.next[index] = newTrieNode;
                }
                node.path++;    //沿途+1
                node = node.next[index];    //指针指向下一个节点
            }
            node.end++;     //字符数组遍历完后最后走到的节点end+1
        }
        public boolean search(String word) {
            if (word == null){
                return false;
            }
            char[] chars = word.toCharArray();
            TrieNode node = root;
            int index = 0;
            for (int i = 0; i < chars.length; i++) {
                index = chars[i] - 'a';
                if (node.next[index] == null){
                    return false;   //当字符的对应索引，在子节点数组上为空，则搜索失败
                }
                node = node.next[index];
            }
            //如果最后end为空，说明不存在word，只是存在以word为前缀的字符串
            return node.end != 0;
        }
        public boolean startsWith(String prefix) {
            if (prefix == null){
                return false;
            }
            char[] chars = prefix.toCharArray();
            TrieNode node = root;
            int index = 0;
            for (int i = 0; i < chars.length; i++) {
                index = chars[i] - 'a';
                if (node.next[index] == null){
                    return false;
                }
                node = node.next[index];
            }
            return true;
        }
    }
    ```
