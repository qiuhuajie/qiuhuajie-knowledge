# 1. 算法介绍
## 1.1 核心思想
1. 哨兵划分
    
    1. 以数组某个元素（一般选取首元素）为 **基准数** ，将所有小于基准数的元素移动至其左边，大于基准数的元素移动至其右边
    
    1. 通过一轮 **哨兵划分** ，可将数组排序问题拆分为 **两个较短数组的排序问题** （本文称之为左（右）子数组）
    
1. 递归
    
    1. 对 **左子数组** 和 **右子数组** 分别递归执行 **哨兵划分**，直至子数组长度为 1 时终止递归，即可完成对整个数组的排序
    
    1. 如下图所示，为示例数组 `[2,4,1,0,3,5]` 的快速排序流程。观察发现，快速排序和 **二分法** 的原理类似，都是以 $log$ 时间复杂度实现搜索区间缩小

## 1.2 基本快排⭐
```Java

```
```Java
public static void quickSort(int[] nums, int left, int right) {
    // 递归出口，只剩一个元素时返回
    if (left >= right) return;
    // 划分
    int i = partition(nums, left, right);
    // 对划分后的左右子数组，进一步划分
    quickSort(nums, left, i - 1);
    quickSort(nums, i + 1, right);
}
public static int partition(int[] nums, int left, int right) {
    // 以数组的第一个元素为基准
    int p = left;
    while (left < right) {
        // 从右往左找到第一个小于基准的元素（要先移动 right！）
        while (left < right && nums[right] >= nums[p]) right--;
        // 从左往右找到第一个大于基准的元素
        while (left < right && nums[left] <= nums[p]) left++;
        // 交换找到的这两个元素
        swap(nums, left, right);
    }
    // 交换最终位置的元素与基准元素
    swap(nums, p, left);
    // 返回最后基准的位置
    return left;
}
// 交换函数（将原数组也要传入！）
private static void swap(int[] nums, int a, int b) {
    int temp = nums[a];
    nums[a] = nums[b];
    nums[b] = temp;
}
```
## 1.3 优化：随机基准数⭐
1. 存在的问题
    
    1. 由于快速排序每轮选取「子数组最左元素」作为「基准数」，因此在输入数组 **完全有序** 或 **完全倒序** 时
    
    1. `partition()` 每轮只划分一个元素，达到最差时间复杂度 $O(N^2)$
    
    1. 例如：912 题中，基础的排序算法会超时
        
        ![[IMG-20260405035401610.png|Untitled 94.png]]
        
    
1. 优化方法
    
    1. 可使用 **随机函数** ，每轮在子数组中随机选择一个元素作为基准数，这样就可以极大概率避免以上劣化情况
    
    1. 值得注意的是，由于仍然可能出现最差情况，因此快速排序的**最差**时间复杂度仍为 $O(N^2)$
    
1. 代码仅需修改 `partition()` 方法
    
    ```Java
    //在输入数组完全有序或完全倒序 时，partition() 每轮只划分一个元素，达到最差时间复杂度 O(N^2)
    public static int partition(int[] nums, int left, int right) {
    		// 随机选择基准
        int randomIndex = (int) (left + Math.random() * (right - left + 1));
        swap(nums, randomIndex, left);
        int p = left;
        while (left < right) {
            while (left < right && nums[right] >= nums[p]) right--;
            while (left < right && nums[left] <= nums[p]) left++;
            swap(nums, left, right);
        }
        swap(nums, p, left);
        return left;
    }
    ```

## **📑75. 颜色分类**🔥

> [!info] 力扣  
>  
> [https://leetcode.cn/problems/sort-colors/?favorite=2cktkvj](https://leetcode.cn/problems/sort-colors/?favorite=2cktkvj)  
1. 问题描述：
    
    ```Plain
    给定一个包含红色、白色和蓝色、共 n 个元素的数组 nums ，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。
    
    我们使用整数 0、 1 和 2 分别表示红色、白色和蓝色。
    
    必须在不使用库内置的 sort 函数的情况下解决这个问题。
    
    进阶：
    你能想出一个仅使用常数空间的一趟扫描算法吗？
    ```
    
1. 示例：
    
    ```Plain
    示例 1：
    输入：nums = [2,0,2,1,1,0]
    输出：[0,0,1,1,2,2]
    
    示例 2：
    输入：nums = [2,0,1]
    输出：[0,1,2]
    ```
    
1. 代码：
    
    1. **基本快排（略），但是不满足进阶的要求**
    
    1. **三路快排**
        
        1. 分别设置两个索引 zero 和 two，保证下标 0 到 zero 对应的数组元素都为 0，下标 two 到 numsSize - 1 对应的数组元素都是 2
        
        1. 设置索引 i 用于遍历整个数组，遍历到 i 个元素的时候，保证下标 zero + 1 到 i - 1 对应的数组元素都是 1
        
        1. 当索引 i 遍历到元素 1 时，直接将元素 1 纳入到属于 1 的部分，然后继续遍历 `i++`
        
        1. 如果遍历到元素 2 时，将 two 前面的索引 `two - 1` 对应的元素与遍历到的元素 2 交换，将元素 2 纳入到属于 2 的部分
        
        1. 如果遍历到元素 0 时，交换下标为 zero 后面的索引 `zero + 1` 对应的元素与遍历到的元素 0，将元素 0 纳入到属于 0 的部分，==同时 i 后移== `i++`
        
        ```Java
        public static void sortColors(int[] nums) {
            // [0,zero] 保存 0
            int zero = -1;
            // [zero,two] 保存 1
            // [two,nums.length - 1] 保存 2
            int two = nums.length;
        
            // 一次遍历
            for (int i = 0; i < two;) {
                if (nums[i] == 1) {
                    i++;
                } else if (nums[i] == 0) {
                    swap(nums, ++zero, i);
                    i++;  // ⭐
                } else if (nums[i] == 2) {
                    swap(nums, --two, i);
                }
            }
        }
        
        // 交换函数
        private static void swap(int[] nums, int a, int b) {
            int temp = nums[a];
            nums[a] = nums[b];
            nums[b] = temp;
        }
        ```
        
    
## **📑215. 数组中的第 K 个最大元素**🔥
1. 问题描述：
    
    ```Plain
    给定整数数组 nums 和整数 k，请返回数组中第 k 个最大的元素。
    
    请注意，你需要找的是数组排序后的第 k 个最大的元素，而不是第 k 个不同的元素。
    
    你必须设计并实现时间复杂度为 O(n) 的算法解决此问题。
    ```
    
1. 示例：
    
    ```Plain
    示例 1:
    输入: [3,2,1,5,6,4], k = 2
    输出: 5
    
    示例 2:
    输入: [3,2,3,1,2,4,5,5,6], k = 4
    输出: 4
    ```
    
1. 代码：
    
    1. [[优先队列（堆）]]
    
    1. **快排后取出倒数第 k 个**
        
        ```Java
         public static int findKthLargest(int[] nums, int k) {
            quickSort(nums, 0, nums.length - 1);
            return nums[nums.length - k];
        }
        ```
        
    
    1. **快排基础上使用二分进行优化⭐（实习美团二面）**
        
        ```Java
        int k = 0;
        int N = 0;
        public int findKth(int[] a, int n, int K) {
            k = K;
            N = n;
            return quickSort(a, 0, n - 1);
        }
        
        public int quickSort(int[] nums, int left, int right) {
        
            // 划分
            int i = partition(nums, left, right);
        		
        		// 用到了二分查找的思想，不断接近 len - k 的位置，最终返回结果
        		// partition 能直接定位到 len - k 的位置就找到了结果
            if (i == N - k) {
                return nums[i];
            } else if (i < N - k) {
                return quickSort(nums, i + 1, right);
            } else {
                return quickSort(nums, left, i - 1);
            }
        }
        
        public int partition(int[] nums, int left, int right) {
            // 以数组的第一个元素为基准
            int p = left;
            while (left < right) {
                // 从右往左找到第一个小于基准的元素
                while (left < right && nums[right] >= nums[p]) right--;
                // 从左往右找到第一个大于基准的元素
                while (left < right && nums[left] <= nums[p]) left++;
                // 交换找到的这两个元素
                swap(nums, left, right);
            }
            // 交换最终位置的元素与基准元素
            swap(nums, p, left);
            // 返回最后基准的位置
            return left;
        }
        
        // 交换函数
        private void swap(int[] nums, int a, int b) {
            int temp = nums[a];
            nums[a] = nums[b];
            nums[b] = temp;
        }
        ```
