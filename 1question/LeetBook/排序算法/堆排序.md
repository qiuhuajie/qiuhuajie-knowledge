- **🙋‍♂️为什么现实中多用快排，堆排序用的不多❓**
    
    1. 性能优势：
        
        1. 在大多数情况下，快速排序的性能表现更好。它的平均时间复杂度为O(nlogn)，并且在大部分情况下具有较好的实际表现
        
        1. 相比之下，堆排序的平均时间复杂度也为O(nlogn)，但是由于其对数据的访问方式不连续，对于缓存的利用不够友好，因此实际性能可能会稍逊于快速排序
        
    
    1. 空间复杂度：
        
        1. 快速排序是原地排序算法，不需要额外的空间开销，只需要基于原数组进行元素交换和划分操作
        
        1. 而堆排序需要使用额外的堆数据结构来构建堆，因此需要更多的空间
        
    
    1. 可控性和稳定性：
        
        1. 堆排序是一种不稳定的排序算法，即对于相同的元素可能无法保持其原有的相对顺序
        
        1. 而快速排序可以通过合理地选择划分元素来保持相同元素的相对顺序，并且具有比较好的可控性。
        
    
    1. 实现简单：
        
        1. 相对于堆排序，快速排序的实现更为简单和直观，只需要基于递归或迭代的方式进行分治操作和元素交换即可
        
        1. 而堆排序需要实现堆结构的构建和调整过程，相对来说更为复杂
        
    
- 虽然快速排序在大多数情况下更常用，但是堆排序仍然具有一些特殊的应用场景和优势
    
    - 例如在需要实时维护一个动态数据集合的最大或最小值时，堆排序相对更适用

public class HeapSort {
    public static void heapSort(int[] array) {
        int n = array.length;
        // 1️⃣从无序堆构建大根堆
        // 从第一个有子节点的节点（上面图中从无序堆的下标为n / 2 - 1 = 4 的节点开始）开始，递归调整，将原数组 array 构建成一个大根堆
        for (int i = n / 2 - 1; i >= 0; i--) {
            heapify(array, n, i);
        }
        // 2️⃣堆排序，每遍历一次当前从根交换下来的元素，就会被认为移除堆了，那么下一个最后一个节点的下标就是 i--
        for (int i = n - 1; i >= 0; i--) {
            // 每次取出最后一个节点与根交换（根元素是当前最大，与最后一个元素交换后，最大元素就排到了最终位置）
            swap(array, 0, i);
            // 调整剩余元素的最大堆
            heapify(array, i, 0);
        }
    }
		// 调整堆⭐
    private static void heapify(int[] array, int n, int i) {
        int largest = i; // 假设最大值为根节点
        int leftChild = 2 * i + 1; // 左孩子节点
        int rightChild = 2 * i + 2; // 右孩子节点
        // 如果左孩子比根节点大，更新最大值
        if (leftChild < n && array[leftChild] > array[largest]) {
            largest = leftChild;
        }
        // 如果右孩子比根节点大，更新最大值
        if (rightChild < n && array[rightChild] > array[largest]) {
            largest = rightChild;
        }
        // 如果最大值不是根节点，则交换根节点和最大值，并递归调整子树
        if (largest != i) {
            swap(array, largest, i);
            heapify(array, n, largest);
        }
    }
    private static void swap(int[] array, int a, int b) {
        int temp = array[a];
        array[a] = array[b];
        array[b] = temp;
    }
    public static void main(String[] args) {
        int[] ints = {1, 5, 3, 7, 5, 9, 2};
        heapSort(ints);
        for (int anInt : ints) {
            System.out.println(anInt);
        }
    }
}
