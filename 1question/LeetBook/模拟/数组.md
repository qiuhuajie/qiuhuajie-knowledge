# 一、📑88. 合并有序数组
> ℹ️ 力扣（LeetCode）官网 - 全球极客挚爱的技术成长平台

> [https://leetcode.cn/problems/merge-sorted-array/description/](https://leetcode.cn/problems/merge-sorted-array/description/)
1. 问题描述：

    ```Plain
     给你两个按 非递减顺序 排列的整数数组 nums1 和 nums2，另有两个整数 m 和 n ，分别表示 nums1 和 nums2 中的元素数目。
    请你 合并 nums2 到 nums1 中，使合并后的数组同样按 非递减顺序 排列
    ```
2. 示例：

    ```Plain
    输入：nums1 = [1,2,3,0,0,0], m = 3, nums2 = [2,5,6], n = 3
    输出：[1,2,2,3,5,6]
    解释：需要合并 [1,2,3] 和 [2,5,6] 。
    合并结果是 [1,2,2,3,5,6] ，其中斜体加粗标注的为 nums1 中的元素。
    ```
3. 代码：

    ![[IMG-20260620205448578.png|476]]

    原地版本

    ![[IMG-20260620205448636.png|512]]

    ![[IMG-20260620205448671.png|479]]

    ```Java
     class Solution {
        public void merge(int[] nums1, int m, int[] nums2, int n) {
            int[] res = new int[m + n];
            int p1 = 0;
            int p2 = 0;
            int p = 0;
    				// 双指针
            while (p1 != m && p2 != n) {
                if (nums1[p1] <= nums2[p2]) {
                    res[p++] = nums1[p1++];
                } else {
                    res[p++] = nums2[p2++];
                }
            }
    				// 处理长数组多出的那部分
            if (p1 == m) {
                while (p2 != n) {
                    res[p++] = nums2[p2++];
                }
            }
            if (p2 == n) {
                while (p1 != m) {
                    res[p++] = nums1[p1++];
                }
            }
    				// 重新赋值回 nums1
            for(int i = 0; i < res.length; i++) {
                nums1[i] = res[i];
            }
        }
    }
    ```

> ℹ️ 力扣（LeetCode）官网 - 全球极客挚爱的技术成长平台
> 备战技术面试？力扣提供海量技术面试资源，帮助你高效提升编程技能，轻松拿下世界 IT 名企 Dream Offer。
> [https://leetcode.cn/problems/merge-intervals/?envType=study-plan-v2&envId=top-100-liked](https://leetcode.cn/problems/merge-intervals/?envType=study-plan-v2&envId=top-100-liked)
1. 问题描述：

    ```Plain
    以数组 intervals 表示若干个区间的集合，其中单个区间为 intervals[i] = [starti, endi] 。请你合并所有重叠的区间，并返回 一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间 。
    ```
2. 示例：

    ```Plain
    示例 1：
    输入：intervals = [[1,3],[2,6],[8,10],[15,18]]
    输出：[[1,6],[8,10],[15,18]]
    解释：区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6].
    ```
3. 代码：

    ![[IMG-20260620205448739.png|459]]

    ```Java
     public int[][] merge(int[][] intervals) {
        List<int[]> list = new ArrayList<>();
    		// 排序
        Arrays.sort(intervals, (o1, o2) ->{
            if (o1[0] == o2[0]) {
                return o1[1] - o2[1];
            } else {
                return o1[0] - o2[0];
            }
        });
        int len = intervals.length;
        list.add(intervals[0]);
        for (int i = 0; i < len; i++) {
    				// list里的最后一个的右边数字
            int right = list.get(list.size() - 1)[1];
    				// 元素组当前遍历的里左边数字
            int left = intervals[i][0];
    				// 比较
            if (right >= left) {
    						// 替换右边数字
                list.get(list.size() - 1)[1] = Math.max(intervals[i][1], right); 
            } else {
    						// 往list里新增
                list.add(new int[]{left, intervals[i][1]}); 
            }
        }
        return list.toArray(new int[list.size()][]);
    }
    ```
# 三、📑BM97 旋转数组🐮🔥✔️

1. 问题描述：

    ```Plain
    给定一个整数数组 nums，将数组中的元素向右轮转 k 个位置，其中 k 是非负数。
    ```
2. 示例：

    ```Plain
    示例1
    输入：6,2,[1,2,3,4,5,6]
    返回值：[5,6,1,2,3,4]
    示例2
    输入：4,0,[1,2,3,4]
    返回值：[1,2,3,4]
    ```
3. 代码：

    ![[IMG-20260620205448776.png|582]]

    ```Java
     public int[] solve (int n, int m, int[] a) {
        int offset = m % n;
        // 反转三次
        reverse(a, 0, n - 1);
        reverse(a, 0, offset - 1);
        reverse(a, offset, n - 1);
        return a;
    }
    public void reverse(int[] nums, int left, int right) {
        while (left < right) {
            swap(nums, left, right);
            left++;
            right--;
        }
    }
    public void swap(int[] nums, int a, int b) {
        int tmp = nums[a];
        nums[a] = nums[b];
        nums[b] = tmp;
    }
    ```

1. 问题描述：

    ```Plain
    给定一个包含 n + 1 个整数的数组 nums ，其数字都在 [1, n] 范围内（包括 1 和 n），可知至少存在一个重复的整数。
    假设 nums 只有 一个重复的整数 ，返回 这个重复的数 。
    你设计的解决方案必须 不修改 数组 nums 且只用常量级 O(1) 的额外空间。 
    ```
2. 示例：

    ```Plain
    示例 1：
    输入：nums = [1,3,4,2,2]
    输出：2
    示例 2：
    输入：nums = [3,1,3,4,2]
    输出：3
    ```
3. 代码：
    - 因为其数字都在 `[1, n]` 范围内，可以使用 **桶** 来统计

    ```Java
    public int findDuplicate(int[] nums) {
        int[] count = new int[nums.length];
        for(int i = 0;i < nums.length; i++) {
            count[nums[i]]++; 
        }
        for(int i = 0;i < count.length; i++) {
            if (count[i] > 1) {
                return i;
            } 
        }
        return -1;
    }
    ```
# 五、📑面试题61. 扑克牌中的顺子⚔️
1. 问题描述：

    ```Plain
    从若干副扑克牌中随机抽 5 张牌，判断是不是一个顺子，即这5张牌是不是连续的。2～10为数字本身，A为1，J为11，Q为12，K为13，而大、小王为 0 ，可以看成任意数字。A 不能视为 14。
    ```
2. 示例：

    ```Plain
    示例 1:
    输入: [1,2,3,4,5]
    输出: True
    示例 2:
    输入: [0,0,1,2,5]
    输出: True
    ```
3. 代码：

    ![[IMG-20260620205448814.png|581]]

    ```Java
     class Solution {
        public boolean isStraight(int[] nums) {
            Set<Integer> repeat = new HashSet<>();
            int max = 0, min = 14;
            for(int num : nums) {
                if(num == 0) continue; // 跳过大小王
                max = Math.max(max, num); // 最大牌
                min = Math.min(min, num); // 最小牌
                if(repeat.contains(num)) return false; // 若有重复，提前返回 false
                repeat.add(num); // 添加此牌至 Set
            }
            return max - min < 5; // 最大牌 - 最小牌 < 5 则可构成顺子
        }
    }
    ```
