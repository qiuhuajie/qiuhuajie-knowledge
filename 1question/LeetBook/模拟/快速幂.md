# 一、📑剑指 Offer 16. 数值的整数次方
1. 问题描述：

    ```Plain
     实现 pow(x, n) ，即计算 x 的 n 次幂函数（即，xn）。不得使用库函数，同时不需要考虑大数问题。
    ```
2. 示例：

    ```Plain
    示例 1：
    输入：x = 2.00000, n = 10
    输出：1024.00000
    ```
3. 代码：

    ![[IMG-20260620205639241.png|554]]

    ![[IMG-20260620205639316.png|487]]

    ```Java
    public double myPow(double x, int n) {
        double res = 1.0;
        long m =  n;
        if (m < 0) {
            m = -m;
            x = 1 / x;
        }
        while (m > 0) {
            if (m % 2 == 1) {
                res *= x;
            }
            x *= x;
            m >>= 1;
        }
        return res;
    }
    ```
