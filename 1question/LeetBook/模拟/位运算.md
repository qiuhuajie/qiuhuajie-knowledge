# 一、位运算知识补充
1. 介绍

| 操作符 | 描述 |
| --- | --- |
| `＆` 与 | **如果相对应位都是1，则结果为1，否则为0**。`true & true = true` |
| `\|` 或 | **如果相对应位都是 0，则结果为 0，否则为 1**。`false ｜ false = false` |
| `^` 异或 | **如果相对应位值相同，则结果为0，否则为1** |
| `〜` 反 | 按位取反运算符翻转操作数的每一位，即0变成1，1变成0。 |
| `<<` | **按位左移**运算符。左操作数按位左移右操作数指定的位数。 |
| `>>` | **按位右移**运算符。左操作数按位右移右操作数指定的位数。 |
| `>>>` | **按位右移**==**补零**==操作符。左操作数的值按右操作数指定的位数右移，移动得到的空位以零填充。 |

2. 示例

    ```Plain
    A = 0011 1100
    B = 0000 1101
    -----------------
    A&B = 0000 1100
    A | B = 0011 1101
    A ^ B = 0011 0001
    ~A = 1100 0011
    ```
3. 常用技巧
    1. ==异或运算有以下三个性质==
        1. 任何数和 0 做异或运算，结果仍然是原来的数，即 a⊕0 = a
        2. 任何数和其自身做异或运算，结果是 0，即 a⊕a= 0
        3. 异或运算满足交换律和结合律，即 a⊕b⊕a = b⊕a⊕a = b⊕(a⊕a) = b⊕0 = b
    2. ==判断一个数 num 的某一位是不是 1==
        1. 判断最低位是不是 1，将 `num & 1` 看结果是不是 1
        2. 如果还要判断其他位是不是 1，可以让 `1 << 1` ，也可以让 `num >>> 1` ，再做 `&` 操作

        ```Java
        // 1 << 1
        int div = 1;
        for(int j = 0; j < 32; j++) {
            if ((num & div) != 0) {
                counts[j]++;
            }
            div <<= 1;
        }
        // sum >>> 1
        int div = 1;
        for(int j = 0; j < 32; j++) {
            counts[j] += num & 1;
            num >>>= 1;
        }
        ```
        ```Java
        // 找到 tmp 自左向右第一个为 1 的位
        int div = 1;
        while ((tmp & div) == 0) {
            div <<= 1;
        }
        ```
# 二、📑371. 两整数之和
> ℹ️ 力扣
> [https://leetcode.cn/problems/sum-of-two-integers/](https://leetcode.cn/problems/sum-of-two-integers/)
1. 问题描述：

    ```Plain
    给你两个整数 a 和 b ，不使用 运算符 + 和 - ，计算并返回两整数之和 
    1 + 2 = 3
    ```
2. 代码：
    1. **==异或运算==** **`^`** **==相当于不进位的加法==**，比如：`1101 ^ 1011 = 0110`

        ![[IMG-20260620205757287.png|232]]

    2. 而**进位和** `&` **运算一样**，我们**只需要**==**把**== **`a & b`** ==**的结果左移一位即可得到进位**==
    3. 因此，**==不进位的加法 + 进位 就等于两个数相加==**：`a + b == ((a & b) << 1) + (a ^ b)`

        ![[IMG-20260620205757333.png|688]]

    4. 但是我们发现计算 `(a ^ b) + ((a & b) << 1)` 时，又用到了加法，但其实本质上还是两个数相加
    5. 因此我们递归或者循环计算，直到进位 `(a & b) << 1` 为 0 为止。

    ```Java
     public int getSum(int a, int b) {
        // 进位和 & 运算一样，我们只需要把 a & b 的结果左移一位即可得到进位
        int c = (a & b) << 1;
        // 异或运算相当于不进位的加法
        int d = a ^ b;
        // 直到进位 (a & b) << 1 为 0 为止
        if (c == 0) {
            return d;
        }
        return getSum(c, d); 
    }
    ```
# 三、📑剑指 Offer 15. 二进制中1的个数⚔️✔️
> ℹ️ 力扣
> [https://leetcode.cn/problems/er-jin-zhi-zhong-1de-ge-shu-lcof/description/](https://leetcode.cn/problems/er-jin-zhi-zhong-1de-ge-shu-lcof/description/)
1. 问题描述：

    ```Plain
     编写一个函数，输入是一个无符号整数（以二进制串的形式），返回其二进制表达式中数字位数为 '1' 的个数（也被称为 汉明重量).）
    ```
2. 示例：

    ```Plain
    示例 1：
    输入：n = 11 (控制台输入 00000000000000000000000000001011)
    输出：3
    解释：输入的二进制串 00000000000000000000000000001011 中，共有三位为 '1'。
    ```
3. 代码：

    ![[IMG-20260620205757385.png|582]]

    ```Java
     public int hammingWeight(int n) {
        int res = 0;
        while (n != 0) {
            if ((n & 1) == 1) {
                res++;
            }
            n >>>= 1;
        }
        return res;
    }
    ```
# 四、📑剑指 Offer 56 - I. 数组中数字出现的次数⚔️✔️
> ℹ️ 力扣
> [https://leetcode.cn/problems/shu-zu-zhong-shu-zi-chu-xian-de-ci-shu-lcof/](https://leetcode.cn/problems/shu-zu-zhong-shu-zi-chu-xian-de-ci-shu-lcof/)
1. 问题描述：

    ```Plain
    一个整型数组 nums 里除两个数字之外，其他数字都出现了两次。请写程序找出这两个只出现一次的数字。要求时间复杂度是O(n)，空间复杂度是O(1)。
    ```
2. 示例：

    ```Plain
    示例 1：
    输入：nums = [4,1,4,6]
    输出：[1,6] 或 [6,1]
    示例 2：
    输入：nums = [1,2,10,4,1,4,3,3]
    输出：[2,10] 或 [10,2]
    ```
3. 代码：
    > ℹ️ 力扣
    > [https://leetcode.cn/problems/shu-zu-zhong-shu-zi-chu-xian-de-ci-shu-lcof/solutions/572857/jian-zhi-offer-56-i-shu-zu-zhong-shu-zi-tykom/](https://leetcode.cn/problems/shu-zu-zhong-shu-zi-chu-xian-de-ci-shu-lcof/solutions/572857/jian-zhi-offer-56-i-shu-zu-zhong-shu-zi-tykom/)

    ![[IMG-20260620205757459.png|608]]

    ```Java
     public int[] singleNumbers(int[] nums) {
        int n = nums.length;
    		// 遍历数组异或，得到的 tmp = x ^ y
        int tmp = 0;
        for (int i = 0; i < n; i++) {
            tmp ^= nums[i];
        }
    		// 找到 x ^ y 自左向右第一个为 1 的位，最终找到的这一位上 x y 一定不同
        int div = 1;
        while ((tmp & div) == 0) {
            div <<= 1;
        }
    		// 用找到的这位将数组分为两个字数组，再分别异或就可得到 x y
        int x = 0, y = 0;
        for (int i = 0; i < n; i++) {
            if ((nums[i] & div) == 0) {
                x ^= nums[i];
            } else {
                y ^= nums[i];
            }
        }
        return new int[]{x, y};
    }
    ```
# 五、📑剑指 Offer 56 - II. 数组中数字出现的次数 II⚔️✔️
> ℹ️ 力扣
> [https://leetcode.cn/problems/shu-zu-zhong-shu-zi-chu-xian-de-ci-shu-ii-lcof/submissions/](https://leetcode.cn/problems/shu-zu-zhong-shu-zi-chu-xian-de-ci-shu-ii-lcof/submissions/)
1. 问题描述：

    ```Plain
     在一个数组 nums 中除一个数字只出现一次之外，其他数字都出现了三次。请找出那个只出现一次的数字。
    ```
2. 示例：

    ```Plain
    示例 1：
    输入：nums = [3,4,3,3]
    输出：4
    ```
3. 代码：

    ![[IMG-20260620205757513.png|614]]

    ```Java
     public int singleNumber(int[] nums) {
        // 建立一个长度为 32 的数组 counts ，通过以上方法可记录所有数字的各二进制位的 1 的出现次数
        int[] counts = new int[32];
        for(int num : nums) {
            for(int j = 0; j < 32; j++) {
                counts[j] += num & 1;
                num >>>= 1;
            }
        }
        // 将 counts 各元素对 3 求余，则结果为 “只出现一次的数字” 的各二进制位
        for(int i = 0; i < 32; i++) {
            counts[i] %= 3;
        }
        // 将二进制转为十进制的结果
        int res = 0;
        int power = 0;
        for (int i = 0; i < 32; i++) {
            res += counts[i] * Math.pow(2, power);
            power++;
        }
        return res;
    }
    ```
# 六、📑136. 只出现一次的数✔️
1. 问题描述：

    ```Plain
    给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素
    ```
2. 示例：

    ```Plain
    输入: [2,2,1]
    输出: 1
    ```
3. 代码：

    ```Java
    //异或：本题数组的特点，假设有 2m+1 个元素，这些元素 有 m 对相同
    //满足上面特点的元素集合，且目标是找出 唯一不同的那个，则可以使用异或
    //异或：
    //异或运算有以下三个性质。
    // 1.任何数和 00 做异或运算，结果仍然是原来的数，即 a⊕0 = a。
    // 2.任何数和其自身做异或运算，结果是 0，即 a⊕a= 0。
    // 3.异或运算满足交换律和结合律，即 a⊕b⊕a = b⊕a⊕a = b⊕(a⊕a) = b⊕0 = b
    public int singleNumber(int[] nums) {
        int single = 0;
        for (int num : nums) {
            single ^= num;
        }
        return single;
    }
    ```
