## **📑155. 最小栈**🔥
1. 问题描述：
    
    ```Plain
    设计一个支持 push ，pop ，top 操作，并能在常数时间内检索到最小元素的栈。
    
    实现 MinStack 类:
    
    MinStack() 初始化堆栈对象。
    void push(int val) 将元素val推入堆栈。
    void pop() 删除堆栈顶部的元素。
    int top() 获取堆栈顶部的元素。
    int getMin() 获取堆栈中的最小元素。 
    ```
    
1. 思路：
    
    1. 两个栈，一个保存元素，一个保存当前栈中的最小元素
        
        ![[IMG-20260405035401923.png|Untitled 32.png]]
        
    
    1. 也可以使用一个栈，存一个二元组（当前值，栈内最小值）
    
1. 代码：
    
    ```Java
     public class MinStack {
        Stack<Integer> eStack;
        Stack<Integer> minStack;
        
        public MinStack() {
            eStack = new Stack<>();
            minStack = new Stack<>();
        }
    
        public void push(int val) {
            eStack.push(val);
            if (!minStack.isEmpty()) {
                minStack.push(Math.min(val, getMin()));
            } else {
                minStack.push(val);
            }
        }
    
        public void pop() {
            eStack.pop();
            minStack.pop();
        }
    
        public int top() {
            return eStack.peek();
        }
    
        public int getMin() {
            return minStack.peek();
        }
    }
    ```
    
## **📑20. 有效的括号**🔥
1. 问题描述：
    
    ```Plain
    给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s ，判断字符串是否有效。
    
    有效字符串需满足：
    
    左括号必须用相同类型的右括号闭合。
    左括号必须以正确的顺序闭合。
    每个右括号都有一个对应的相同类型的左括号。
    ```
    
1. 示例：
    
    ```Plain
    示例 1：
    输入：s = "()"
    输出：true
    
    示例 2：
    输入：s = "()[]{}"
    输出：true
    ```
    
1. 代码：
    
    ```Java
     public boolean isValid(String s) {
        char[] chars = s.toCharArray();
        Stack<Character> stack = new Stack<>();
    
        for (char c : chars) {
            if (c == '(' || c == '[' || c == '{') {
                stack.push(c);
            }
    
            if (c == ')') {
                if (stack.isEmpty()) return false;
                if (stack.peek() == '(') {
                    stack.pop();
                } else {
                    return false;
                }
            }
    
            if (c == ']') {
                if (stack.isEmpty()) return false;
                if (stack.peek() == '[') {
                    stack.pop();
                } else {
                    return false;
                }
            }
    
            if (c == '}') {
                if (stack.isEmpty()) return false;
                if (stack.peek() == '{') {
                    stack.pop();
                } else {
                    return false;
                }
            }
        }
    
        if (stack.isEmpty()) {
            return true;
        } else {
            return false;
        }
    }
    ```
    
## **📑剑指 Offer 31. 栈的压入、弹出序列**⚔️
1. 问题描述：
    
    ```Plain
    输入两个整数序列，第一个序列表示栈的压入顺序，请判断第二个序列是否为该栈的弹出顺序。假设压入栈的所有数字均不相等。例如，序列 {1,2,3,4,5} 是某栈的压栈序列，序列 {4,5,3,2,1} 是该压栈序列对应的一个弹出序列，但 {4,3,5,1,2} 就不可能是该压栈序列的弹出序列。
    ```
    
1. 示例：
    
    ```Plain
    示例 1：
    输入：pushed = [1,2,3,4,5], popped = [4,5,3,2,1]
    输出：true
    解释：我们可以按以下顺序执行：
    push(1), push(2), push(3), push(4), pop() -> 4,
    push(5), pop() -> 5, pop() -> 3, pop() -> 2, pop() -> 1
    ```
    
1. 代码：
    
    ![[IMG-20260405035402277.png|Untitled 1 13.png]]
    
    ```Java
     public boolean validateStackSequences(int[] pushed, int[] popped) {
        Stack<Integer> popStack = new Stack<>();
        Stack<Integer> pushStack = new Stack<>();
    
        // 先把 pop 倒序入栈 1
        for (int i = popped.length - 1; i >= 0 ; i--) {
            popStack.push(popped[i]);
        }
    
        // 遍历 pushed 数组
        for (int cur : pushed) {
            // 先把当前元素压进栈 2
            pushStack.push(cur);
            // 循环判断 栈 2 和 栈 1 栈顶元素是否一样，如果一样两个都弹出
            while (!popStack.isEmpty() && !pushStack.isEmpty() && pushStack.peek().equals(popStack.peek())) {
                popStack.pop();
                pushStack.pop();
            }
    
        }
        // 最后 栈 1 为空则匹配成功
        return popStack.isEmpty();
    }
    ```

    class CQueue {
        LinkedList<Integer> A, B;
        public CQueue() {
            A = new LinkedList<Integer>();
            B = new LinkedList<Integer>();
        }
        public void appendTail(int value) {
            A.addLast(value);
        }
        public int deleteHead() {
            if(!B.isEmpty()) return B.removeLast();
            if(A.isEmpty()) return -1;
            while(!A.isEmpty())
                B.addLast(A.removeLast());
            return B.removeLast();
        }
    }
    
    class CQueue {
    
        Stack<Integer> stack1 = null;
        Stack<Integer> stack2 = null;
    
        public CQueue() {
            stack1 = new Stack<>();
            stack2 = new Stack<>();
        }
        
        public void appendTail(int value) {
            stack1.push(value);
        }
        
        public int deleteHead() {
            if (stack2.isEmpty()) {
                while (!stack1.isEmpty()) {
                   stack2.push(stack1.pop());
                }
            }
            if (!stack2.isEmpty()) {
                return stack2.pop();
            } else {
                return -1;
            }
            
        }
    }

## **📑394. 字符串解码**🔥
1. 问题描述：
    
    ```Plain
    给定一个经过编码的字符串，返回它解码后的字符串。
    
    编码规则为: k[encoded_string]，表示其中方括号内部的 encoded_string 正好重复 k 次。注意 k 保证为正整数。
    
    你可以认为输入字符串总是有效的；输入字符串中没有额外的空格，且输入的方括号总是符合格式要求的。
    
    此外，你可以认为原始数据不包含数字，所有的数字只表示重复的次数 k ，例如不会出现像 3a 或 2[4] 的输入。
    ```
    
1. 示例：
    
    ```Plain
    示例 1：
    输入：s = "3[a]2[bc]"
    输出："aaabcbc"
    
    示例 2：
    输入：s = "3[a2[c]]"
    输出："accaccacc"
    
    示例 3：
    输入：s = "2[abc]3[cd]ef"
    输出："abcabccdcdcdef"
    ```
    
1. 代码：
    
    - 使用两个栈
    
    - 题解地址
        
        > [!info] 力扣  
        >  
        > [https://leetcode.cn/problems/decode-string/solutions/19447/decode-string-fu-zhu-zhan-fa-di-gui-fa-by-jyd/](https://leetcode.cn/problems/decode-string/solutions/19447/decode-string-fu-zhu-zhan-fa-di-gui-fa-by-jyd/)  
        
    
    ```Java
     public static String decodeString(String s) {
    
            char[] chars = s.toCharArray();
            int len = chars.length;
    
            // 一个栈存重复的次数
            Stack<Integer> numStack = new Stack<>();
            // 一个栈存之前的临时结果
            Stack<String> sbStack = new Stack<>();
    
            StringBuilder res = new StringBuilder();
            int num = 0;
    
            for (int i = 0; i < len; i++) {
    
                if (chars[i] > '0' && chars[i] <= '9') {
                    // 当遍历到数字， 使用 num 存下来（这里的数字可能是有多位，即多个 char 组成）
                    num = num * 10 + chars[i] - '0';
                } else if (chars[i] == '['){
                    // 遍历到 [，要将当前的 res 和 num 分别入到两个栈，num 重置为 0，res 重置为空串
                    numStack.push(num);
                    sbStack.push(res.toString());
                    num = 0;
                    res = new StringBuilder();
                } else if (chars[i] == ']'){
                    // 遍历到 ]，从栈中取出之前的 res 和 num
                    Integer preN = numStack.pop();
                    String preRes = sbStack.pop();
    
                    // 将当前的 res 进行重复
                    StringBuilder tmp = new StringBuilder();
                    for (int j = 0; j < preN; j++) {
                        tmp.append(res);
                    }
                    // 再与从栈中取出的之前的 res 拼接
                    res = new StringBuilder(preRes + tmp);
                } else {
                    // 如果是字符直接拼接在当前 res 后面
                    res.append(chars[i]);
                }
            }
    
            return res.toString();
        }
    ```
    
## **📑中缀表达式求值**
1. 问题描述：
    
1. 示例：
    
    ```Plain
    3 + (4 * 5 - 6) * 2 - 7  =  24 
    ```
    
1. 代码：
    
    1. 创建一个运算符栈和一个操作数栈
    
    1. 从左到右遍历中缀表达式的每个字符，如果遇到操作数，则将其压入操作数栈中
    
    1. 如果遇到运算符，则判断当前运算符与运算符栈栈顶运算符的优先级
        
        1. 如果当前运算符的优先级大于栈顶运算符的优先级，则将当前运算符压入运算符栈中
        
        1. 否则，==从====**运算符栈和操作数栈中弹出一个运算符和两个操作数**====，进行运算，并将结果压入操作数栈中，直到当前运算符的优先级大于栈顶运算符的优先级为止==
        
    
    1. 遍历完中缀表达式后，从运算符栈和操作数栈中依次弹出运算符和操作数，进行运算，最终得到的结果就是中缀表达式的值
    
    ```Java
    public class InfixExpressionEvaluator {
        public static int evaluate(String expression) {
            Stack<Integer> operandStack = new Stack<>();
            Stack<Character> operatorStack = new Stack<>();
    
            for (int i = 0; i < expression.length(); i++) {
                char c = expression.charAt(i);
    
                if (Character.isDigit(c)) {
                    int operand = 0;
                    while (i < expression.length() && Character.isDigit(expression.charAt(i))) {
                        operand = operand * 10 + (expression.charAt(i) - '0');
                        i++;
                    }
                    i--;
                    operandStack.push(operand);
                } else if (c == '(') {
                    operatorStack.push(c);
                } else if (c == ')') {
                    while (!operatorStack.isEmpty() && operatorStack.peek() != '(') {
                        evaluateOperator(operandStack, operatorStack);
                    }
                    operatorStack.pop();
                } else if (isOperator(c)) {
                    while (!operatorStack.isEmpty() && operatorPrecedence(operatorStack.peek()) >= operatorPrecedence(c)) {
                        evaluateOperator(operandStack, operatorStack);
                    }
                    operatorStack.push(c);
                }
            }
    
            while (!operatorStack.isEmpty()) {
                evaluateOperator(operandStack, operatorStack);
            }
    
            return operandStack.pop();
        }
    
        private static boolean isOperator(char c) {
            return c == '+' || c == '-' || c == '*' || c == '/';
        }
    
    		// 判断操作符优先级
        private static int operatorPrecedence(char operator) {
            if (operator == '+' || operator == '-') {
                return 1;
            } else if (operator == '*' || operator == '/') {
                return 2;
            }
            return 0;
        }
    		
    		// 操作数栈弹两个，操作符栈弹一个，运算
        private static void evaluateOperator(Stack<Integer> operandStack, Stack<Character> operatorStack) {
            char operator = operatorStack.pop();
            int operand2 = operandStack.pop();
            int operand1 = operandStack.pop();
    
            switch (operator) {
                case '+':
                    operandStack.push(operand1 + operand2);
                    break;
                case '-':
                    operandStack.push(operand1 - operand2);
                    break;
                case '*':
                    operandStack.push(operand1 * operand2);
                    break;
                case '/':
                    operandStack.push(operand1 / operand2);
                    break;
            }
        }
    
        public static void main(String[] args) {
            String expression = "3 + (4 * 5 - 6) * 2 - 7";
            int result = evaluate(expression);
            System.out.println("Result: " + result);
        }
    }
    ```
