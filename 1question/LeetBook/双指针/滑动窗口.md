# 1. 找到符合某个条件的子区间
- **求一个** ==**满足某个条件（无重复/和大于target）**==**的** ==**最大/最小/最长/最短**== **的** ==**子串/子数组**==
- **两个字符串，一长一短，找**==**长串中能完全覆盖短串的子串**==
# 2. 滑动窗口使用的思路
## 2.1 ==**寻找最长**==
1. 思路描述
    
    1. 左右双指针（L，R）在起始点，R 向右逐位滑动循环
    
    1. 每次滑动的过程中
        
        1. 如果窗内元素满足条件，R 向右扩大窗口，并更新最优结果
        
        1. 如果窗内元素不满足条件，L 向右缩小窗口
        
    
    1. R 到达结尾
    
1. **模板：**
    
    ```Java
    public static void forMax() {
        初始化 left，right，result，bestResult;
    		初始化 用来描述窗口的数据结构;
        while (右指针没有到结尾) {
            窗口扩大，加入 right 对应元素;
    				right++;
            更新当前 result;
            while (result 不满足要求) {
                窗口缩小，移除 left 对应元素;
                left++;
            }
            更新最优结果 bestResult;        // 始终在在满足要求时更新
        }
        return bestResult;
    }
    ```
    
## 2.2 ==**寻找最短**==
1. 思路描述
    
    1. 左右双指针（L，R）在起始点，R 向右逐位滑动循环
    
    1. 每次滑动的过程中
        
        1. 如果窗内元素满足条件，L 向右缩小窗口，并更新最优结果
        
        1. 如果窗内元素不满足条件，R 向右扩大窗口
        
    
    1. R 到达结尾
    
1. **模板：**
    
    ```Java
    public static void forMin() {
        初始化 left，right，result，bestResult;
    		初始化 用来描述窗口的数据结构;
        while (右指针没有到结尾) {
            窗口扩大，加入 right 对应元素;
    				right++;
            更新当前 result;
            while (result 满足要求) {
    						更新最优结果 bestResult;       // 始终在在满足要求时更新
                窗口缩小，移除 left 对应元素;
                left++;
            }
        }
        return bestResult;
    }
    ```
    
# 2.例题
## **📑209. 长度最小的子数组**

> [!info] 力扣  
>  
> [https://leetcode.cn/problems/minimum-size-subarray-sum/description/](https://leetcode.cn/problems/minimum-size-subarray-sum/description/)  
1. 问题描述：
    
    ```Plain
     给定一个含有 n 个正整数的数组和一个正整数 target 。
     
     找出该数组中满足其和 ≥ target 的长度最小的 连续子数组 [numsl, numsl+1, ..., numsr-1, numsr] ，并返回其长度。如果不存在符合条件的子数组，返回 0 。
    ```
    
1. 示例：
    
    ```Plain
     示例 1：
     输入：target = 7, nums = [2,3,1,2,4,3]
     输出：2
     解释：子数组 [4,3] 是该条件下的长度最小的子数组。
     
     示例 2：
     输入：target = 4, nums = [1,4,4]
     输出：1
     
     示例 3：
     输入：target = 11, nums = [1,1,1,1,1,1,1,1]
     输出：0
    ```
    
1. 代码：
    
    - **滑动窗口**
    
    ```Java
    public int minSubArrayLen(int target, int[] nums) {
        // 1. 初始化
        int result = Integer.MAX_VALUE;
        int left = 0, right = 0;
    
    		// 初始化 用 sum 来描述窗口
        int sum = 0;
    
        while (right < nums.length){
            // 2. 窗口扩大，加入 right 对应元素，更新当前 result
            sum += nums[right];
            right++;
            
            // 3. 寻找最短问题：窗内元素满足条件时，控制 left 缩小窗口
            while (sum >= target){
                // 4. 更新最优结果
                result = Math.min(result, right - left);
                // 5. 窗口缩小，移除 left 对应元素，更新当前 result
                sum -= nums[left];
                left++;
            }
        }
        return result==Integer.MAX_VALUE ? 0 : result;
    }
    ```
    
## **📑3. 无重复字符的最长子串**🔥⚔️✔️

> [!info] 力扣  
>  
> [https://leetcode.cn/problems/longest-substring-without-repeating-characters/](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)  
1. 问题描述：
    
    ```Plain
     给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串 的长度。
    ```
    
1. 示例：
    
    ```Plain
    示例 1:
    输入: s = "abcabcbb"
    输出: 3 
    解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
    
    示例 2:
    输入: s = "bbbbb"
    输出: 1
    解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
    
    示例 3:
    输入: s = "pwwkew"
    输出: 3
    解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
         请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
     
    提示：
    0 <= s.length <= 5 * 104
    s 由英文字母、数字、符号和空格组成
    ```
    
1. 代码：
    
    - 窗口用 map 描述，可以统计每个字符出现的次数
    
    ![[IMG-20260405035400221.png|Untitled 92.png]]
    
    ```Java
     public int lengthOfLongestSubstring(String s) {
        // 1. 初始化
        int left = 0, right = 0;
        int res = 0;
        
        // 初始化描述窗口的数据结构
        HashMap<Character, Integer> window = new HashMap<>();
        
        while (right < s.length()){
            // 2. 窗口扩大，加入 right 对应元素
            char c = s.charAt(right);
            window.put(c, window.getOrDefault(c, 0) + 1);
            right++;
            
            // 3. 寻找最长问题时：窗内元素不满足条件时，控制 left 缩小窗口
            while (window.get(c) > 1){
                // 4. 窗口缩小，移除 left 对应元素
                char d = s.charAt(left);
                window.put(d, window.getOrDefault(d, 0) - 1);
    						left++;
            }
    				
            //5. 更新最优结果，注意此时 left 指向窗口第一个元素，right 指向窗口的最右元素的后一个（上面提前执行了 right++）
            res = Math.max(res, right - left);
        }
        return res;
    }
    ```
    
## **📑567. 字符串的排列**

> [!info] 力扣  
>  
> [https://leetcode.cn/problems/permutation-in-string/](https://leetcode.cn/problems/permutation-in-string/)  
1. 问题描述：
    
    ```Plain
    给你两个字符串 s1 和 s2 ，写一个函数来判断 s2 是否包含 s1 的排列。如果是，返回 true ；否则，返回 false 。
    
    换句话说，s1 的排列之一是 s2 的 子串 。
    ```
    
1. 示例：
    
    ```Plain
    示例 1：
    输入：s1 = "ab" s2 = "eidbaooo"
    输出：true
    解释：s2 包含 s1 的排列之一 ("ba").
    
    示例 2：
    输入：s1= "ab" s2 = "eidboaoo"
    输出：false
    ```
    
1. 代码：
    
    - **相当给一个 s 和一个 T，请问你 s 中是否存在一个子串，包含 T 中所有字符且不包含其他字符**
    
    - ==**由于题目中的条件是 两个字符串之间比较，则还需要使用一个 need（map） 来保存被覆盖字符串中的所有字符**==
    
    ```Java
     public List<Integer> findAnagrams(String s, String p) {
        int len1 = s.length();
        int len2 = p.length();
    
        int valid = 0;
        int left = 0, right = 0;
    
        List<Integer> res = new ArrayList<>();
    
        HashMap<Character, Integer> window = new HashMap<>();
    
        HashMap<Character, Integer> need = new HashMap<>();
        for (int i = 0; i < len2; i++) {
            need.put(p.charAt(i), need.getOrDefault(p.charAt(i), 0) + 1);
        }
    
        while (right < len1) {
            char c = s.charAt(right);
            right++;
            if (need.containsKey(c)) {
                window.put(c, window.getOrDefault(c, 0) + 1);
                if (window.get(c).equals(need.get(c))) {
                    valid++;
                }
            }
    
            while (right - left >= len2) {
    						
                if (valid == need.size()) {
    								// 满足条件时，记录结果
                    res.add(left);
                }
    
                char d = s.charAt(left);
                left++;
                if (need.containsKey(d)) {
                    if (window.get(d).equals(need.get(d))) {
                        valid--;
                    }
                    window.put(d, window.getOrDefault(d, 0) - 1);
                }
            }
        }
        return res;
    }
    ```


    ```Java
     public List<Integer> findAnagrams(String s, String p) {

        int len2 = p.length();
    
        int valid = 0;
        int left = 0, right = 0;
    
        List<Integer> res = new ArrayList<>();
    
        HashMap<Character, Integer> window = new HashMap<>();
    
        HashMap<Character, Integer> need = new HashMap<>();
        for (int i = 0; i < len2; i++) {
            need.put(p.charAt(i), need.getOrDefault(p.charAt(i), 0) + 1);
        }
    
        while (right < len1) {
            char c = s.charAt(right);
            right++;
            if (need.containsKey(c)) {
                window.put(c, window.getOrDefault(c, 0) + 1);

                }

    				// 找最小，满足条件时，左侧缩小

    						// 记录结果
                if (valid == need.size()) {
                    res.add(left);
                }
    
                char d = s.charAt(left);
                left++;
                if (need.containsKey(d)) {
                    if (window.get(d).equals(need.get(d))) {
    ```

## **📑76. 最小覆盖子串**🔥😡

> [!info] 力扣  
>  
> [https://leetcode.cn/problems/minimum-window-substring/](https://leetcode.cn/problems/minimum-window-substring/)  
1. 问题描述：
    
    ```Plain
    给你一个字符串 s 、一个字符串 t 。返回 s 中涵盖 t 所有字符的最小子串。如果 s 中不存在涵盖 t 所有字符的子串，则返回空字符串 "" 。
    
    注意：
    
    对于 t 中重复字符，我们寻找的子字符串中该字符数量必须不少于 t 中该字符数量。
    如果 s 中存在这样的子串，我们保证它是唯一的答案。
    ```
    
1. 示例：
    
    ```Plain
    示例 1：
    
    输入：s = "ADOBECODEBANC", t = "ABC"
    输出："BANC"
    解释：最小覆盖子串 "BANC" 包含来自字符串 t 的 'A'、'B' 和 'C'。
    示例 2：
    
    输入：s = "a", t = "a"
    输出："a"
    解释：整个字符串 s 是最小覆盖子串。
    示例 3:
    
    输入: s = "a", t = "aa"
    输出: ""
    解释: t 中两个字符 'a' 均应包含在 s 的子串中，
    因此没有符合条件的子字符串，返回空字符串。
    ```
    
1. 思路：
    
    1. ==**由于题目中的条件是 两个字符串之间的覆盖，则还需要使用一个 need（map） 来保存被覆盖字符串中的所有字符**==
    
    1. 过程：
        
        1. 我们在字符串 s 中使用双指针中的左右指针技巧，初始化 left =right = 0 ，把索引**左闭右开区间** `**[left，right)**` 称为一个[窗]
        
        1. 我们**先不断地增加 right 指针扩大窗口** `**[left，right)**` **，直到窗口中的字符串符合要求（包含了 t 中的所有字符）**
        
        1. 此时，我们**停止增加 right ，转而不断增加 left 指针缩小窗口** `**[left，right)**`**，直到窗口中的字符串不再符合要求 (不包合 t 中的所有字符了)** 。同时，每次增加 left ，我们都要更新一轮结果
        
        1. 重复第 2 和第 3步，直到 right 到达字符串 S 的尽头
        
    
    1. 这个思路其实也不难，==第 2 步相当于在寻找一个 [可行解]，然后第 3 步在优化这个[可行解]，最终找到最优解==，也就是最短的覆盖子串
    
    1. 左右指针轮流前进，窗口大小增增减减，窗口不断向右滑动，这就是[滑动窗口]这个名字的来历
    
1. 代码：
    
    - 其中 **`valid`** 变量表示**窗口中满足 覆盖 need 条件的字符个数**，如果 valid 和 need.size 的大小相同，则说明窗口已满足条件，已经完全覆盖了串 T
    
    ```Java
     public static String minWindow(String s, String t) {
    
        // 1. 初始化
        int left = 0;
        int right = 0;
        int valid = 0;  // 使用 valid 来判断覆盖条件是否满足
    
        // 保存最小覆盖子串起始位置和长度，用于最后截取字符串返回
        int resLen = Integer.MAX_VALUE;  // 保存当前最优解的长度
        int start = 0;  // 保存当前最优解的起始位置，用于最后截取子串后返回
    
        // 初始化描述窗口的数据结构
        HashMap<Character, Integer> window = new HashMap<>();
    
        // 由于题目中的条件是 两个字符串之间的覆盖，则还需要使用一个 map 来保存被覆盖字符串中的所有字符
        HashMap<Character, Integer> need = new HashMap<>();
        for (int i = 0; i < t.length(); i++) {
            need.put(t.charAt(i), need.getOrDefault(t.charAt(i), 0) + 1);
        }
        
        while (right < s.length()){
            // 2. 窗口扩大，加入 right 对应元素
            char c = s.charAt(right);
            right++;
            if (need.containsKey(c)){
                window.put(c, window.getOrDefault(c, 0) + 1);
    						// 当前字符 c 达到覆盖，标记 + 1
                if (window.get(c).equals(need.get(c))){
                    valid++;
                }
            }
    
            // 3. 寻找最短问题：窗内元素满足条件时，控制 left 缩小窗口
            while (valid == need.size()){
    
                // 4. 更新最优结果
                if (right - left < resLen){
                    start = left;
                    resLen = right - left;
                }
    
                // 5. 窗口缩小，移除 left 对应元素
                char d = s.charAt(left);
                left++;
                if (need.containsKey(d)){
                    if (window.get(d).equals(need.get(d))){
                        valid--;
                    }
                    window.put(d, window.getOrDefault(d, 0) - 1);
                }
            }
        }
    
        return resLen == Integer.MAX_VALUE ? "" : s.substring(start, start + resLen);
    }
    ```
