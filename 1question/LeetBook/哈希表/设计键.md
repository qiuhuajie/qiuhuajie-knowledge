> 模式识别：一旦需要特征进行分类，就应该利用散列表
> 
> 设计键：实际就是==**提取共同特征**==来分类
1. 在以前的问题中，键的选择相对简单。不幸的是，有时你必须考虑在使用哈希表时设计合适的键
2. 示例：
    1. 给定一组字符串，将字母异位词组合在一起
    2. 众所周知，哈希映射可以很好地按键分组信息。但是我们不能直接使用原始字符串作为键
    3. 我们必须设计一个合适的键来呈现字母异位词的类型。例如，有字符串 “eat” 和 “ate” 应该在同一组中。但是 “eat” 和 “act” 不应该组合在一起
3. 实际上，设计关键是在原始信息和哈希映射使用的实际键之间建立映射关系。设计键时，需要保证：
    1. 属于同一组的所有值都将映射到同一组中
    2. 需要分成不同组的值不会映射到同一组
4. 在上面的示例中，我们的映射策略可以是：对字符串进行排序并使用排序后的字符串作为键。也就是说，“eat” 和 “ate” 都将映射到 “aet”

    public class IsValidSudoku {
        public boolean isValidSudoku(char[][] board) {
            //统计每行、列、九宫格 出现数字的次数
            // 9行10个数字
            int[][] row = new int[9][10];
            // 9列10个数字
            int[][] colume = new int[9][10];
            // 9个九宫格10个数组
            int[][] box = new int[9][10];
            for (int i = 0; i < 9; i++) {
                for (int j = 0; j < 9; j++) {
                    char c = board[i][j];
                    if (c == '.'){
                        //结束本次循环
                        continue;
                    }
                    int curNum = c - '0';
                    if (row[i][curNum] >= 1){
                        return false;
                    }
                    if (colume[j][curNum] >= 1){
                        return false;
                    }
                    // box[(i / 3) * 3 + j / 3 ：计算当前数字，在哪个九宫格内
                    if (box[(i / 3) * 3 + j / 3][curNum] >= 1){
                        return false;
                    }
                    row[i][curNum] = 1;
                    colume[j][curNum] = 1;
                    box[(i / 3) * 3 + j / 3][curNum] = 1;
                }
            }
            return true;
        }

    public static boolean isValidSudoku(char[][] board) {

        //统计每行、列、九宫格 出现数字的次数

        HashMap<Character, Integer>[] row = new HashMap[9];

        HashMap<Character, Integer>[] colume = new HashMap[9];

        HashMap<Character, Integer>[] box = new HashMap[9];

        for (int i = 0; i < 9; i++) {

            row[i] = new HashMap<Character, Integer>();

            colume[i] = new HashMap<Character, Integer>();

            box[i] = new HashMap<Character, Integer>();

        }

        for (int i = 0; i < 9; i++) {

            for (int j = 0; j < 9; j++) {

                char c = board[i][j];

                if (c == '.') {

                    //结束本次循环

                    continue;

                }

                int curNum = c - '0';

                Integer rowNum = row[i].getOrDefault(c, 0);

                row[i].put(c, rowNum + 1);

                Integer columeNum = colume[j].getOrDefault(c, 0);

                colume[j].put(c, columeNum + 1);

                int boxIndex = (i / 3) * 3 + j / 3;

                Integer boxNum = box[(i / 3) * 3 + j / 3].getOrDefault(c, 0);

                box[boxIndex].put(c, boxNum + 1);

                if (row[i].get(c) > 1 || colume[j].get(c) > 1 || box[boxIndex].get(c) > 1) {

                    return false;

                }

            }

        }

        return true;

    }

### 4.3 📑380. O(1) 时间插入、删除和获取随机元素
1. 问题描述：

    ```Plain
    实现RandomizedSet 类：
    RandomizedSet() 初始化 RandomizedSet 对象
    bool insert(int val) 当元素 val 不存在时，向集合中插入该项，并返回 true ；否则，返回 false 。
    bool remove(int val) 当元素 val 存在时，从集合中移除该项，并返回 true ；否则，返回 false 。
    int getRandom() 随机返回现有集合中的一项（测试用例保证调用此方法时集合中至少存在一个元素）。每个元素应该有 相同的概率 被返回。
    你必须实现类的所有函数，并满足每个函数的 平均 时间复杂度为 O(1)
    ```
2. 示例：

    ```Plain
    输入
    ["RandomizedSet", "insert", "remove", "insert", "getRandom", "remove", "insert", "getRandom"]
    [[], [1], [2], [2], [], [1], [2], []]
    输出
    [null, true, false, true, 2, true, false, 2]
    解释
    RandomizedSet randomizedSet = new RandomizedSet();
    randomizedSet.insert(1); // 向集合中插入 1 。返回 true 表示 1 被成功地插入。
    randomizedSet.remove(2); // 返回 false ，表示集合中不存在 2 。
    randomizedSet.insert(2); // 向集合中插入 2 。返回 true 。集合现在包含 [1,2] 。
    randomizedSet.getRandom(); // getRandom 应随机返回 1 或 2 。
    randomizedSet.remove(1); // 从集合中移除 1 ，返回 true 。集合现在包含 [2] 。
    randomizedSet.insert(2); // 2 已在集合中，所以返回 false 。
    randomizedSet.getRandom(); // 由于 2 是集合中唯一的数字，getRandom 总是返回 2
    ```
3. 代码：
    1. 删除任意索引元素需要线性时间，这里的解决方案是总是删除最后一个元素
        1. 将要删除元素和最后一个元素交换
        2. 将最后一个元素删除
    2. 为此，==必须在====**常数时间**====获取到要删除元素的索引，因此需要一个哈希表来存储值到索引的映射==
    3. **哈希表 + 动态数组**

        ![[IMG-20260620203902683.png|632]]

    ```Java
    public class RandomizedSet {
        HashMap<Integer, Integer> hashMap;
        ArrayList<Integer> list;
        Random random = new Random();
        public RandomizedSet() {
            hashMap = new HashMap<>();
            list = new ArrayList<>();
        }
        public boolean insert(int val) {
            if (hashMap.containsKey(val)){
                return false;
            }
            hashMap.put(val,list.size());
            list.add(list.size(),val);
            return true;
        }
        public boolean remove(int val) {
            if (!hashMap.containsKey(val)){
                return false;
            }
            int lastNum = list.get(list.size() -1);
            //⭐必须在常数时间获取到要删除元素的索引，因此需要一个哈希表来存储值到索引的映射
            int index = hashMap.get(val);
            list.set(index,lastNum);
            list.remove(list.size() - 1);
            // hashmap 要先放再删：如果要删除前，map只有唯一一个元素
            // 先删再放 会导致删除失败
            hashMap.put(lastNum,index);
            hashMap.remove(val);
            return true;
        }
        public int getRandom() {
            return list.get(random.nextInt(list.size()));
        }
    }
    ```
