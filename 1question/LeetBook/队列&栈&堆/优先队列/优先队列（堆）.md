# 一、📑215. 数组中的第K个最大元素🔥
1. 问题描述：

    ```Plain
    给定整数数组 nums 和整数 k，请返回数组中第 k 个最大的元素。
    请注意，你需要找的是数组排序后的第 k 个最大的元素，而不是第 k 个不同的元素。
    你必须设计并实现时间复杂度为 O(n) 的算法解决此问题。
    ```
2. 示例：

    ```Plain
    示例 1:
    输入: [3,2,1,5,6,4], k = 2
    输出: 5
    示例 2:
    输入: [3,2,3,1,2,4,5,5,6], k = 4
    输出: 4
    ```
3. 代码：
    1. [[Export-e8249ac1-4ae1-42cd-83d5-041a266841fe/1question/LeetBook/排序算法/快速排序|快速排序]]
    2. 优先队列（堆）

        ```Java
        public static int findKthLargest(int[] nums, int k) {
            // 优先队列（小根堆）
            PriorityQueue<Integer> minHeap = new PriorityQueue<>(k, (o1, o2) -> o1 - o2);
            // 先将前 k 个放在堆中
            for (int i = 0; i < k; i++) {
                minHeap.offer(nums[i]);
            }
            // 遍历剩下的元素
            for (int i = k; i < nums.length; i++) {
                // 只有当前元素大于堆顶元素时，将当前元素插入堆中
                if (nums[i] > minHeap.peek()) {
                    // 插入方式：先 poll() 出来，然后再放回去
                    minHeap.poll();
                    minHeap.offer(nums[i]);
                }
            }
            return minHeap.peek();
        }
        ```

    ```Plain
     给你链表的头结点 head ，请将其按 升序 排列并返回 排序后的链表。
    ```
2. 示例：

    [![|331](https://assets.leetcode.com/uploads/2020/09/14/sort_list_1.jpg)](https://assets.leetcode.com/uploads/2020/09/14/sort_list_1.jpg)

    ```Plain

    输出：[1,2,3,4]
    ```
3. 代码：

    ```Java
    public ListNode sortList(ListNode head) {
    		// 1. 入堆
        PriorityQueue<ListNode> minHeap = new PriorityQueue<>((o1, o2) -> {
            return o1.val - o2.val;
        });
        ListNode p = head;
        while (p != null) {
            minHeap.add(new ListNode(p.val));
            p = p.next;
        }
    		// 2. 依次弹出堆，构建新链表
        ListNode res = new ListNode(-1);
        p = res;
        while (!minHeap.isEmpty()) {
            p.next = minHeap.poll();
            p = p.next;
        }
        return res.next;
    }
    ```

        ```Java
        public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
            PriorityQueue<ListNode> heap = new PriorityQueue<>((o1, o2) -> o1.val - o2.val);
            while (l1 != null) {
                heap.offer(new ListNode(l1.val));     // 新 new 结点加入堆中，不然结果会成环
                l1 = l1.next;
            }
            while (l2 != null) {
                heap.offer(new ListNode(l2.val));
                l2 = l2.next;
            }
            ListNode pre = new ListNode(-1);
            ListNode p = pre;
            while (!heap.isEmpty()) {
                p.next = heap.poll();
                p = p.next;
            }
            return pre.next;
        }
        ```


        ```Java
        public ListNode mergeKLists(ListNode[] lists) {
            // 定义一个小根堆
            PriorityQueue<ListNode> queue = new PriorityQueue<>(new Comparator<ListNode>() {
                @Override
                public int compare(ListNode o1, ListNode o2) {
                    return o1.val - o2.val;
                }
            });
            // queue add() 时要新 new 一个节点，或者要将放入堆中元素的 next 置为 null
            for (ListNode list : lists) {
                ListNode cur = list;
                while (cur != null) {
                    queue.add(new ListNode(cur.val));
                    cur = cur.next;
        						// heap.add(p);
        						// pre = p;
                    // p = p.next;
                    // pre.next = null;
                }
            }
            // 将堆中数据依次取出，串成一个链
            ListNode pre = new ListNode(0);
            ListNode p = pre;
            while (!queue.isEmpty()) {
                p.next = queue.poll();
                p = p.next;
            }
            return pre.next;
        }
        ```

1. 问题描述：

    ```Plain
    如何得到一个数据流中的中位数？如果从数据流中读出奇数个数值，那么中位数就是所有数值排序之后位于中间的数值。如果从数据流中读出偶数个数值，那么中位数就是所有数值排序之后中间两个数的平均值。
    例如，
    [2,3,4] 的中位数是 3
    [2,3] 的中位数是 (2 + 3) / 2 = 2.5
    ```
2. 代码：
    1. 用一个小顶堆和一个大顶堆，将数据流中的数据以较大的一半和较小的一半维护起来
    2. 维持小顶堆个数（m）比大顶堆个数（n）多1，或相等（要么是 `m = n`，要么是 `m = n + 1`）
    3. 添加元素：
        1. 当 m = n 时：**为了维护数量关系，要向小顶堆中加元素**。因此：==当前元素加入大顶堆，将大顶堆堆顶元素弹出加入小顶堆==（看起来像从大顶堆中换一个出来到小顶堆）
            > 💡 如何保证当前元素被正确的放大他应该在的那一半？
            > 1. 如果当前元素比大顶堆的所有元素都大，意味着当前元素会到大顶堆顶，自然也就能加入小顶堆中
            > 2. 如果当前元素比大顶堆的堆顶元素小，意味着当前元素就应该呆在大顶堆中
        2. 当 m = n + 1 时：**为了维护数量关系，要向大顶堆中加元素**。因此：==当前元素加入小顶堆，将小顶堆堆顶元素弹出加入小顶堆==（原因同上）

    ![[IMG-20260620210300984.png|542]]

    ```Java
     class MedianFinder {
        PriorityQueue<Integer> minHeap = null;
        PriorityQueue<Integer> maxHeap = null;
        public MedianFinder() {
            // 小根堆（存储较大的一半）
            minHeap = new PriorityQueue<>((o1, o2) -> o2 - o1);
            // 大根堆（存储较小的一半）
            maxHeap = new PriorityQueue<>((o1, o2) -> o1 - o2);
        }
        public void addNum(int num) {
            if (minHeap.size() == maxHeap.size()) {
                maxHeap.offer(num);
                minHeap.offer(maxHeap.poll());
            } else {
                minHeap.offer(num);
                maxHeap.offer(minHeap.poll());
            }
        }
        // 返回元素
        public double findMedian() {
            if (minHeap.isEmpty()) {
                return -1;
            }
            if (minHeap.size() == maxHeap.size()) {
    						// 偶数个数时，返回中间的平均值
                return (double) (minHeap.peek() + maxHeap.peek()) / 2;
            } else {
    						// 奇数个数时，返回中间的值
                return minHeap.peek();
            }
        }
    }
    ```
# 六、📑剑指 Offer 49. 丑数⚔️
1. 问题描述：

    ```Plain
    给你一个整数 n ，请你找出并返回第 n 个 丑数 。
    丑数 就是只包含质因数 2、3 和/或 5 的正整数。
    ```
2. 示例：

    ```Plain
    示例 1：
    输入：n = 10
    输出：12
    解释：[1, 2, 3, 4, 5, 6, 8, 9, 10, 12] 是由前 10 个丑数组成的序列。
    示例 2：
    输入：n = 1
    输出：1
    解释：1 通常被视为丑数。
    ```
3. 代码：

    ![[IMG-20260620210301009.png|566]]

    ```Java
     public int nthUglyNumber(int n) {
        PriorityQueue<Long> heap = new PriorityQueue<>();
        HashSet<Long> set = new HashSet<>();
        int[] factors = {2, 3, 5};
        int res = 0;
        heap.offer(1L);
        set.add(1L);
        for (int i = 0; i < n; i++) {
            long cur = heap.poll();
            res = (int) cur;
            for (int f : factors) {
                long next = f * cur;
    						// 去重，保证往堆中加的数都是没加入过的
                if (!set.contains(next)) {
                    heap.offer(next);
                    set.add(next);
                }
            }
        }
        return res;
    }
    ```
