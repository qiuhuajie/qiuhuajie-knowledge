# 一、单调队列介绍
1. 单调队列和单调栈相似，也是在元素加入队列时做了一点特殊处理
2. **单调队列中所有元素都是单调递增或递减的**
3. 可以解决滑动窗口的一系列问题：比如滑动窗口的最大值
4. 单调队列的实现

    ```Java
    public class MonotoneQueue {
        static LinkedList<Integer> q = null;
        public MonotoneQueue() {
            q = new LinkedList<>();
        }
        // 从尾部入队
        public void push(int n) {
            // 将前面小于自己的元素都删除
            while (!q.isEmpty() && q.getLast() < n) {
                q.pollLast();
            }
            q.addLast(n);
        }
        // 获取队中最大值
        public int max() {
            // 有单调队列的性质得，队头的元素一定是最大的
            return q.getFirst();
        }
        // 从头部出队
        public void pop(int n) {
            // 之所以要判断要删除的元素n 是不是队头的元素，是因为整个元素n 可能已经在push时被弹出了
            if (n == q.getFirst()) {
                q.pollFirst();
            }
        }
    }
    ```
# 二、例题
## 1. 📑剑指 Offer 59 - II. 队列的最大值⚔️
1. 问题描述：

    ```Plain
    请定义一个队列并实现函数 max_value 得到队列里的最大值，要求函数max_value、push_back 和 pop_front 的均摊时间复杂度都是O(1)。
    若队列为空，pop_front 和 max_value 需要返回 -1
    ```
2. 示例：

    ```Plain
    示例 1：
    输入: 
    ["MaxQueue","push_back","push_back","max_value","pop_front","max_value"]
    [[],[1],[2],[],[],[]]
    输出: [null,null,null,2,1,2]
    示例 2：
    输入: 
    ["MaxQueue","pop_front","max_value"]
    [[],[],[]]
    输出: [null,-1,-1]
    ```
3. 代码：

    ![[IMG-20260620210107812.png|585]]

    ```Java
    public class MaxQueue {
        LinkedList<Integer> queue = null;
        LinkedList<Integer> max_queue = null;
        public MaxQueue() {
            queue = new LinkedList<>();
            max_queue = new LinkedList<>();
        }
        public int max_value() {
            if (!max_queue.isEmpty()) {
                return max_queue.getFirst();
            } else {
                return -1;
            }
        }
        public void push_back(int value) {
            queue.offer(value);
    				// max_queue 是一个单调双向队列
            if (!max_queue.isEmpty()) {
                while (max_queue.getLast() <= value) {
                    max_queue.removeLast();
                }
            }
            max_queue.addLast(value);
        }
        public int pop_front() {
            if (!queue.isEmpty()) {
                Integer poll = queue.poll();
    						// 判断要删除的元素是不是当前最大值，如果是，要维护 max_queue 
                if (max_queue.getFirst().equals(poll)) {
                    max_queue.removeFirst();
                }
                return poll;
            } else {
                return -1;
            }
        }
    }
    ```
# 三、📑239. 滑动窗口最大值🤬
> ℹ️ 力扣
> [https://leetcode.cn/problems/sliding-window-maximum/](https://leetcode.cn/problems/sliding-window-maximum/)
1. 问题描述：

    ```Plain
    给你一个整数数组 nums，有一个大小为 k 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 k 个数字。滑动窗口每次只向右移动一位。
    返回 滑动窗口中的最大值 。
    ```
2. 示例：

    ```Plain
    示例 1：
    输入：nums = [1,3,-1,-3,5,3,6,7], k = 3
    输出：[3,3,5,5,6,7]
    解释：
    滑动窗口的位置                最大值
    ---------------               -----
    [1  3  -1] -3  5  3  6  7       3
     1 [3  -1  -3] 5  3  6  7       3
     1  3 [-1  -3  5] 3  6  7       5
     1  3  -1 [-3  5  3] 6  7       5
     1  3  -1  -3 [5  3  6] 7       6
     1  3  -1  -3  5 [3  6  7]      7
    示例 2：
    输入：nums = [1], k = 1
    输出：[1]
    ```
3. 分析：
    1. 难点在于如何在 $O(1)$ 时间算出每个窗口中的最大值，使得整个算法在线性时间内完成
    2. 窗口是滑动的，也就是要动态地计算窗口中的最大值
    3. 在⼀堆数字中， 已知最值， 如果给这堆数添加⼀个数， 那么⽐较⼀下就可以很快算出最值； **但如果减少⼀个数， 就不⼀定能很快得到最值了， 而要遍历所有数重新找最值**
    4. 回到这道题的场景， 每个窗⼝前进的时候， 要添加⼀个数同时减少⼀个数，所以想在 $O(1)$ 的时间得出新的最值， 就需要「单调队列」 这种特殊的数据结构来辅助了
4. 代码：

    ```Java
     public static int[] maxSlidingWindow(int[] nums, int k) {
            // 定义一个固定窗口
            MonotoneQueue window = new MonotoneQueue();
            // 结果集
            ArrayList<Integer> res = new ArrayList<>();
            for (int i = 0; i < nums.length; i++) {
                // 先填满窗口的前 k - 1
                if (i < k - 1) {
                    window.push(nums[i]);
                } else {
                    // 向右滑动窗口
                    window.push(nums[i]);
                    // 记录当前窗口的最大值
                    res.add(window.max());
                    // 将窗口第一个数字出队（这里固定指出窗口第一个，但是注意：很可能那个元素已经在新元素入队时因单调队列的性质被弹出了）
                    window.pop(nums[i - k + 1]);
                }
            }
            // list 转 int[]
            int[] ints = new int[res.size()];
            for (int i = 0; i < ints.length; i++) {
                ints[i] = res.get(i);
            }
            return ints;
        }
    ```
