# 一、链表介绍
1. 单链表中的每个结点不仅包含值，还包含链接到下一个结点的引用字段。通过这种方式，单链表将所有结点按顺序组织起来
2. 下面是一个单链表的例子：蓝色箭头显示单个链接列表中的结点是如何组合在一起的

    ![[IMG-20260620200925546.png|476]]

3. 结点结构：以下是单链表中结点的典型定义

    ```Java
     // Definition for singly-linked list.
     public class SinglyListNode {
         int val;
         SinglyListNode next;
         SinglyListNode(int x) { val = x; }
     }
    ```
4. 在大多数情况下，我们将**使用头结点(第一个结点)来表示整个列表**
5. 单链表的**遍历操作**
    1. 与数组不同，无法在常量时间内访问单链表中的随机元素
    2. 如果我们想要获得第 i 个元素，我们**必须从头结点逐个遍历**。 按索引来访问元素平均要花费 O(N) 时间，其中 N 是链表的长度
6. 单链表的**添加操作**
    1. 与数组不同，**不需要将所有元素移动**到插入元素之后，可以在 O(1) 时间复杂度中将新结点插入到链表中，这非常高效
    2. 如果我们想在给定的结点 prev 之后添加新值，我们应该：
        1. 使用给定值初始化新结点 cur
        2. 将 cur 的 next 字段链接到 prev 的下一个结点 next
        3. 将 prev 中的 next 字段链接到 cur

1. **在开头添加结点**
    1. 由于使用头结点来代表整个列表，因此，在列表开头添加新节点时**更新头结点 head** 至关重要
        1. 初始化一个新结点 cur
        2. 将新结点链接到我们的原始头结点 head
        3. 将 cur 指定为 head
    2. 例如，在列表的开头添加一个新结点 9 。初始化一个新结点 9 并将其链接到当前头结点 23。再指定结点 9 为新的头结点

        ![[IMG-20260620200925845.png|505]]

2. 单链表**删除操作**
    1. 如果我们想从单链表中删除现有结点 cur，可以分两步完成：
        1. 找到 cur 的上一个结点 prev 及其下一个结点 next
        2. 接下来链接 prev 到 cur 的下一个节点 next
    2. 在第一步中，需要找出 prev 和 next
        1. 使用 cur 的参考字段很容易找出 next
        2. 但是，**必须从头结点遍历链表，以找出 prev**，它的平均时间是 O(N)，其中 N 是链表的长度
        3. 因此，删除结点的时间复杂度将是 O(N)
    3. 空间复杂度为 O(1)，因为只需要常量空间来存储指针
3. 设计链表：

    ```Java
     class ListNode{
         int val;
         ListNode next;
         ListNode(int x){
             val = x;
             next = null;
         }
     }
     class MyLinkedList {
         int size;
         ListNode head;
         public MyLinkedList(){
             size = 0;
             head = new ListNode(0);
         }
         //get(index)：获取链表中第 index 个节点的值。如果索引无效，则返回-1。
         public int get(int index){
             if (index < 0  || index >= size){
                 return -1;
             }
             ListNode curr = head;
             for (int i = 0; i < index + 1; i++) {
                 curr = curr.next;
             }
             return curr.val;
         }
         //addAtHead(val)：在链表的第一个元素之前添加一个值为 val 的节点。插入后，新节点将成为链表的第一个节点。
         public void addAtHead(int val){
             addAtIndex(0,val);
         }
         //addAtTail(val)：将值为 val 的节点追加到链表的最后一个元素。
         public void addAtTail(int val){
             addAtIndex(size,val);
         }
         //addAtIndex(index,val)：在链表中的第 index 个节点之前添加值为 val  的节点。如果 index 等于链表的长度，则该节点将附加到链表的末尾。如果 index 大于链表长度，则不会插入节点。如果index小于0，则在头部插入节点。
         public void addAtIndex(int index,int val){
             ListNode curr = new ListNode(val);
             if (index < 0){
                 index = 0;
             }else if (index > size) return;
             ListNode pre = head;
             for (int i = 0; i < index; i++) {
                 pre = pre.next;
             }
             curr.next = pre.next;
             pre.next = curr;
             size++;
         }
         //deleteAtIndex(index)：如果索引 index 有效，则删除链表中的第 index 个节点。
         public void deleteAtIndex(int index){
             if (index >= size || index < 0){
                 return;
             }
             ListNode pre = head;
             for (int i = 0; i < index ; i++) {
                 pre = pre.next;
             }
             pre.next = pre.next.next;
             size--;
         }
     }
    ```

3. 代码：

    2. 双指针

        ```Java
         public class MergeTwoLists {
             public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
                 if (l1 == null){
                     return l2;
                 }else {
                     if (l2 == null){
                         return l1;
                     }
                 }
                 //选取两个链表首结点较小的作为结果链表的头部
                 ListNode head = null;
                 if (l1.val > l2.val){
                     head = l2;
                     l2 = l2.next;
                 }else {
                     head = l1;
                     l1 = l1.next;
                 }
                 //last 结果链的最后一个结点指针
                 ListNode last = head;
                 while (l1 != null && l2 != null){
                     if (l1.val > l2.val){
                         last.next = l2;
                         last = last.next;
                         l2 = l2.next;
                     }else {
                         last.next = l1;
                         last = last.next;
                         l1 = l1.next;
                     }
                 }
        				 // 处理剩余部分：合并后 l1 和 l2 最多只有一个还未被合并完，我们直接将链表末尾指向未合并完的链表即可
                 if (l1 == null){
                     last.next = l2;
                 }else {
                     last.next = l1;
                 }
                 return head;
             }
         }
        ```

## 2. 📑23. 合并K个升序链表🔥✔️
> ℹ️ 力扣
> [https://leetcode.cn/problems/merge-k-sorted-lists/submissions/](https://leetcode.cn/problems/merge-k-sorted-lists/submissions/)
1. 问题描述：

    ```Plain
    给你一个链表数组，每个链表都已经按升序排列。
    请你将所有链表合并到一个升序链表中，返回合并后的链表。
    ```
2. 示例：

    ```Plain
     输入：lists = [[1,4,5],[1,3,4],[2,6]]
    输出：[1,1,2,3,4,4,5,6]
    解释：链表数组如下：
    [
      1->4->5,
      1->3->4,
      2->6
    ]
    将它们合并到一个有序链表中得到。
    1->1->2->3->4->4->5->6
    ```
3. 代码：
    1. 方法1：使用堆[[优先队列（堆）]]
    2. 方法2：两两合并

        ```Java
        public ListNode mergeKLists(ListNode[] lists) {
            if (lists.length == 0) return null;
            // 将所有链表放在一个 LinkedList 中
            LinkedList<ListNode> list = new LinkedList();
            list.addAll(Arrays.asList(lists));
            // 合并后再放回去，直至只剩一个合并好的链表
            while (list.size() != 1) {
                ListNode l1 = list.removeFirst();
                ListNode l2 = list.removeFirst();
                ListNode sortedList = mergeKLists(l1, l2);
                list.addFirst(sortedList);
            }
            return list.removeFirst();
        }
        // 合并两个链表
        public ListNode mergeKLists(ListNode l1, ListNode l2) {
            if (l1 == null) {
                return l2;
            }
            if (l2 == null) {
                return l1;
            }
            ListNode head;
            if (l1.val >= l2.val) {
                head = l2;
                l2 = l2.next;
            } else {
                head = l1;
                l1 = l1.next;
            }
            ListNode p = head;
            while (l1 != null && l2 != null) {
                if(l1.val >= l2.val) {
                    p.next = l2;
                    l2 = l2.next;
                } else {
                    p.next = l1;
                    l1 = l1.next;
                }
                p = p.next;
            }
            if(l1 == null) {
                p.next = l2;
            }else {
                p.next = l1;
            }
            return head;
        ```
## 3. 📑2. 两数相加🔥✔️
> ℹ️ 力扣
> [https://leetcode.cn/problems/add-two-numbers/](https://leetcode.cn/problems/add-two-numbers/)
1. 问题描述：

    ```Plain
     给你两个 非空 的链表，表示两个非负的整数。它们每位数字都是按照 逆序 的方式存储的，并且每个节点只能存储 一位 数字。
     请你将两个数相加，并以相同形式返回一个表示和的链表。
     你可以假设除了数字 0 之外，这两个数都不会以 0 开头
    ```
2. 示例：

    ```Plain
     输入：l1 = [7,8,9], l2 = [3,2]
     输出：[0,1,0,1]
     解释：987 + 23 = 987 + 023 = 1010.
    ```

![[IMG-20260620200925941.png|302]]

![[IMG-20260620200926045.png|278]]

![[IMG-20260620200926102.png|295]]

1. 代码：

    ```Java
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode head = null, tail = null;
        int carry = 0;
        // 只要有一个非空就相加
        while (l1 != null || l2 != null) {
            // 如果出现 null，则用 0 补位相加（较短的链表则在前面补 0）
            int n1 = l1 == null ? 0 : l1.val;
            int n2 = l2 == null ? 0 : l2.val;
            int sum = n1 + n2 + carry;
            // 第一个节点要特殊处理
            if (head == null) {
                head = tail = new ListNode(sum % 10);
            } else {
                tail.next = new ListNode(sum % 10);
                tail = tail.next;
            }
            // 计算进位
            carry = sum / 10;
            // 指针后移
            if (l1 != null) {
                l1 = l1.next;
            }
            if (l2 != null) {
                l2 = l2.next;
            }
        }
        // 看最后剩的进位数是否为 0，不为 0 的话要在答案链表的后面附加一个节点
        if (carry > 0) {
            tail.next = new ListNode(carry);
        }
        return head;
    }
    ```

    ```Plain
     顺序存放在链表的两个数字相减，链表1放9，8，7，链表2放1,2,3输出一个链表放结果8,6,4
    ```
2. 代码：先将顺序改为逆序，在运算

    ![[IMG-20260620200926179.png|520]]

    ![[IMG-20260620200926269.png|527]]

    ```Java
    class ListNode {
        int val;
        ListNode next;
        ListNode(int val) {
            this.val = val;
        }
    }
    public class Diff {
        public static ListNode subtractLinkedList(ListNode l1, ListNode l2) {
            ListNode dummy = new ListNode(0);
            ListNode curr = dummy;
            int carry = 0;
            while (l1 != null || l2 != null) {
                int val1 = l1 != null ? l1.val : 0;
                int val2 = l2 != null ? l2.val : 0;
                int diff = val1 - val2 - carry;
                if (diff < 0) {
                    diff += 10;
                    carry = 1;
                } else {
                    carry = 0;
                }
                curr.next = new ListNode(diff);
                curr = curr.next;
                if (l1 != null) {
                    l1 = l1.next;
                }
                if (l2 != null) {
                    l2 = l2.next;
                }
            }
            if (carry > 0) {
                curr.next = new ListNode(carry);
            }
            return dummy.next;
        }
        public static void main(String[] args) {
            ListNode l1 = new ListNode(9);
            l1.next = new ListNode(8);
            l1.next.next = new ListNode(7);
            ListNode l2 = new ListNode(2);
            l2.next = new ListNode(9);
            l2.next.next = new ListNode(5);
            ListNode l11 = reverseList(l1);
            ListNode l22 = reverseList(l2);
            ListNode result = subtractLinkedList(l11, l22);
            ListNode res = reverseList(result);
            while (res != null) {
                System.out.print(res.val + " ");
                res = res.next;
            }
        }
        public static ListNode reverseList(ListNode head) {
            if (head == null || head.next == null) {
                return head;
            }
            ListNode pre = head;
            ListNode point = head.next;
            while (point != null) {
                pre.next = point.next;
                point.next = head;
                head = point;       //⭐要及时更新 head
                point = pre.next;
            }
            return head;
        }
    } 
    ```

## 6. 📑剑指 Offer II 026. 重排链表（综合应用）⚔️
1. 问题描述：

    ```Plain
     给定一个单链表 L 的头节点 head ，单链表 L 表示为：
     L0 → L1 → … → Ln-1 → Ln 
    请将其重新排列后变为：
    L0 → Ln → L1 → Ln-1 → L2 → Ln-2 → …
    不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。
    ```
2. 示例：

    ```Plain
     输入: head = [1,2,3,4,5]
    输出: [1,5,2,4,3]
    ```
3. 代码：==**寻找链表中点 + 链表逆序 + 合并链表**==

    ```Java
    class Solution {
        public void reorderList(ListNode head) {
            if (head == null) {
                return;
            }
    				// 找到中点，切分链表
            ListNode mid = middleNode(head);
            ListNode l1 = head;
            ListNode l2 = mid.next;
            mid.next = null;
    				// 反转后一个链表
            l2 = reverseList(l2);
    				// 合并两个链表
            mergeList(l1, l2);
        }
        public ListNode middleNode(ListNode head) {
            ListNode slow = head;
            ListNode fast = head.next;
            while (fast != null && fast.next != null) {
                slow = slow.next;
                fast = fast.next.next;
            }
            return slow;
        }
        public ListNode reverseList(ListNode head) {
            if (head == null || head.next == null) {
                return head;
            }
            ListNode pre = head;
            ListNode p = head.next;
            while (p != null) {
                pre.next = p.next;
                p.next = head;
                head = p;
                p = pre.next;
            }
            return head;
        }
        public void mergeList(ListNode l1, ListNode l2) {
            ListNode l1_tmp;
            ListNode l2_tmp;
            while (l1 != null && l2 != null) {
                l1_tmp = l1.next;
                l2_tmp = l2.next;
                l1.next = l2;
                l1 = l1_tmp;
                l2.next = l1;
                l2 = l2_tmp;
            }
        }
    }
    ```
## 7. 📑24. 两两交换链表中的节点🔥✔️
1. 问题描述：

    ```Plain
    给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。你必须在不修改节点内部的值的情况下完成本题（即，只能进行节点交换）。
    ```
2. 示例：

    ```Plain
    输入：head = [1,2,3,4]
    输出：[2,1,4,3]
    输入：head = []
    ```

    ```Java
    public ListNode reverseKGroup(ListNode head, int k) {
        ListNode node = new ListNode(-1);
        node.next = head;
        ListNode pre = node;
        ListNode start = head;
        ListNode tail = node;
        while (start != null) {
            for (int i = 0; i < k; i++) {
                tail = tail.next;
                // 如果剩下的节点不够 k 了，直接返回最终结果不做处理
                if (tail == null) {
                    return node.next;
                }
            }
            // 记住下一个待翻转链表的起始位置
            ListNode next = tail.next;
            // 翻转当前起始位置内的链表
            ListNode[] reverse = reverse(start, tail);
            // 返回反转后的起始节点
            start = reverse[0];
            tail = reverse[1];
            // 将反转后的链表重新连接回原链表
            pre.next = start;
            tail.next = next;
            // 移动 pre 和 start
            pre = tail;
            start = tail.next;
        }
        return node.next;
    }
    private ListNode[] reverse(ListNode start, ListNode tail) {
        ListNode end = tail.next;
        ListNode pre = start;
        ListNode p = start.next;
        while (p != end) {
            pre.next = p.next;
            p.next = start;
            start = p;
            p = pre.next;
            tail = pre;
        }
        return new ListNode[]{start, tail};
    }
    ```
