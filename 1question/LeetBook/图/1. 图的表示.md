# **1. 邻接矩阵**
1. 邻接矩阵法：
    
    - 用一个**一维数组**存储图中**顶点**的信息
    
    - 用一个**二维数组**存储图中**边**的信息（即各顶点之间的邻接关系），存储顶点之间邻接关系的二维数组称为邻接矩阵
        

        ![[IMG-20260405035400229.png|Untitled 107.png]]

        
    
1. 数据结构
    
    ```C++
     \#define MaxVertexNum 100 //顶点数目的最大值
     typedef struct
     {
         char Vex[MaxVertexNum]; //顶点表
         int Edge[MaxVertexNum][MaxVertexNum]; //领接矩阵，边表
         int vexnum, arcnum; //图的当前顶点数和弧数
     }MGraph;
    ```
    
1. 结点数为 `n` 的图 `G=(V, E)` 的邻接矩阵 `A` 是 `n*n`的，将G的顶点编号为`v1, v2,…, vn`，若`(vi, vj)∈E`，则`A[i][j]=1`，否则`A[i][j]=0`
    

    ![[IMG-20260405035401324.png|Untitled 1 69.png]]

    
1. 邻接矩阵的性能：
    
    1. 无向图的邻接矩阵对称且唯一
    
    1. 有向图的邻接矩阵的第 i 行非零元素个数为第 i 个顶点的出度；第 j 列非零元素个数为第 j 个顶点的入度
    
    1. 稠密图更适合用邻接矩阵存储
    
    1. 无向图的邻接矩阵是对称矩阵，对规模特大的邻接矩阵可采用压缩存储
    
    1. 邻接矩阵表示法的空间复杂度为`O(n2)`，其中n为图的顶点数`|V|`
    
# **2. 邻接表**
1. 当一个图为稀疏图时，使用邻接矩阵表示显然要浪费大量的存储空间，而图的邻接表法结合了顺序存储和链式存储方法，大大减少了这种不必要的浪费
1. 把数组与链表相结合的存储方法称为邻接表
    

    ![[IMG-20260405035403375.png|Untitled 2 56.png]]

    
    1. 图中顶点用一个一维数组存储，对于一维顶点数组中，每个数组元素还需要存储指向第一个邻接点的指针
    
    1. 图中每个顶点的 `V_i` 所有邻接点构成一个线性表，由于邻接点的个数不确定，所以用单链表来存储，无向图称为顶点 `V_i` 的边表，有向图称为顶点 `V_i` 作为弧尾的出边表
    
# **3. 📑图的数据结构**

> [!important]
> 
> - 这是一个==自己设计的数据结构==（==图的题目最大的难点就在于，题目中给的图结构各种各样，有很多变式==）
> 
> - 可以在做题过程中，先将题目中给出的各种各样的图的表示方式**转换成这个自己的图结构**，那么所有处理方法就统一了
1. 顶点：
    
    ```Java
     class Node {
         public int value;  //顶点值
         public int in;  //入度
         public int out;  //出度
         public ArrayList<Node> nexts;  //顶点的邻接顶点
         public ArrayList<Edge> edges;  //顶点的邻边
     
         //构造函数
         public Node(int value) {
             this.value = value;
             in = 0;
             out = 0;
             nexts = new ArrayList<>();
             edges = new ArrayList<>();
         }
     }
    ```
    
1. 边：
    
    ```Java
     class Edge {
         public int weight;  //边的权重
         public Node from;  //边的出发顶点
         public Node to;  //边的指向顶点
     
         //边的构造函数
         public Edge(int weight, Node from, Node to) {
             this.weight = weight;
             this.from = from;
             this.to = to;
         }
     }
    ```
    
1. 图：
    
    ```Java
     public class Graph {
         public HashMap<Integer,Node> nodes;  //点集
         public HashSet<Edge> edges;  //边集
     
         public Graph() {
             nodes = new HashMap<>();
             edges = new HashSet<>();
         }
     }
    ```
