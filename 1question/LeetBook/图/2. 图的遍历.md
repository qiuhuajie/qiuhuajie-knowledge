# **1. 📑宽度优先遍历 BFS**
1. 原理：
    
    1. 利用队列实现
    
    1. 从源节点开始依次按照宽度进队列， 然后弹出
    
    1. 每弹出一个点， 把该节点**==所有==**没有进过队列的邻接点放入队列
    
    1. 直到队列变空
    
1. 代码：
    
    ```Java
     public class BFS {
         public static void bfs(Node node){
             if (node == null) {
                 return;
             }
             //需要使用一个set来保证不重复遍历顶点
             HashSet<Node> set = new HashSet<>();
             set.add(node);
    
             //广度优先遍历需要使用一个队列
             Queue<Node> queue = new LinkedList<>();
             queue.add(node);
    
             while (!queue.isEmpty()){
                 Node p = queue.poll();
                 //遍历当前节点的所有邻接顶点
                 for (Node next : p.nexts) {
                     //如果之前没有遍历过，则进入队列等待遍历，并加入set
                     if (!set.contains(next)) {
                         set.add(node);  //遍历过了就需要加入set
                         System.out.println(next.value);
                         queue.add(next);
                     }
                 }
             }
         }
     }
    ```
    
# **2. 📑深度优先遍历 DFS**
1. 原理：
    
    1. 利用栈实现
    
    1. 从源节点开始把节点按照深度放入栈， 然后弹出
    
    1. 每弹出一个点，把该节点**==下一个==**没有进过栈的邻接点放入栈
    
    1. 直到栈变空
    
1. 代码：
    
    ```Java
     public class DFS {
         public static void dfs(Node node){
             if (node == null) {
                 return;
             }
             //需要使用一个set来保证不重复遍历顶点
             HashSet<Node> set = new HashSet<>();
             set.add(node);
     
             //深度优先遍历需要使用一个栈
             Stack<Node> stack = new Stack<>();
             stack.add(node);
     
             while (!stack.isEmpty()){
                 Node p = stack.pop();
                 //遍历当前节点的所有邻接顶点
                 for (Node next : p.nexts) {
                     //如果之前没有遍历过，则进入队列等待遍历，并加入set
                     if (!set.contains(next)){
                         set.add(next);  //遍历过了就需要加入set
                         System.out.println(next.value);
                         //⭐深度优先遍历的思想就是逮住一个一直往下遍历
                         stack.push(p);  //故这里还需要将当前节点再次入栈
                         stack.push(next);
                         break;  //由于这里只需要在p.nexts中找到一个未遍历过的节点就好，剩下的就不需要在遍历了，这里要跳出遍历
                     }
                 }
             }
         }
     }
    ```
