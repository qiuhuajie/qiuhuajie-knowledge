当对某个指令不熟悉时，可以使用Linux提供的帮助指令来了解这个指令的使用方法

# 一、man：获得帮助信息

语法：`man [命令或配置文件]`

```Plain
 [root@hadoop1 ~]# man ls
```
> Linux下，隐藏文件是以 . 开头的命令的选项可以组合使用，如：ls -al \to ls -al /root
# 二、help：获得shell内置命令的帮助信息

语法：`help 命令`

```Plain
 [root@hadoop1 ~]# help cd
```
