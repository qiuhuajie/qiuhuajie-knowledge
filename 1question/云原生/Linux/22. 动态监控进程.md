1. 介绍：
    1. top与ps命令很相似。它们都用来显示正在执行的进程
    2. ==Top与ps最大的不同之处，在于== ==**top 会动态更新进程的运行情况**==
2. 基本语法：`top [选项]`
3. 常用选项：
    1. `-d` 秒数：指定top指令每隔几秒更新，默认是3秒
    2. `-i`：使top不显示任何闲置或僵死(zombie)的进程
    3. `-p`：通过指定监控进程ID来仅仅监控某个进程的状态

        ![[IMG-20260404032000280.png|800]]
4. 交互操作的说明：

| 操作 | 功能 |
| --- | --- |
| P | 以CPU使用率排序（默认） |
| M | 以内存使用率排序 |
| N | 以PID排序 |
| q | 退出top |
5. **监控指定的用户进程：**
    1. 输入top
    2. 输入u，再输入指定进程的用户名
6. **终止指定的进程：**
    1. 输入top
    2. 输入k，再输入指定进程的PID
