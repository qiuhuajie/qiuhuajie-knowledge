# 一、监控网络状态
1. 基本语法：`netstat [选项]`
2. 选项说明
    1. `an` 按一定顺序排列输出
    2. `p` 显示哪个进程在调用

    ![[IMG-20260404032000309.png|800]]

    ![[IMG-20260404032000374.png|800]]

    > 💡 **注意：**
    >
    > 如果此时tom用户logout，但`9408`端口的链接并不会立马断开，而是进入**`TIME_WAIT`**状态（**TCP协议的机制，担心网络延时原因，故会超时等待一分钟**）
3. 应用案例

    ```Bash
     //请查看服务名为 sshd 的服务的信息
     netstat -anp | sshd
    ```
