# 一、pwd 指令（print Working directory）
1. 基本语法：`pwd`
2. 功能描述：显示当前工作目录的绝对路径
# 二、ls 指令（list）
1. 基本语法：`ls [选项] [目录或是文件]`
2. 常用选项
    1. a ：显示当前目录所有的文件和目录，包括隐藏的
    2. l ：以列表的方式显示信息

# 三、cd指令（change directory）
1. 基本语法：`cd [参数]`
2. 功能描述：切换到指定目录
3. 常用参数：
    1. `cd ~` 或 `cd :`回到自己的家目录
    2. `cd ..`回到当前目录的上一级目

        ```Plain
         [root@hadoop1 tom]# who am i
         root     pts/0        2021-06-25 19:17 (192.168.176.1)
         [root@hadoop1 tom]# cd~
         bash: cd~: 未找到命令...//有空格
         [root@hadoop1 tom]# cd ~
         [root@hadoop1 ~]#
         [root@hadoop1 ~]# cd /home
         [root@hadoop1 home]# cd tom
         [root@hadoop1 tom]# cd ..
         [root@hadoop1 home]#
        ```

        也可以直接使用路径切换目录

        ```Plain
         //使用绝对路径
         [root@hadoop1 ~]# cd /home/tom
         [root@hadoop1 tom]# cd /root
         [root@hadoop1 ~]#
         //使用相对路径
         [root@hadoop1 ~]# cd /home/tom
         [root@hadoop1 tom]# cd ../../root
         [root@hadoop1 ~]#
        ```
# 四、mkdir 指令（make directories）
1. 基本语法：`mkdir [选项] 要创建的目录`
2. 功能描述：用于创建目录

    ```Plain
     [root@hadoop1 ~]# mkdir /home/dog
     [root@hadoop1 ~]# cd /home
     [root@hadoop1 home]# ls
     cool  dog  jack  tom
    ```
3. 常用选项：
    - p 创建多级目录

    ```Plain
     [root@hadoop1 home]# mkdir /home/animal/cat
     mkdir: 无法创建目录"/home/animal/cat": 没有那个文件或目录
     //mkdir无法创建多级目录
     [root@hadoop1 home]# mkdir -p /home/animal/cat
     [root@hadoop1 home]# ls
     animal  cool  dog  jack  tom
     [root@hadoop1 home]# cd animal
     [root@hadoop1 animal]# ls
     cat
    ```
# 五、rmdir 指令
1. 基本语法：`rmdir [选项] 要删除的空目录`
2. 功能描述：删除空目录

    ```Plain
     [root@hadoop1 ~]# cd /home
     [root@hadoop1 home]# rmdir /home/dog
     [root@hadoop1 home]# ls
     animal  cool  jack  tom
     [root@hadoop1 home]# rmdir animal
     rmdir: 删除 "animal" 失败: 目录非空//注意：只能删除非空目录
    ```
3. `rm -rf 要删除的非空目录` 删除非空目录

    ```Plain
     [root@hadoop1 home]# rm -rf animal
     [root@hadoop1 home]# ls
     cool  jack  tom
    ```
# 六、touch 指令
1. 基本语法：`touch 文件名`
2. 功能描述：创建空文件
# 七、cp 指令
1. 基本语法：`cp [选项] source dest`
2. 功能描述：拷贝文件到指定目录

    ```Plain
     //将 /home/aaa.txt 拷贝到 /home/bbb 目录下
     [root@hadoop1 ~]# cd /home
     [root@hadoop1 home]# touch aaa.txt
     [root@hadoop1 home]# ls
     aaa.txt  cool  jack  tom
     [root@hadoop1 home]# mkdir bbb
     [root@hadoop1 home]# ls
     aaa.txt  bbb  cool  jack  tom
     [root@hadoop1 home]# cp aaa.txt bbb
     [root@hadoop1 home]# cd bbb
     [root@hadoop1 bbb]# ls
     aaa.txt
    ```
3. 常用选项 -r ：递归复制整个文件夹

    ```Plain
     //递归复制 /home/bbb整个文件夹到/pot下
     [root@hadoop1 home]# cp -r /home/bbb /opt
     [root@hadoop1 home]# cd ..
     [root@hadoop1 /]# cd /opt
     [root@hadoop1 opt]# ls
     bbb  rh  VMwareTools-10.3.10-13959562.tar.gz  vmware-tools-distrib
     [root@hadoop1 opt]# cd bbb
     [root@hadoop1 bbb]# ls
     aaa.txt
    ```
4. 如果复制到的目标文件夹内有同名的文件，如果**默认执行覆盖文件操作**怎么办

    ```Plain
     [root@hadoop1 home]# vim aaa.txt
     [root@hadoop1 home]# cp aaa.txt bbb
     cp：是否覆盖"bbb/aaa.txt"？ y//每次都要确认覆盖操作
     [root@hadoop1 home]# \cp aaa.txt bbb//默认覆盖
    ```
# 八、rm 指令
1. 基本语法：`rm [选项] 要删除的文件或目录`
2. 功能描述：移除文件或者目录
3. 常用选项：
    1. r ：递归删除整个文件夹
    2. f ：强制删除不提示

# 九、mv 指令

基本语法、功能描述：

1. `mv oldNameFile newNameFile` (功能描述：重命名)
2. `mv /temp/movefile /targetFolder` (功能描述：移动文件) = 剪切

    ```Plain
     //移动整个目录
     [root@hadoop1 home]# ls
     aaa.txt  bbb  ccc.txt  cool  jack  tom
     [root@hadoop1 home]# rm -rf bbb
     [root@hadoop1 home]# ls
     aaa.txt  ccc.txt  cool  jack  tom
     [root@hadoop1 home]# cd /opt
     [root@hadoop1 opt]# mv bbb /home
     [root@hadoop1 opt]# ls
     rh  VMwareTools-10.3.10-13959562.tar.gz  vmware-tools-distrib
     [root@hadoop1 opt]# cd /home
     [root@hadoop1 home]# ls
     aaa.txt  bbb  ccc.txt  cool  jack  tom
    ```
# 十、cat 指令（concatenate）
1. 基本语法：`cat [选项] 要查看的文件`
2. 功能描述：查看文件内容
3. 常用选项：-n 显示行号

    ```Plain
     [root@hadoop1 home]# cat -n aaa.txt
         1   hello
    ```
4. cat 只能浏览文件，而不能修改文件，为了浏览方便，一般会带上管道命令 `| 其他指令`
# 十一、more 指令
1. 基本语法：`more 要查看的文件`
2. 功能描述：是一个基于VI编辑器的文本过滤器，它以全屏幕的方式按页显示文本文件的内容

    ![[IMG-20260404031958217.png|648]]

3. 内置的若干快捷键：

| 操作 | 功能说明 |
| --- | --- |
| 空白键 (space) | 代表向下翻一页； |
| Enter | 代表向下翻『一行』 ； |
| q | 代表立刻离开 more ，不再显示该文件内容。 |
| Ctrl+F | 向下滚动一屏 |
| Ctrl+B | 返回上一屏 |
| = | 输出当前行的行号 |
| :f | 输出文件名和当前行的行号 |

# 十二、less 指令
1. 基本语法：`less 要查看的文件`
2. 功能描述：用来分屏查看文件内容，它的功能与more指令类似，但是比more指令更加强大，支持各种显示终端。 less指令在显示文件内容时，并不是一次将整个文件加载之后才显示，而是**根据显示需要加载内容， 对于显示大型文件具有较高的效率**
3. 内置的若干快捷键：

| 操作 | 功能说明 |
| --- | --- |
| 空白键 | 向下翻动一页； |
| [pagedown] | 向下翻动一页 |
| [pageup] | 向上翻动一页； |
| /字串 | 向下搜寻『字串』 的功能； n：向下查找； N：向上查找； |
| ?字串 | 向上搜寻『字串』 的功能； n：向上查找； N：向下查找； |
| q | 离开 less 这个程序； |

# 十三、echo 指令
1. 基本语法：`echo [选项] [输出内容]`
2. 功能描述：输出内容到控制台

    ```Plain
     //使用echo 指令输出环境变量
     [root@hadoop1 home]# echo $HOSTNAME
     hadoop1
     //使用echo 指令输出 hello,world!
     [root@hadoop1 home]# echo "hello world"
     hello world
    ```
# 十四、head 指令
1. 基本语法、功能描述：
    1. head 文件（功能描述：查看文件头10行内容）
    2. head -n 5 文件（功能描述：查看文件头5行内容， 5可以是任意行数）

        ```Plain
         [root@hadoop1 home]# head -n 5 /etc/profile
         # /etc/profile
         # System wide environment and startup programs, for login setup
         # Functions and aliases go in /etc/bashrc
        ```
# 十五、tail 指令

基本语法、功能描述：

1. `tail 文件`（功能描述：查看文件头10行内容）
2. `tail -n 5 文件` （功能描述：查看文件头5行内容， 5可以是任意行数）
3. `tail -f 文件` （功能描述：实时追踪该文档的所有更新）

    ```Plain
     [root@hadoop1 home]# tail -f aaa.txt
     hello
     tail: aaa.txt：文件已截断
     hello,world
    ```
# 十六、> 指令 和 >> 指令
1. 基本语法、功能描述：
    1. `ls -l >文件` （功能描述：列表的内容写入文件a.txt中（**覆盖写**））
    2. `ls -al >>文件` （功能描述：列表的内容**追加到**文件aa.txt的**末尾**）
    3. `cat 文件1 > 文件2` （功能描述：将文件1的内容**覆盖**到文件2）
    4. `echo "内容">> 文件`

# 十七、ln 指令
1. 基本语法、功能描述：

    `ln -s [原文件或目录] [软链接名]` （功能描述：给原文件创建一个软链接）=快捷方式

2. 应用实例 ：

    ```Plain
     //在/home 目录下创建一个软连接 linkToRoot，连接到 /root 目录
     [root@hadoop1 home]# ln -s /root/ /home/myroot
    ```

    ![[IMG-20260404031958275.png|800]]

    ```Plain
     //访问myroot就是root目录
     [root@hadoop1 home]# cd myroot
     [root@hadoop1 myroot]# ls
     anaconda-ks.cfg  hello.java  initial-setup-ks.cfg  公共  模板  视频  图片  文档  下载  音乐  桌面
    ```
    ```Plain
     //删除软链接
     [root@hadoop1 myroot]# rm /home/myroot
     rm：是否删除符号链接 "/home/myroot"？y
    ```
# 十八、history 指令
1. 基本语法、功能描述：

    history（功能描述：查看已经执行过历史命令）

    ![[IMG-20260404031958332.png|587]]

2. 显示最近使用过的10个指令

    ```Plain

       208  cd ..
       209  cd /home
       210  ll
       211  history
       212  history 5
    ```
3. 执行指定历史编号为210的指令

