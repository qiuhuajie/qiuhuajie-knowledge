1. 查看pod logs 被拒绝：

    ```Plain
     [root@master ~]# kubectl logs nginx-f89759699-gfdx7
     Error from server (Forbidden): Forbidden (user=kubernetes, verb=get, resource=nodes, subresource=proxy) ( pods/log nginx-f89759699-gfdx7)
    ```
    解决办法：
    ```Plain
     kubectl create clusterrolebinding kubernetes --clusterrole=cluster-admin --user=kubernetes
     \#执行后：clusterrolebinding.rbac.authorization.k8s.io/kubernetes created
    ```
2. pod 一直处于 ContainerCreating

    ```Plain
     [root@master ~]# kubectl get pod
     NAME                    READY   STATUS              RESTARTS   AGE
     nginx-f89759699-gfdx7   0/1     ContainerCreating   0          10m
    ```
    查看pod 详情
    ```Plain
     [root@master ~]# kubectl describe pod nginx-f89759699-gfdx7
    ```
3. 删除 pod 先删除

    先查看 deployment
    ```Plain
     kubectl get deployment
    ```
    删除 deployment
    ```Plain
     kubectl delete deployment
    ```
    再删除对应的 pod
4. 强制删除 处于 Terminating 状态的 Pod

    ```Plain
     无论各种方式生成的 pod, 均可以使用如下命令强制删除：
     kubectl delete pods <pod> --grace-period=0 --force
    ```
