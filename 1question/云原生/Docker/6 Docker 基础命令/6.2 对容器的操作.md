# **docker commit**
将**自定义配置好的容器打包成镜像**，以便进行复用移植 `docker commit [OPTIONS] CONTAINER [REPOSITORY[:TAG]]`
```Bash
 docker commit -a "qhj"  -m "修改首页" 341d81f7504f guignginx:v1.0
```

# **docker rm**
删除停止的容器
```Bash
 docker rm -f        # -f 强制删除
```
删除全部容器
```Bash
 docker rm -f $(docker ps -a -q)
```
# **docker start**
启动容器
```Bash
 \#使用容器别名
 [root@docker001 ~]# docker start db
 db
 \#使用容器id‘
 [root@docker001 ~]# docker start c6610f7ed6d8
 c6610f7ed6d8
```

# **docker exec**
进入容器执行命令（进入容器内的小Linux bash命令行）
```Bash
 [root@docker001 ~]# docker exec db ls /
 bin
 boot
 dev
 docker-entrypoint-initdb.d
 etc
 home
 lib
```
命令：`docker exec -it 容器名 /bin/bash` ：登入容器的bash
```Bash
 [root@docker001 ~]# docker exec -it db /bin/bash
 
 \#此时已经进入db容器内部的交互式命令行
 root@40dfa03f445c:/# ls
 bin  boot  dev  docker-entrypoint-initdb.d  etc  home  lib  lib32  lib64  libx32  media  mnt  opt  proc  root  run  sbin  srv  sys  tmp  usr  var
 \#退出
 root@40dfa03f445c:/# exit
 exit
```
