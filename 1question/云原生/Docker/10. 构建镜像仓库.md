1. 克隆虚拟机
    
    ```Bash
     \#修改IP
     vim /etc/sysconfig/network-scripts/ifcfg-ens33 #192.168.10.120
     vim /etc/hostname # docker002
     service network restart
     reboot
    ```
    
1. 在001上pull 一个官方给的registry 的模板镜像，用于在001上构建仓库
    
    ```Bash
    docker run -d -v /opt/registry:/var/lib/registry -p 5000:5000 --restart=always registry
    ```
    
1. 修改端口配置文件（默认使用的仓库端口是443，修改表名5000端口也是安全的）
    
    ```Bash
     vim /etc/docker/daemon.json
    ```
    
    ```JSON
     {
             "insecure-registries": ["192.168.10.110:5000"]
     }
    ```
    
1. 重启docker
    
    ```Bash
     systemctl restart docker
    ```
    
1. 上传镜像需要的镜像标识符：`仓库地址/userName/imageName:tag`
    
    ```Bash
     # 修改要上传的镜像的名字
    docker tag tomcat:v1.0 192.168.10.110:5000/tomcat:v1.0
    ```
    
1. 上传本地镜像到仓库
    
    ```Bash
    docker push 192.168.10.110:5000/tomcat:v1.0
    ```
    
1. 查看仓库上的镜像
    
    ```Bash
     [root@docker001 ~]# curl -XGET http://192.168.10.110:5000/v2/_catalog
     {"repositories":["tomcat"]}
    ```
    
1. 在docker002 上将所有容器和镜像都删除
    
    ```Bash
    docker rm -f $(docker ps -a -q)
     
    docker rmi -f $(docker images -q)
    ```
    
1. docker002 也需要修改端口配置文件
    
    ```Bash
    vim /etc/docker/daemon.json
    
    systemctl restart docker
    ```
