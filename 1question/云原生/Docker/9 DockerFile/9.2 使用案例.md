# 1. 案例一：部署 Tomcat
1. 下载`tomcat`和 `Linux JDK`
    
    ```Bash
     http://jdk.java.net/18/
     
     https://tomcat.apache.org/download-80.cgi
    ```
    
1. 将这两个文件复制到 src 文件目录下
    
    ```Bash
     [root@docker001 ~]# mkdir tomcat
     [root@docker001 ~]# cd tomcat
     [root@docker001 tomcat]# ls
     apache-tomcat-8.5.72.tar.gz  openjdk-18-ea+22_linux-x64_bin.tar.gz
    ```
    
1. 在该目录下创建 Dockerfile 文件
    
    ```Bash
    vim Dockerfile
    ```
    
    ```Bash
     # hub.c.163.com/public/centos:6.7 是之前创建的centos 6.7 的基础镜像
     FROM hub.c.163.com/public/centos:6.7
     MAINTAINER coolcool@qq.com
     
     ADD ./apache-tomcat-8.5.72.tar.gz  /root        # 将 tomcat与openjdk 复制到容器内的 /root目录下
     ADD ./openjdk-18-ea+22_linux-x64_bin.tar.gz /root
     
     # 配置环境变量
     ENV JAVA_HOME /root/jdk-18              # 这里jdk-18是解压后的jdk文件名
     ENV PATH $JAVA_HOME/bin:$PATH
     
     EXPOSE 8080
     
     # 镜像一旦启动就会执行这条命令（启动 tomcat 服务器）
     ENTRYPOINT /root/apache-tomcat-8.5.72/bin/startup.sh && tailf /root/apache-tomcat-8.5.72/logs/catalina.out
    ```
