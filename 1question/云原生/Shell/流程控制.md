# 流程控制
1. **if 判断**
    1. 基本语法：

        ```Bash
         if [ 条件判断式 ];then
         程序
         fi
        ```
        ```Bash
         if [ 条件判断式 ]
         then
         程序
         elif [条件判断式]
         then
         程序
         fi
        ```
    2. 应用案例：

        请编写一个shell程序，如果输入的参数，大于等于60，则输出 "及格了"，如果小于60,则输出 "不及格"

        ```Bash
         #!/bin/bash
         if [ $1 -ge 60 ]
         then
             echo "A"
         elif [ $1 -lt 60 ]
         then
             echo "B"
         fi
        ```
2. **case 语句**
    1. 基本语法：

        ```Bash
         case $变量名 in
         "值1"）
         如果变量的值等于值1，则执行程序1
         ;;
         "值2"）
         如果变量的值等于值2，则执行程序2
         ;;
         …省略其他分支…
         *）
         如果变量的值都不是以上的值，则执行此程序
         ;;
         esac
        ```
    2. 应用案例：

        当命令行参数是 1 时，输出 "周一", 是2 时，就输出"周二"， 其它情况输出 "other"

        ```Bash
         #!/bin/bash
         case $1 in
         "1")
         echo "A"
         ;;
         "2")
         echo "B"
         ;;
         *)
         echo "C"
         ;;
         esac
        ```
3. **for 循环**
    1. 基本语法：

        ```Bash
         \#语法一：
         for 变量 in 值1 值2 值3…
         do
         程序
         done
        ```
        ```Bash
         \#b语法二：
         for (( 初始值;循环控制条件;变量变化 ))
         do
         程序
         done
        ```
    2. 应用实例：
        1. 语法一：打印命令行输入的参数（**这里可以看出$* 和 $@ 的区别**）

            ```Bash
             #!/bin/bash
             for i in "$*"
             do
                 echo "num is $i"
             done
             for j in "$@"
             do
                 echo "num is $j"
             done
            ```
            ```Bash
             [root@hadoop001 shcode]# ./fordemo.sh 100 200 300
             num is 100 200 300
             num is 100
             num is 200
             num is 300
            ```
        2. 语法二：从1加到100的值输出显示

            ```Bash
             SUM=0;
             for (( i=1; i<=100; i++))
             do
                 SUM=$[$SUM+i]
             done
             echo "SUM=$SUM"
            ```
            ```Bash
             SUM=5050
            ```
4. **while 循环**
    1. 基本语法：

        ```Bash
         while [ 条件判断式 ]
         do
         程序
         done
        ```
    2. 应用实例：

        从命令行输入一个数n，统计从 1+..+ n 的值是多少

        ```Bash
         #!/bin/bash
         SUM=0;
         i=0;
         while [ $i -le $1 ]
         do
             SUM=$[$SUM+$i]
             i=$[$i+1]
         done
         echo "SUM=$SUM"
        ```
        ```Bash
         [root@hadoop001 shcode]# ./whiledemo.sh 10
         SUM=55
         [root@hadoop001 shcode]# ./whiledemo.sh 100
         SUM=5050
        ```
