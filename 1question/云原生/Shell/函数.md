1. 函数介绍：
    
    shell编程和其它编程语言一样，有系统函数，也可以自定义函数
    
1. **系统函数：**
    
    1. **basename函数：**
        
        1. 基本语法：
            
            1. `basename [pathname] [suffix]`
            
            1. `basename [string] [suffix]`
            
        
        1. 功能：返回完整路径最后 `/`的部分，常用于获取文件名
        
        1. 选项：suffix为后缀，如果suffix被指定了，basename会将pathname或string中的suffix去掉
        
        1. 应用实例：请返回 /home/aaa/test.txt 的 "test.txt" 部分
            
            ```Bash
             [root@hadoop001 home]# basename /home/homeinfo.txt
             homeinfo.txt
             [root@hadoop001 home]# basename /home/homeinfo.txt .txt
             homeinfo
            ```
            
        
    
    1. **dirname函数：**
        
        1. 基本语法：dirname 文件绝对路径
        
        1. 功能：返回完整路径最后 / 的前面的部分，常用于返回路径部分
        
        1. 应用实例：请返回 /home/aaa/test.txt 的 /home/aaa
            
            ```Bash
             [root@hadoop001 home]# dirname /home/homeinfo.txt
             /home
            ```
            
        
    
1. **自定义函数：**
    
    1. 基本语法：
        
        ```Bash
         [ function ] funname[()]
         {
         Action;
         [return int;]
         }
        ```
        
    
    1. 应用案例：
        
        ```Bash
         #!/bin/bash
         function getsum() {
        
             SUM=$[$n1+$n2]
             echo "SUM=$SUM"
         }
         
         read -p "please input n1=" n1
         read -p "please input n2=" n2
         
         getsum $n1 $n2
        ```
        
        ```Bash
         [root@hadoop001 home]# vim getSum.sh
         [root@hadoop001 home]# sh getSum.sh
         please input n1=10
         please input n2=20
         SUM=30
        ```
