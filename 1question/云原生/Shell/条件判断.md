1. 基本语法：
    
    `[ condition ]`（注意condition前后要有空格）
    
    \#非空返回true，可使用$?验证（0为true， >1为false）
    
1. 常用判断条件
    
    1. **两个整数的比较**
        
        1. `=` 字符串比较
        
        1. `lt` 小于
        
        1. `le` 小于等于
        
        1. `eq` 等于
        
        1. `gt` 大于
        
        1. `ge` 大于等于
        
        1. `ne` 不等于
        
    
    1. **按照文件权限进行判断**
        
        1. `r` 有读的权限
        
        1. `w` 有写的权限
        
        1. `x` 有执行的权限
        
    
    1. **按照文件类型进行判断**
        
        1. `f` 文件存在并且是一个常规的文件
        
        1. `e`文件存在
        
        1. `d` 文件存在并是一个目录
        
    
1. 应用实例：
    
    1. 案例1： "ok"是否等于"ok"
        
        ```Bash
         if [ "ok" = "ok" ]
         then
                 echo "equal"
         fi
        ```
        
    
    1. 案例2： 23是否大于等于22
        
        ```Bash
         if [ "23" -ge "22" ]
         then
                 echo "大于"
         fi
        ```
        
    
    1. 案例3： /root/shell/aaa.txt 目录中的文件是否存在
        
        ```Bash
         if [ -f /root/shcode/aaa.txt ]
         then
                 echo "存在"
         fi
        ```
        
    
    1. 案例4：只有空格表示假
        
        ```Bash
         \#保证中括号周围有空格
         if [   ]
         then
                 echo "为假"
         fi
        ```
