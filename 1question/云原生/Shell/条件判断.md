# 一、条件判断
1. 基本语法：

    `[ condition ]`（注意condition前后要有空格）

    \#非空返回true，可使用$?验证（0为true， >1为false）

2. 常用判断条件
    1. **两个整数的比较**
        1. `=` 字符串比较
        2. `lt` 小于
        3. `le` 小于等于
        4. `eq` 等于
        5. `gt` 大于
        6. `ge` 大于等于
        7. `ne` 不等于
    2. **按照文件权限进行判断**
        1. `r` 有读的权限
        2. `w` 有写的权限
        3. `x` 有执行的权限
    3. **按照文件类型进行判断**
        1. `f` 文件存在并且是一个常规的文件
        2. `e`文件存在
        3. `d` 文件存在并是一个目录
3. 应用实例：
    1. 案例1： "ok"是否等于"ok"

        ```Bash
         if [ "ok" = "ok" ]
         then
                 echo "equal"
         fi
        ```
    2. 案例2： 23是否大于等于22

        ```Bash
         if [ "23" -ge "22" ]
         then
                 echo "大于"
         fi
        ```
    3. 案例3： /root/shell/aaa.txt 目录中的文件是否存在

        ```Bash
         if [ -f /root/shcode/aaa.txt ]
         then
                 echo "存在"
         fi
        ```
    4. 案例4：只有空格表示假

        ```Bash
         \#保证中括号周围有空格
         if [   ]
         then
                 echo "为假"
         fi
        ```
    5. 案例5：只有空格表示假
