# Entropy-based Reinforcement Learning for Computation Offloadingservice in Software-defined Multi-access Edge Computing

期刊：

# 1. 摘要

物联网（loT）设备的快速增长和多种边缘应用的出现导致了网络边缘数据流量的爆炸性增长。多接入边缘计算（MEC）中的计算卸载服务使网络有可能提供比传统网络更好的服务质量（Oualityof Service）。它们有望减少传播延迟，提高对延迟敏感的任务的计算能力，特别是。然而，边缘设备的分布式计算资源迫切需要合理的资源控制器来确保这些分布式计算资源得到有效的调度。软件定义网络（SDN）的好处可以通过MEC服务来充分展示其潜力，以减少程序的响应时间。本文提出了一个新的基于SDN的MEC计算卸载服务架构，以提高控制面的协调和卸载能力。此外，为了应对网络的动态变化和提高探索程度，我们提出了一种新型的基于熵的强化学习算法，用于网络边缘的延迟敏感任务计算卸载。最后，评估结果表明，我们提出的模型有可能显著改善网络资源分配和平衡性能。
