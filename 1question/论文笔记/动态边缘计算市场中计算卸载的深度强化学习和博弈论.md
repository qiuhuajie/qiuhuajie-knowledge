# Deep Reinforcement Learning and Game Theory for Computation Offloading in Dynamic Edge Computing Markets

期刊：

# 1. 摘要

作为一种有前途的模式，计算卸载技术可以将计算任务卸载到多接入边缘计算（MEC）服务器上，这对资源受限的终端设备来说是一个吸引人的选择，可以减少它们的计算工作量。然而，由于资源有限，计算卸载的一个关键研究挑战是设计适当的卸载策略，以确定在某些复杂情况下应该卸载哪些任务。在本文中，我们研究了在一个由软件定义的网络（SDN）驱动的具有多个用户和多个服务器的MEC环境中的卸载决策问题。为了确保终端用户不滥用MEC系统中的计算资源，我们将MEC服务器的利润制定为我们的优化目标。我们联合**优化MEC服务器的选择、卸载数据的大小和MEC计算服务的价格**，以实现MEC服务器的利润最大化。我们将深度强化学习（DRL）和博弈论应用于我们提出的方法。具体来说，我们**提出了一个近似策略优化（PPO）的强化学习框架来解决MEC服务器的选择问题**。其次，我们制定了一个两步优化问题，以确定卸载数据的大小和计算服务的定价。通过实现终端用户之间战略博弈的纳什均衡，确定了这两个问题的最优值。广泛的模拟结果证明，我们的建议在收敛时间和稳定性方面比现有解决方案有更好的表现。
