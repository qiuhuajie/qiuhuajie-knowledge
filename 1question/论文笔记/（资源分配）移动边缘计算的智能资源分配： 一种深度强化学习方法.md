# Smart Resource Allocation for Mobile Edge Computing: A Deep Reinforcement Learning Approach

期刊：

# 1. 摘要

随着通信和感知能力的提高，移动设备的发展带来了大量复杂和计算密集型的移动应用，资源有限的移动设备面临着比以往更严重的容量限制。作为网络架构的一个新概念和云计算的延伸，移动边缘计算（MEC）似乎是一个有前途的解决方案，以满足这一新的挑战。然而，MEC也有一些局限性，如基础设施部署和维护的高成本，以及复杂和突变的边缘计算环境给MEC服务器带来的严重压力。在这一点上，如何合理分配计算资源和网络资源以满足移动设备在多变的MEC条件下的需求，已经成为一个巨大的难题。为了解决这个问题，我们提出了一种智能的、基于深度强化学习的资源分配（DRLRA）方案，它可以在变化的MEC环境下自适应地分配计算和网络资源，减少平均服务时间并平衡资源的使用。 实验结果表明，所提出的DRLRA在变化的MEC条件下比传统的OSPF算法表现更好。
