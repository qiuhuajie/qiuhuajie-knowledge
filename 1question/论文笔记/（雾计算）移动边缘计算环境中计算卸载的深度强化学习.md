# Deep Reinforcement Learning for Computation Offloading in Mobile Edge Computing Environment

期刊：

# 1. 摘要

最近，为了将计算、网络资源、服务分布在终端附近，移动雾逐渐成为移动边缘计算（MEC）的范式，在移动雾环境中，服务质量受到卸载速度和雾处理的影响，然而传统的雾方法解决计算资源分配的问题是困难的，因为网络状态分布环境复杂（即F-AP状态、 本文为了提高移动设备的雾资源配置性能，提出了基于学习的**移动雾**方案与**深度确定性策略梯度（DDPG）算法**，一个卸载块脉动的离散事件系统被建模为马尔可夫决策过程（MDPs），它可以在不知道不同网络状态之间的转换概率的情况下实现卸载计算。此外，DDPG算法被用来解决状态空间爆炸的问题，并学习分布式移动雾计算的最佳卸载策略。仿真结果表明，与政策梯度（PG）、确定性政策梯度（DPG）和行为批评（AC）方法相比，我们提出的方案在相关性能上分别取得了20%、37%和46%的改进。此外，与传统的雾资源配置方案相比，我们的方案在不同地点数量和不同任务到达率的情况下，显示出更好的雾资源配置成本性能。
