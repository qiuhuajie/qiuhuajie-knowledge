# Approach for Large-Scale Mobile EdgeComputing via Deep Reinforcement Learning

期刊：

# 1. 摘要

摘要-移动边缘计算(MobileEDGE，MEC)被认为是一种能够有效提高无线用户设备(WUDS)计算能力和移动应用体验质量的有前涂的范例。MEC最关键的问题之一是计算卸载，它决定了如何卸载WUDS的任务，以便进行进一步的密集计算，传统的基于数学规划的卸载方法由于信道条件的时变(主要是Wud移动性)在动态MEC环境中可能面临困难。针对这一问题，提出了基于强化学习(RL)的知载方法。通过将MEC状态映到知载操作来开发知载策略。不过。由于指数增长的状态和作用空间，这些方法可能无法在大规模MEC中收敛。在本文中，我们提出了一种新的在线计算知载方法，能够有效地降低大规模WUD动态MEC中的任务延迟和能耗。首先，提出了一种基于RL的计算卸载和能量传输算法，以加速学习过程。然后，采用一种联合优化方法来开发分配算法，从而获得能源和计算资源分配的近似最优解。仿真结果表明，所提出的方法可以有效地收敛，并取得显着的性能改善比基线方法  
索引词-移动边缘计算，计算卸载，强化学习，资源分配
