# An Efficient Online Computation Offloading Approach for Large-Scale Mobile Edge Computing via Deep Reinforcement Learning
期刊：
# 1. 摘要
摘要--移动边缘计算（MEC）被认为是一种有前途的模式，可以有效地提高无线用户设备WUD的计算能力和移动应用的体验质量。MEC最关键的问题之一是计算卸载，它决定了如何将WUD的任务卸载到边缘设备上，以便进一步进行密集的计算。传统的基于数学编程的卸载方法在动态的MEC环境中可能会由于时变的信道条件（主要由WUD的移动性引起）而面临麻烦。为了解决这个问题，人们**提出了基于强化学习（RL）的卸载方法**，这些方法通过将MEC状态映射到卸载动作来制定卸载策略。然而，由于指数级增长的状态和行动空间，这些方法在大规模的MEC中可能无法收敛。在这篇文章中，我们**提出了一种新的在线计算卸载方法，可以有效地减少大规模WUD动态MEC中的任务延迟和能量消耗**。首先，我们提出了一种基于RL的计算卸载和能量传输算法来加速学习过程。然后，采用了一种优化方法来开发分配算法，该算法获得了能量和计算资源分配的接近最优的解决方案。仿真结果表明，与基线方法相比，所提出的方法可以收敛，并取得明显的性能改进。
