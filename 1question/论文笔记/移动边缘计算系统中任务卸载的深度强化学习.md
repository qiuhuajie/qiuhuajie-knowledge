# Deep Reinforcement Learning for Task Offloading in Mobile Edge Computing Systems
期刊：
# 1. 摘要
在移动边缘计算系统中，当大量移动设备将任务卸载给边缘节点时，边缘节点可能会有很高的负载。那些被卸载的任务可能会经历很大的处理延迟，甚至在它们的截止日期到期时被丢弃。由于边缘节点的负载动态不确定，对于每个设备来说，以分散的方式确定其卸载决策(即是否卸载，以及应该将其任务卸载到哪个边缘节点)是一个挑战。在这项工作中，我们考虑了不可分割和延迟敏感的任务以及edoe负载动态。并制定一个任务卸载问题，以最小化预期的长期成本。提出了一种基于深度强化学习的无模型分布式算法，每个设备可以在不知道其他设备的任务模型和卸载决策的情况下确定自己的卸载决策。改进了算法中对长期代价的估计。我们结合了==**长短期记忆(LSTM)**==**、决斗深q网络(DON)和双DON技术**。仿真结果表明，与现有算法相比，该算法能更好地利用边缘节点的处理能力，显著降低任务丢失率和平均延迟。
