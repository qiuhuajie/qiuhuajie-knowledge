# Resource Allocation Based on Deep ReinforcementLearning in IoT Edge Computing
# 1. 摘要

通过利用移动边缘计算(MEC)，物联网(物联网)设备产生的大量数据可以在网络边缘进行处理和分析。然而，MEC系统通常只有有限的虚拟资源，这些资源被物联网边缘应用所共享和竞争因此，我们提出了一种用于边缘计算系统的资源分配策略，以提高资源利用效率。建议的政策的目的，是尽量减少职位平均完成时间的长期加权和所需资源的平均数目。将MEC系统中的资源分配问题描述为一个马尔可夫决策过程。针对这一问题，本文提出了一种深度强化学习方法。我们还提出了一种改进的深度Q-网络(DQN)算法来学习该策略，其中使用多个重放存储器分别存储相互影响较小的经验。仿真结果表明，与原DQN算法相比，该算法具有更好的收敛性能，相应的策略以更低的完成时间和更少的请求资源优于其它参考策略。
