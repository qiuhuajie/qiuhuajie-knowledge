# Cooperative Data Sensing and Computation Offloading in UAV-Assisted Crowdsensing With Multi-Agent Deep Reinforcement Learning
期刊：
# 1. 摘要
无人机（UAV）可以在移动众测（MCS）中通过计算卸载和数据传感来完成远程或农村地区的传感任务，但在大多数现有的研究中，计算卸载和数据传感都是单独研究的。在本文中，我们为无人机辅助的MCS系统提出了一个新的合作数据传感和计算卸载方案，目的是使整个系统效用最大化。首先，通过共同考虑飞行方向、飞行距离、任务卸载比例和每个无人机的服务器卸载选择，制定了一个多目标函数来评估系统效用。然后，该问题被建模为一个部分可观察的马尔可夫决策过程，并提出了一个多代理行为者-批评算法框架来训练无人机的策略网络。由于多个代理人之间的通信引起的高延迟和能源成本，我们训练一个**集中的批评网络**来模拟其他代理人，并在所有无人机之间寻求平衡，而不是采用明确的信息交流渠道。此外，我们还引入了**注意力机制**来提高模型训练阶段的收敛性能。最后，实验结果证明了我们方案的有效性和适用性。与基线相比，我们的算法在收敛性能、系统效用、任务成本和任务完成率方面表现出明显的优势。
